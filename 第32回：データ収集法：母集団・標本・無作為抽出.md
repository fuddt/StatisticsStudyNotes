今回は、**データ収集法**に入ります。

ここまでの回では、集められたデータをどう見るかを学びました。

```text
第27回：データの種類・グラフ・ヒストグラム
第28回：平均・中央値・四分位・箱ひげ図
第29回：標準化・変動係数・歪度・尖度・ジニ係数
第30回：散布図・共分散・相関係数
第31回：見かけの相関・偏相関・クロス表
```

今回からは少し視点が変わります。

テーマは、

```text
そもそも、そのデータはどうやって集めたのか？
```

です。

統計検定2級の範囲にも、観察研究、実験研究、調査の設計、母集団、標本、全数調査、標本調査、ランダムネス、無作為抽出、標本サイズ、標本誤差、偏りの源、各種標本抽出法が含まれています。

# 1. 今回のゴール

今回のゴールはこれです。

```text
1. 母集団と標本の違いが分かる
2. 全数調査と標本調査の違いが分かる
3. なぜ標本調査が必要なのか分かる
4. 無作為抽出の意味が分かる
5. 標本誤差と偏りの違いが分かる
6. 「データが多いだけでは信頼できない」と分かる
```

今回の内容は、計算よりも概念です。

でも、かなり重要です。

なぜなら、統計は

```text
手元のデータから、見えない全体を推測する技術
```

だからです。

この「手元のデータ」が偏っていたら、その後にどれだけ高度な分析をしても意味がありません。

---

# 2. 母集団とは何か

母集団とは、**本当に知りたい対象全体**のことです。

たとえば、

```text
日本の高校3年生の平均身長を知りたい
```

とします。

このとき母集団は、

```text
日本の高校3年生全員
```

です。

別の例も見ます。

|知りたいこと|母集団|
|---|---|
|日本の有権者の支持政党|日本の有権者全員|
|ある学校の生徒の平均点|その学校の生徒全員|
|JRAレースにおける単勝1番人気の勝率|対象期間のJRA全レースの1番人気馬|
|ある工場の製品不良率|その工場で作られた全製品|
|あるWebサイトの平均応答時間|対象期間内の全リクエスト|

母集団は、

```text
本当に知りたい全体
```

です。

---

# 3. 標本とは何か

標本とは、**母集団から一部だけ取り出したデータ**です。

たとえば、日本の高校3年生全員の身長を測るのは大変です。

そこで、全国から1,000人だけ選んで身長を測る。

この1,000人が標本です。

```text
母集団：日本の高校3年生全員
標本：その中から選んだ1,000人
```

図にするとこうです。

```text
母集団：知りたい全体
┌────────────────────┐
│  ○ ○ ○ ○ ○ ○ ○ ○ ○ │
│  ○ ○ ○ ○ ○ ○ ○ ○ ○ │
│  ○ ○ ○ ○ ○ ○ ○ ○ ○ │
│  ○ ○ ○ ○ ○ ○ ○ ○ ○ │
└────────────────────┘
          ↓ 一部を選ぶ
標本：実際に調べる一部
┌────────────┐
│  ○ ○ ○ ○ ○ │
└────────────┘
```

統計では、この標本を使って母集団を推測します。

---

# 4. 母数とは何か

母集団に関する本当の値を、**母数**といいます。

たとえば、

|母集団で知りたい値|名前|
|---|---|
|母集団全体の平均|母平均|
|母集団全体の分散|母分散|
|母集団全体の割合|母比率|
|母集団全体の相関|母相関係数|

たとえば、

```text
日本の高校3年生全員の平均身長
```

は母平均です。

でも、普通は全員を測れないので、正確な母平均は分かりません。

だから標本から推定します。

```text
標本平均を使って、母平均を推定する
```

これが、これまで学んできた信頼区間や仮説検定につながります。

---

# 5. 標本統計量とは何か

標本から計算した値を、**標本統計量**といいます。

|標本から計算する値|名前|
|---|---|
|標本の平均|標本平均|
|標本の分散|標本分散|
|標本の割合|標本比率|
|標本の相関|標本相関係数|

整理するとこうです。

|対象|平均|分散|割合|
|---|---|---|---|
|母集団|母平均|母分散|母比率|
|標本|標本平均|標本分散|標本比率|

ここはかなり重要です。

統計検定2級では、母集団と標本の区別が曖昧だと、推定・検定で崩れます。

---

# 6. なぜ全員を調べないのか

理想を言えば、母集団全体を調べればよいです。

これを**全数調査**といいます。

たとえば、

```text
学校の1クラス40人全員の点数を調べる
```

なら、全数調査できます。

でも、多くの場合、全数調査は難しいです。

理由は主に4つあります。

|理由|例|
|---|---|
|費用がかかる|全国民への調査は高コスト|
|時間がかかる|全製品検査に時間がかかる|
|物理的に難しい|海の魚の数を全部数えられない|
|調査で対象が壊れる|電池寿命テスト、強度試験など|

特に最後は重要です。

たとえば、製品の耐久試験で、

```text
壊れるまで試験する
```

なら、全数調査をしたら商品が全部壊れます。

だから、一部を調べるしかありません。

---

# 7. 標本調査とは何か

標本調査とは、母集団の一部だけを調べて、母集団全体を推測する方法です。

```text
母集団全体を直接見るのは難しい
だから、一部の標本を見る
その標本から全体を推測する
```

これが標本調査です。

たとえば、選挙前の世論調査では、有権者全員に聞くのではなく、数千人に聞いて全体の支持率を推測します。

```text
標本：調査した数千人
母集団：有権者全員
```

ただし、ここで重要なのは、

```text
一部を調べれば何でもよいわけではない
```

ということです。

標本が偏っていたら、推測も偏ります。

---

# 8. 悪い標本の例

日本の有権者の支持政党を知りたいとします。

でも、次のような調査をしたらどうでしょう。

```text
自分のSNSフォロワー100人にアンケートした
```

これはかなり危険です。

理由は、自分のSNSフォロワーは、日本の有権者全体を代表していない可能性が高いからです。

たとえば、

```text
年齢層が偏っている
地域が偏っている
政治関心が高い人に偏っている
似た考えの人が集まっている
```

かもしれません。

この場合、サンプルサイズが100人でも、1,000人でも、10,000人でも、偏った標本なら危険です。

ここが本質です。

```text
データ数が多い = 正しい
```

ではありません。

```text
母集団を代表するように集められているか
```

が重要です。

---

# 9. 無作為抽出とは何か

無作為抽出とは、母集団の各対象が、一定のルールに従って偶然に選ばれるようにする抽出方法です。

ざっくり言えば、

```text
人間の都合や好みで選ばない
偶然に任せて選ぶ
```

ということです。

たとえば、1,000人の生徒に番号を振り、乱数で100人を選ぶ。

これは無作為抽出です。

```text
1番〜1000番の生徒
↓
乱数で100人を選ぶ
↓
標本
```

---

# 10. なぜ無作為抽出が重要なのか

無作為抽出が重要なのは、偏りを減らすためです。

もし調査員が好きに選ぶと、

```text
話しかけやすい人だけ選ぶ
目立つ人だけ選ぶ
協力的な人だけ選ぶ
近くにいた人だけ選ぶ
```

という偏りが入ります。

無作為抽出では、そういう人間側の選択バイアスをなるべく消します。

つまり、無作為抽出は、

```text
標本を母集団に近づけるための仕組み
```

です。

---

# 11. ランダムなら必ず正確なのか

ここは誤解しやすいです。

無作為抽出をしても、標本は偶然にブレます。

たとえば、母集団の男女比が50:50でも、100人を無作為抽出したら、

```text
男性52人、女性48人
```

になるかもしれません。

あるいは、

```text
男性46人、女性54人
```

になるかもしれません。

これは偶然のブレです。

つまり、無作為抽出をしても、

```text
完全に母集団と同じ構成になる
```

わけではありません。

ただし、無作為抽出なら、そのブレを確率的に扱えます。

これが推定・信頼区間・仮説検定の土台です。

---

# 12. 標本誤差とは何か

標本誤差とは、標本を使うことによって生じる偶然のズレです。

たとえば、母集団の本当の支持率が50%だったとします。

でも、標本100人を取ると、

```text
標本支持率 = 47%
```

になるかもしれません。

別の100人を取ると、

```text
標本支持率 = 53%
```

になるかもしれません。

このように、標本を取るたびに結果がブレます。

これが標本誤差です。

```text
標本誤差 = 標本を使う以上、偶然に生じるズレ
```

です。

---

# 13. 標本誤差は悪者ではない

標本誤差は避けられません。

標本調査をする以上、必ずあります。

でも、標本誤差は確率的に扱えます。

たとえば、標本サイズを大きくすると、標本平均のばらつきは小さくなります。

これはすでに第20回の標本分布でやった話につながります。

標本平均の標準誤差は、母標準偏差を σ、標本サイズを n とすると、

```text
標準誤差 = σ / √n
```

でした。

つまり、nが大きくなると、標本平均のブレは小さくなります。

ここで「√n」が出てくるのが重要です。

標本サイズを4倍にしても、誤差は4分の1ではなく、だいたい2分の1になります。

---

# 14. 偏りとは何か

偏りとは、調査方法や標本の取り方によって、一方向にズレることです。

標本誤差が「偶然のブレ」だとすると、偏りは「構造的なズレ」です。

たとえば、日本の有権者全体の支持率を知りたいのに、若者ばかり集めて調査したとします。

その場合、結果は若者寄りになります。

これは偶然ではなく、調査設計の問題です。

```text
偏り = データの集め方が悪くて、結果が一方向にズレること
```

です。

---

# 15. 標本誤差と偏りの違い

ここは試験でも重要です。

|項目|標本誤差|偏り|
|---|---|---|
|原因|一部だけを調べることによる偶然|調査方法・抽出方法の問題|
|方向|上にも下にもブレる|一方向にズレやすい|
|標本サイズを増やすと|小さくなりやすい|消えないことがある|
|数学的扱いやすさ|扱いやすい|扱いにくい|
|例|たまたま支持率が少し高く出る|若者だけ調査して若者寄りになる|

これを一言で言うと、

```text
標本誤差は偶然のズレ
偏りは設計ミスによるズレ
```

です。

---

# 16. 大量データでも偏りは消えない

ここはかなり重要です。

たとえば、SNSで10万人にアンケートを取ったとします。

データ数は多いです。

でも、そのSNSの利用者が若者に偏っていたり、政治関心の高い人に偏っていたりしたら、日本全体の意見を正しく表しているとは限りません。

```text
偏った10万人
```

より、

```text
きちんと無作為抽出された1,000人
```

の方が、母集団推定としては信頼できる場合があります。

これは直感に反するかもしれません。

でも、かなり重要です。

```text
量より設計
```

です。

もちろん、設計が同じならデータ数が多い方がよいです。

でも、設計が悪いデータを増やしても、偏りが増幅されるだけです。

---

# 17. 偏りの源

代表的な偏りの源を整理します。

|偏りの種類|内容|例|
|---|---|---|
|選択バイアス|特定の人だけ選ばれる|SNS利用者だけ調査|
|非回答バイアス|回答しない人が偏る|忙しい人ほど回答しない|
|測定バイアス|測り方が偏る|誘導的な質問|
|生存者バイアス|残ったものだけ見る|成功企業だけ分析|
|自己選択バイアス|参加者が自分で選ぶ|興味がある人だけアンケートに答える|
|観測範囲の偏り|見える範囲だけ見る|ログに残った行動だけ分析|

統計検定2級では、細かい用語暗記より、

```text
なぜその調査結果が偏るのか
```

を説明できることが重要です。

---

# 18. 競馬AIでの偏り

ここはあなたの文脈だとかなり重要です。

競馬データでも、偏りは普通に起きます。

たとえば、

```text
過去に存在したレースだけを見る
欠損が少ないレースだけを見る
特定のオッズが取れるレースだけ見る
人気馬が揃ったレースだけ見る
払戻データが取れる条件だけ見る
特定期間だけ見る
```

こういう時点で、すでに標本が偏っている可能性があります。

さらに危険なのは、

```text
勝ったパターンだけを後から探す
```

ことです。

これは典型的な生存者バイアス・後出し選択です。

データ分析で一番危ないのは、

```text
都合のいい標本を、母集団の代表だと思い込む
```

ことです。

ここを見落とすと、バックテストで勝っても、実運用で崩れます。

---

# 19. 標本サイズとは何か

標本サイズとは、標本に含まれるデータの数です。

よく n で表します。

```text
n = 標本サイズ
```

たとえば、100人を調査したら、

```text
n = 100
```

です。

1,000レースを分析したら、

```text
n = 1000
```

です。

標本サイズが大きいほど、標本誤差は小さくなりやすいです。

ただし、さっき言った通り、

```text
偏りは標本サイズを増やしても消えない
```

ことがあります。

---

# 20. 標本サイズと標準誤差

標本サイズが大きくなると、推定値のブレは小さくなります。

標本平均の標準誤差は、

```text
標準誤差 = σ / √n
```

でした。

具体例で見ます。

母標準偏差を20とします。

|標本サイズ n|標準誤差 20/√n|
|--:|--:|
|25|4.0|
|100|2.0|
|400|1.0|
|1600|0.5|

nを4倍にすると、標準誤差は半分になります。

ここで大事なのは、

```text
標本サイズを増やす効果は、√で効く
```

ということです。

データ数を増やせば無限に簡単に精度が上がる、という感覚は危険です。

---

# 21. 無作為抽出と代表性

無作為抽出の目的は、標本に代表性を持たせることです。

代表性とは、

```text
標本が母集団の縮図になっていること
```

です。

たとえば、日本の有権者全体を推測したいなら、標本にも年齢、性別、地域などが大きく偏らないようにしたい。

ただし、完全な縮図を作るのは難しいです。

だから無作為抽出や層化抽出などの方法を使って、できるだけ偏りを減らします。

次回扱う標本抽出法は、この代表性を高めるための方法です。

---

# 22. 悪い調査例

次の調査を考えます。

```text
ある学校で「昼休みに図書館にいた生徒100人」に、
学校生活に満足しているかを聞いた。
```

この結果を使って、

```text
この学校の生徒全体の満足度
```

を推測してよいでしょうか。

かなり危険です。

理由は、

```text
昼休みに図書館にいる生徒
```

が学校全体を代表していない可能性があるからです。

たとえば、

```text
静かな場所が好きな生徒に偏っている
部活動をしていない生徒に偏っている
友人関係の傾向が違うかもしれない
```

という偏りがあります。

このように、標本の取り方が悪いと、結果は信用しにくくなります。

---

# 23. 良い調査例

同じ学校の満足度を調べるなら、たとえば次の方がよいです。

```text
全校生徒の名簿から、乱数で100人を選ぶ
```

この場合、特定の場所にいた生徒だけを選ぶよりは、偏りが小さくなります。

さらに、

```text
学年ごとに人数比に応じて選ぶ
```

とすれば、学年の偏りも減らせます。

これは次回扱う**層化抽出**につながります。

---

# 24. 統計検定で問われやすいポイント

## 問い方1：母集団と標本

```text
全国の高校3年生の平均睡眠時間を知るため、全国から無作為に選んだ2,000人を調査した。
このとき母集団はどれか？
```

答えは、

```text
全国の高校3年生全員
```

です。

調査した2,000人は標本です。

---

## 問い方2：標本誤差

```text
標本調査では、調査対象を無作為に選んでも結果が母集団の値からズレることがある。
この偶然によるズレを何というか？
```

答えは、

```text
標本誤差
```

です。

---

## 問い方3：偏り

```text
ある市の住民の意見を知るため、市役所に来た人だけにアンケートを取った。
この調査の問題点として最も適切なものはどれか？
```

答えは、

```text
市役所に来た人だけに偏っており、住民全体を代表していない可能性がある
```

です。

---

## 問い方4：無作為抽出

```text
無作為抽出の目的として最も適切なものはどれか？
```

答えは、

```text
標本に人為的な偏りが入るのを防ぎ、母集団を代表しやすくするため
```

です。

# 26. 今回の最重要ポイント

今回一番大事なのはこれです。

```text
統計では、データの量よりも、データの集め方が先に重要。
```

データが大量にあっても、集め方が偏っていれば危険です。

もう1つ大事なのはこれです。

```text
標本誤差と偏りは別物。
```

標本誤差は、標本を使う以上避けられない偶然のブレです。

偏りは、調査設計のまずさによる構造的なズレです。

この2つを混同すると、かなり危ないです。

```text
標本サイズを増やせば標本誤差は小さくなりやすい。
でも、偏りは標本サイズを増やしても消えないことがある。
```

ここは必ず押さえてください。
