# 2. 前回の復習：相関とは何だったか

相関係数は、2つの変数の直線的な関係の強さを表す指標でした。

```text
r > 0：正の相関
r < 0：負の相関
r ≒ 0：直線的な関係が弱い
```

たとえば、

```text
勉強時間が長いほど、テスト点数が高い
```

なら、正の相関です。

```text
単勝オッズが高いほど、勝率が低い
```

なら、負の相関です。

ただし、前回かなり強調したように、

```text
相関は因果を意味しない
```

です。

今回の話は、まさにここを深掘りします。

# 3. 見かけの相関とは何か

見かけの相関とは、本当は直接の関係がない、または弱いのに、別の要因のせいで相関があるように見えることです。

別名で、

```text
擬相関
```

とも言います。

典型例はこれです。

```text
アイスの売上が増えると、水難事故も増える
```

データだけ見ると、アイス売上と水難事故には正の相関がありそうです。

でも、だからといって、

```text
アイスを食べると水難事故が増える
```

とは言えません。

本当は、

```text
気温が高い
```

という第3の変数が関係している可能性があります。

```text
気温が高い
├── アイスが売れる
└── 水遊びが増えて水難事故が増える
```

この場合、アイス売上と水難事故の相関は、気温によって作られた見かけの相関です。

---

# 4. 第3の変数とは何か

第3の変数とは、見ている2つの変数の両方に影響している別の変数です。

たとえば、

```text
X：アイス売上
Y：水難事故
Z：気温
```

とします。

このとき、ZがXにもYにも影響しています。

```text
Z → X
Z → Y
```

すると、XとYに直接の因果関係がなくても、相関が出ることがあります。

図で書くとこうです。

```text
      気温 Z
      /    \
     ↓      ↓
アイス売上 X   水難事故 Y
```

このとき、XとYだけを見ると、

```text
XとYには関係がある
```

ように見えます。

でも本当は、

```text
Zが両方を動かしている
```

可能性があります。

---

# 5. 具体例：身長と語彙力

少し変な例で考えます。

小学生から高校生までをまとめて調査して、

```text
身長が高い人ほど、語彙力テストの点数が高い
```

という相関が出たとします。

これを見て、

```text
身長が伸びると語彙力が上がる
```

と言えるでしょうか。

言えません。

本当は、

```text
年齢
```

が効いている可能性が高いです。

```text
年齢が上がる
├── 身長が高くなる
└── 語彙力も上がる
```

つまり、

```text
身長と語彙力の相関は、年齢による見かけの相関かもしれない
```

ということです。

# 6. 競馬データでの見かけの相関

ここはあなたの文脈だと特に重要です。

たとえば、次のような分析をしたとします。

```text
騎手Aの単勝回収率が高い
```

このとき、すぐに、

```text
騎手Aは買える
騎手Aには市場が過小評価する何かがある
```

と考えるのは危険です。

なぜなら、騎手Aそのものではなく、別の要因が効いている可能性があるからです。

たとえば、

```text
騎手Aは特定の厩舎の馬に多く乗っている
騎手Aはローカル開催での騎乗が多い
騎手Aは短距離戦に偏っている
騎手Aは人気薄の馬に乗ることが多い
騎手Aは出走数が少なく、たまたま高配当を拾った
```

などです。

この場合、

```text
騎手A
```

が原因に見えて、実際には、

```text
厩舎
競馬場
距離
人気帯
馬質
サンプルサイズ
馬場状態
クラス
```

が効いているかもしれません。

これはかなり危険です。

競馬AIでありがちな自己欺瞞は、

```text
特徴量Aの回収率が高い
だからAは儲かる
```

と短絡することです。

でも、そのAは本当に原因なのか。  
それとも、別の条件の代理変数になっているだけなのか。

ここを疑わないと、バックテストでは勝っても、将来データで崩れます。

# 7. 層別とは何か

見かけの相関を疑うときに使う基本的な方法が、層別です。

層別とは、データをグループに分けて見ることです。

たとえば、さきほどの身長と語彙力の例なら、

```text
年齢別に分ける
```

ということです。

全体で見ると、

```text
身長が高いほど語彙力が高い
```

ように見えた。

でも、年齢別に分けてみると、

```text
同じ年齢の中では、身長と語彙力の関係はほとんどない
```

かもしれません。

つまり、層別によって、

```text
本当に関係があるのか
それともグループ差が相関を作っているだけなのか
```

を見ます。

---

# 8. 層別散布図とは何か

層別散布図とは、グループごとに色や記号を変えた散布図です。

たとえば、全体ではこう見えたとします。

```text
語彙力
 90 |                         *
 80 |                    *
 70 |               *
 60 |          *
 50 |     *
 40 | *
    +-------------------------
        身長
```

これだけ見ると、身長と語彙力には強い正の相関がありそうです。

でも、年齢別に色分けすると、実はこうかもしれません。

```text
語彙力
 90 |                 H  H  H
 80 |              H  H
 70 |          M  M  M
 60 |       M  M
 50 |  L  L  L
 40 |  L
    +-------------------------
        身長

L：低学年
M：中学年
H：高学年
```

この場合、全体として右上がりに見えるのは、

```text
低学年 → 身長低い、語彙力低い
中学年 → 中くらい
高学年 → 身長高い、語彙力高い
```

という年齢差のせいです。

同じ年齢グループの中では、そこまで右上がりではないかもしれません。

これが層別散布図の意味です。

---

# 9. 層別すると関係が消える例

次のようなデータを考えます。

|年齢層|身長|語彙力|
|---|--:|--:|
|低学年|120|40|
|低学年|125|42|
|低学年|130|41|
|中学年|140|60|
|中学年|145|61|
|中学年|150|59|
|高学年|160|80|
|高学年|165|82|
|高学年|170|81|

全体で見ると、

```text
身長が高いほど語彙力が高い
```

ように見えます。

でも、年齢層ごとに見ると、

```text
低学年内：身長が高いから語彙力が高いとは言いにくい
中学年内：ほぼ横ばい
高学年内：ほぼ横ばい
```

です。

つまり、全体の相関は年齢層の違いで作られています。

これが見かけの相関です。

---

# 10. 層別すると関係が逆転することもある

さらに危険なのは、層別すると関係が逆転するケースです。

全体では、

```text
Aの方が良い
```

ように見える。

でも、グループ別に見ると、

```text
各グループではBの方が良い
```

となることがあります。

これを広く、**シンプソンのパラドックス**と呼びます。

統計検定2級で深い証明までは不要ですが、考え方は重要です。

---

# 11. シンプソンのパラドックスの例

治療Aと治療Bの成功率を考えます。

全体ではこうだったとします。

|治療|成功|合計|成功率|
|---|--:|--:|--:|
|A|80|100|80%|
|B|150|200|75%|

全体では、Aの方が成功率が高いです。

```text
A：80%
B：75%
```

だから、

```text
Aの方が良い治療だ
```

と言いたくなります。

でも、重症度で分けるとこうだったとします。

## 軽症

|治療|成功|合計|成功率|
|---|--:|--:|--:|
|A|70|80|87.5%|
|B|90|100|90.0%|

軽症ではBの方が高い。

## 重症

|治療|成功|合計|成功率|
|---|--:|--:|--:|
|A|10|20|50.0%|
|B|60|100|60.0%|

重症でもBの方が高い。

つまり、

```text
軽症でもBが良い
重症でもBが良い
でも全体ではAが良く見える
```

という逆転が起きています。

なぜか。

Aは軽症患者の割合が多く、Bは重症患者の割合が多いからです。

このように、グループ構成が違うと、全体の比較が歪みます。

---

# 12. シンプソンのパラドックスの本質

シンプソンのパラドックスの本質はこれです。

```text
全体で見た関係と、層別して見た関係が一致するとは限らない
```

特に危険なのは、グループごとの構成比が違うときです。

たとえば、

```text
軽症・重症の割合が違う
年齢層の割合が違う
人気帯の割合が違う
競馬場の割合が違う
距離条件の割合が違う
サンプルサイズが違う
```

こういう場合、全体平均や全体回収率だけを見ると、簡単に騙されます。

---

# 13. 競馬データにおけるシンプソン的な罠

競馬では、これはかなり起きやすいです。

たとえば、ある特徴量Xについて、

```text
Xありの馬は回収率が高い
```

とします。

でも、層別すると、

|条件|Xありの回収率|Xなしの回収率|
|---|--:|--:|
|1番人気〜3番人気|Xなしの方が高い||
|4番人気〜8番人気|Xなしの方が高い||
|9番人気以下|Xなしの方が高い||

のように、各人気帯ではXなしの方が高い可能性があります。

それでも全体でXありが高く見えることがあります。

なぜか。

Xありのグループに、たまたま高配当が混ざっていたり、人気帯構成が違ったりするからです。

つまり、

```text
全体ROIが高い
```

は、かなり弱い証拠です。

最低でも、

```text
人気帯
競馬場
距離
馬場
クラス
頭数
期間
```

などで層別して、関係が残るかを見る必要があります。

率直に言うと、競馬AIで「条件別回収率」だけを見てルール化するのは、非常に脆いです。  
それは統計というより、ノイズを名前付きで保存しているだけになりやすい。

---

# 14. 偏相関とは何か

次に、偏相関です。

偏相関とは、第3の変数の影響を取り除いたうえで、2つの変数の関係を見る考え方です。

たとえば、

```text
X：身長
Y：語彙力
Z：年齢
```

とします。

単純な相関では、XとYの関係をそのまま見ます。

```text
身長と語彙力の相関
```

でも、偏相関では、

```text
年齢の影響を取り除いたうえで、身長と語彙力に関係があるか
```

を見ます。

つまり、

```text
Zを固定したときのXとYの関係
```

を見るのが偏相関です。

---

# 15. 偏相関の直感

偏相関は、こう考えると分かりやすいです。

```text
同じ年齢の中で、身長が高い人ほど語彙力が高いのか？
```

これが、年齢を調整した身長と語彙力の関係です。

もし、同じ年齢内では関係がないなら、

```text
身長と語彙力の単純相関は、年齢による見かけの相関だった
```

と考えられます。

---

# 16. 偏相関を図で考える

単純相関：

```text
X と Y の関係をそのまま見る

X ───── Y
```

偏相関：

```text
Z の影響を取り除いてから X と Y の関係を見る

      Z
     / \
    ↓   ↓
X ───── Y

Zの影響を調整する
```

偏相関の目的は、

```text
第3の変数の影響を除いても、まだXとYに関係が残るか
```

を見ることです。

---

# 17. 偏相関の注意点

偏相関も万能ではありません。

第3の変数を1つ調整すれば、それで因果が証明されるわけではありません。

なぜなら、

```text
他にも未観測の交絡変数があるかもしれない
変数の関係が非線形かもしれない
測定誤差があるかもしれない
そもそも調整してはいけない変数を調整しているかもしれない
```

からです。

ここはかなり重要です。

```text
偏相関を取ったから因果が分かった
```

ではありません。

正しくは、

```text
特定の変数の線形影響を調整したうえでの相関を見た
```

です。

---

# 18. 交絡変数とは何か

見かけの相関を作る第3の変数は、よく**交絡変数**と呼ばれます。

交絡変数とは、

```text
原因っぽく見ている変数と、結果っぽく見ている変数の両方に関係する変数
```

です。

たとえば、

```text
X：アイス売上
Y：水難事故
Z：気温
```

なら、気温Zは交絡変数です。

```text
気温が高いとアイスが売れる
気温が高いと水遊びが増える
```

つまり、XにもYにも関係しています。

---

# 19. 交絡を疑うチェックリスト

相関を見たら、必ずこう考えるべきです。

```text
1. XとYの両方に影響する第3の変数はないか？
2. グループ構成が違っていないか？
3. 層別しても同じ関係が残るか？
4. 外れ値1つで相関が作られていないか？
5. 時間順序は合っているか？
6. サンプルサイズは十分か？
7. 測定方法に偏りはないか？
```

このチェックをせずに、

```text
相関があるから意味がある
```

と判断するのは危険です。

---

# 20. 層別と偏相関の違い

ここで整理します。

|方法|何をするか|強み|弱み|
|---|---|---|---|
|層別|グループに分けて見る|直感的で分かりやすい|グループ数が増えると複雑|
|層別散布図|グループ別に散布図を見る|見かけの相関に気づきやすい|変数が多いと見づらい|
|偏相関|第3の変数の影響を調整して相関を見る|数値で調整後の関係を見られる|線形関係前提が強い。因果は証明しない|

実務では、最初に層別散布図で見て、必要なら偏相関や回帰分析で調整するのが自然です。

```text
可視化 → 層別 → 数値化
```

この順番が安全です。

---

# 21. 相関行列とは何か

ついでに、相関行列もここで触れておきます。

相関行列とは、複数の変数同士の相関係数を一覧にした表です。

たとえば、

||勉強時間|睡眠時間|点数|
|---|--:|--:|--:|
|勉強時間|1.00|-0.20|0.75|
|睡眠時間|-0.20|1.00|0.30|
|点数|0.75|0.30|1.00|

この表から、

```text
勉強時間と点数：0.75で強めの正の相関
睡眠時間と点数：0.30で弱めの正の相関
勉強時間と睡眠時間：-0.20で弱い負の相関
```

のように読みます。

---

# 22. 相関行列の注意点

相関行列は便利ですが、危険もあります。

```text
数字だけ見て、散布図を見なくなる
```

からです。

相関行列では、

```text
外れ値
非線形関係
グループ混在
見かけの相関
```

が見えません。

だから、相関行列はあくまで探索の入口です。

高い相関を見つけたら、必ず散布図を見るべきです。

---

# 23. 回帰分析との接続

偏相関や層別の話は、重回帰分析につながります。

重回帰分析では、

```text
Yを複数のXで説明する
```

ことをします。

たとえば、

```text
点数 = 勉強時間 + 睡眠時間 + 年齢 + ...
```

のように、複数の説明変数を同時に入れます。

これはある意味で、

```text
他の変数を調整したうえで、特定の変数の効果を見る
```

という考え方です。

もちろん、重回帰でも因果が自動的に分かるわけではありません。

でも、

```text
単純相関よりは、交絡の影響を考慮しやすい
```

という意味で重要です。


です。

```text
第3の変数の影響を取り除いたうえで、2つの変数の相関を見る指標は何か？
```

答えは、

```text
偏相関係数
```

です。

```text
相関係数が0.9だった。
このとき必ず言えることはどれか？
```

答えは、**C. 強い直線的関係がある**です。

# 26. 今回の最重要ポイント

今回一番大事なのはこれです。

```text
全体で見える相関は、真の関係とは限らない。
```

特に、

```text
第3の変数
グループ構成の違い
外れ値
非線形関係
サンプルサイズ不足
```

によって、相関は簡単に歪みます。

そして、もう一つ重要なのはこれです。

```text
相関を見たら、層別しても関係が残るかを疑う。
```

これは試験対策だけではなく、あなたの競馬AIでもかなり重要です。

全体ROI、全体相関、全体平均だけで判断すると、ノイズを規則だと誤認しやすいです。

# 27. 確認問題

## 問題1

アイスの売上と水難事故数に正の相関があった。

```text
A. 目的変数
B. 交絡変数
C. 標準偏差
```

答え：**B. 交絡変数**

答え：**B. 偏相関**
