# 第17回：期待値・分散

## 今日のゴール

今日のゴールはこれです。

> 期待値とは、確率を考慮した平均。  
> 分散とは、期待値からのばらつき。

もう少し言うと、

```text
期待値 = 確率分布の中心
分散 = 確率分布の散らばり
```

です。

前にやった普通の平均・分散は、すでに観測されたデータに対するものでした。

今回の期待値・分散は、**確率変数に対する平均・分散**です。

---

# 1. 普通の平均と期待値の違い

まず、普通の平均を思い出します。

たとえば、次のデータがある。

```text
10, 20, 30
```

平均は、

```text
(10 + 20 + 30) / 3 = 20
```

です。

これは、すでに観測されたデータの平均です。

---

一方、期待値は、

> まだ結果が決まっていない確率変数について、平均的にどの値になりそうか

を表します。

たとえば、くじを1回引く。

|結果|金額|確率|
|---|--:|--:|
|当たり|100円|0.3|
|外れ|0円|0.7|

このとき、期待値は、

```text
100 × 0.3 + 0 × 0.7 = 30
```

です。

つまり、このくじの期待値は30円です。

---

# 2. 期待値とは何か

期待値とは、

> 確率で重みづけした平均

です。

値が大きくても、起こる確率が低ければ影響は小さい。  
値が小さくても、起こる確率が高ければ影響は大きい。

だから、

```text
値 × その値が出る確率
```

を全部足します。

---

## 期待値の公式

離散型確率変数 `X` が値 `x` を取る確率を `P(X=x)` とすると、

```text
E(X) = Σ x P(X = x)
```

です。

ただし、記号で見ると硬いので、最初はこう覚えればいいです。

```text
期待値 = 値 × 確率 を全部足す
```

---

# 3. サイコロの期待値

サイコロを1回振る。  
出る目を `X` とします。

|X|確率|
|--:|--:|
|1|1/6|
|2|1/6|
|3|1/6|
|4|1/6|
|5|1/6|
|6|1/6|

期待値は、

```text
E(X)
= 1 × 1/6
+ 2 × 1/6
+ 3 × 1/6
+ 4 × 1/6
+ 5 × 1/6
+ 6 × 1/6
```

まとめると、

```text
E(X)
= (1 + 2 + 3 + 4 + 5 + 6) / 6
= 21 / 6
= 3.5
```

です。

---

## ここでの注意

サイコロの期待値は3.5です。

でも、サイコロを1回振って3.5が出ることはありません。

ここは大事です。

期待値は、

```text
実際に出る値
```

とは限りません。

期待値は、

```text
長期的な平均
```

です。

サイコロを何万回も振ると、平均は3.5に近づいていきます。

---

# 4. 期待値は「1回あたりの平均的な値」

期待値を一番自然に理解するなら、

> 同じ試行を何度も繰り返したときの、1回あたりの平均

です。

サイコロを6000回振ったら、理想的には各目が1000回くらい出ます。

合計は、

```text
1×1000 + 2×1000 + 3×1000 + 4×1000 + 5×1000 + 6×1000
= 21000
```

1回あたりは、

```text
21000 / 6000 = 3.5
```

です。

これが期待値です。

---

# 5. 期待値は「期待していい値」ではない

日本語の「期待値」という名前が少し悪いです。

期待値は、

```text
期待してよい値
```

ではありません。

むしろ、

```text
長期平均
```

です。

たとえば、宝くじの期待値が100円だったとしても、1回買って100円返ってくるという意味ではありません。

```text
当たれば大きい
外れれば0
長期平均では100円くらい
```

という意味です。

ここは競馬・投資の文脈でもかなり重要です。

---

# 6. 払戻金の期待値

馬券で考えます。

ある馬券を100円買います。

的中したら500円払い戻し。  
外れたら0円。  
的中確率を `p = 0.25` とします。

払戻金を `X` とすると、

|X|確率|
|--:|--:|
|500|0.25|
|0|0.75|

期待値は、

```text
E(X)
= 500 × 0.25 + 0 × 0.75
= 125
```

です。

つまり、払戻金の期待値は125円。

100円買って期待払戻が125円なら、期待利益は、

```text
125 - 100 = 25
```

です。

期待ROIで言うと、

```text
25 / 100 = 0.25
```

つまり25%です。

---

## ただし、1回で勝てるとは限らない

この馬券は期待値がプラスです。

でも、1回買ったら75%の確率で外れます。

```text
期待値プラス
=
短期で勝つ
```

ではありません。

期待値は、長期平均の話です。

ここを混同すると、期待値を信じて破綻します。

---

# 7. 期待値の線形性

期待値には、とても重要な性質があります。

```text
E(aX + b) = aE(X) + b
```

また、

```text
E(X + Y) = E(X) + E(Y)
```

です。

これはかなり強い性質です。

---

## 意味

たとえば、ある確率変数 `X` の期待値が10だとします。

```text
E(X) = 10
```

このとき、

```text
Y = 2X + 5
```

なら、

```text
E(Y) = 2E(X) + 5 = 25
```

です。

期待値は、足し算や定数倍と相性がよいです。

---

## 期待値は独立でなくても足せる

ここは重要です。

```text
E(X + Y) = E(X) + E(Y)
```

は、XとYが独立でなくても成り立ちます。

独立が必要になるのは、後で出る分散の足し算です。

これは前に話した「分散は足せるが標準偏差は足せない」にもつながります。

---

# 8. 分散とは何か

次に分散です。

分散とは、

> 確率変数が期待値からどれくらいズレるか

を表す値です。

期待値だけでは、分布のリスクや不安定さは分かりません。

---

## 例

次の2つのくじを考えます。

## くじA

|X|確率|
|--:|--:|
|50|1|

必ず50円もらえる。

期待値は、

```text
E(X) = 50
```

です。

---

## くじB

|X|確率|
|--:|--:|
|100|0.5|
|0|0.5|

期待値は、

```text
E(X) = 100 × 0.5 + 0 × 0.5 = 50
```

です。

---

## 期待値は同じ

どちらも期待値は50円です。

でも性質は全然違います。

```text
くじA：毎回50円
くじB：100円か0円
```

くじBの方がばらつきが大きい。

この違いを見るのが分散です。

---

# 9. 分散の公式

分散は、期待値からのズレを二乗して、その期待値を取ります。

```text
V(X) = E{(X - μ)^2}
```

ここで、

```text
μ = E(X)
```

です。

つまり、

```text
分散 = 期待値からのズレ²の平均
```

です。

---

## なぜ二乗するのか

理由は大きく2つです。

1つ目は、ズレのプラスとマイナスが打ち消し合わないようにするため。

たとえば、

```text
+10
-10
```

をそのまま足すと0になります。

でも、どちらも平均から10離れています。

だから二乗します。

```text
(+10)^2 = 100
(-10)^2 = 100
```

---

2つ目は、大きなズレをより重く見るためです。

```text
ズレ 2 → 二乗で4
ズレ 10 → 二乗で100
```

つまり、分散は大きなズレに敏感です。

---

# 10. くじAの分散

くじAは必ず50円です。

|X|確率|
|--:|--:|
|50|1|

期待値は、

```text
μ = 50
```

です。

ズレは、

```text
X - μ = 50 - 50 = 0
```

なので、

```text
V(X) = 0
```

です。

必ず同じ値なので、ばらつきはありません。

---

# 11. くじBの分散

くじBは、

|X|確率|
|--:|--:|
|100|0.5|
|0|0.5|

期待値は、

```text
μ = 50
```

です。

それぞれのズレを見ます。

|X|確率|X - μ|(X - μ)^2|
|--:|--:|--:|--:|
|100|0.5|50|2500|
|0|0.5|-50|2500|

分散は、

```text
V(X)
= 2500 × 0.5 + 2500 × 0.5
= 2500
```

です。

くじAは分散0。  
くじBは分散2500。

期待値は同じでも、リスクが違うことが分かります。

---

# 12. 分散の便利公式

分散には、計算でよく使う公式があります。

```text
V(X) = E(X^2) - {E(X)}^2
```

これは重要です。

意味は、

```text
X²の期待値 - Xの期待値の二乗
```

です。

---

## くじBで確認

くじBでは、

|X|確率|
|--:|--:|
|100|0.5|
|0|0.5|

まず、

```text
E(X) = 50
```

です。

次に、

```text
E(X²)
= 100² × 0.5 + 0² × 0.5
= 10000 × 0.5
= 5000
```

だから、

```text
V(X)
= E(X²) - {E(X)}²
= 5000 - 50²
= 5000 - 2500
= 2500
```

同じ結果になります。

---

# 13. 標準偏差とは何か

分散は二乗した単位になります。

たとえば、金額なら、

```text
円²
```

のような変な単位になります。

だから、平方根を取って元の単位に戻します。

これが標準偏差です。

```text
SD(X) = √V(X)
```

くじBでは、

```text
V(X) = 2500
```

なので、

```text
SD(X) = √2500 = 50
```

です。

標準偏差は50円。

つまり、期待値50円の周りに、だいたい50円くらいブレるイメージです。

---

# 14. 分散と標準偏差の違い

|指標|意味|単位|使い道|
|---|---|---|---|
|分散|ズレの二乗平均|元の単位²|数学的に扱いやすい|
|標準偏差|分散の平方根|元の単位|直感的に解釈しやすい|

分散は計算に強い。  
標準偏差は解釈に強い。

前に話したように、標準偏差は平方根を取るので、足し算などの構造が壊れます。

だから理論上は分散をよく使います。

---

# 15. 分散の性質

分散には重要な性質があります。

```text
V(aX + b) = a²V(X)
```

です。

ここで大事なのは、

```text
bを足しても分散は変わらない
a倍すると分散はa²倍になる
```

ということです。

---

## なぜ b を足しても分散は変わらないのか

たとえば、全員の点数に10点加点する。

```text
50, 60, 70
↓
60, 70, 80
```

平均も10上がります。

でも、平均との差の大きさは変わりません。

だから分散は変わりません。

---

## なぜ a倍すると分散はa²倍なのか

点数を2倍にすると、平均との差も2倍になります。

でも分散はズレを二乗するので、

```text
2倍のズレ → 二乗で4倍
```

になります。

だから、

```text
V(2X) = 4V(X)
```

です。

---

# 16. 期待値と分散の違い

期待値と分散は、役割が違います。

|指標|見ているもの|
|---|---|
|期待値|中心・長期平均|
|分散|ばらつき・リスク|

期待値だけ見ても不十分です。

特に投資・競馬・ログ異常検知では、

```text
平均は良いが、ばらつきが大きすぎる
```

ということが普通にあります。

---

## 競馬での注意

期待値がプラスでも、分散が巨大なら、資金管理を間違えると簡単に破綻します。

たとえば、

```text
的中率1%
的中時払戻20000円
外れ時0円
```

のような馬券は、期待値だけ見れば魅力的に見えることがあります。

でも、外れが長く続くので、分散が非常に大きい。

つまり、

```text
期待値プラス
=
安全
```

ではありません。

ここは本当に危ないです。

---

# 17. 0/1変数の期待値と分散

0/1の確率変数はとても重要です。

たとえば、

```text
X = 1：成功
X = 0：失敗
```

成功確率を `p` とします。

|X|確率|
|--:|--:|
|1|p|
|0|1-p|

---

## 期待値

```text
E(X)
= 1 × p + 0 × (1-p)
= p
```

つまり、0/1変数の期待値は成功確率そのものです。

---

## 分散

便利公式を使います。

```text
V(X) = E(X²) - {E(X)}²
```

0/1変数では、

```text
X² = X
```

です。

なぜなら、

```text
0² = 0
1² = 1
```

だからです。

したがって、

```text
E(X²) = E(X) = p
```

なので、

```text
V(X)
= p - p²
= p(1-p)
```

です。

これは次の二項分布の土台になります。

---

# 18. 0/1変数の分散の感覚

0/1変数の分散は、

```text
p(1-p)
```

です。

たとえば、

|p|分散 p(1-p)|
|--:|--:|
|0.1|0.09|
|0.5|0.25|
|0.9|0.09|

分散が最大になるのは `p = 0.5` のときです。

なぜか。

```text
p = 0.5
```

だと、成功か失敗かが一番読みにくいからです。

逆に、

```text
p = 0.99
```

なら、ほぼ成功なのでブレは小さい。

```text
p = 0.01
```

なら、ほぼ失敗なのでブレは小さい。

0/1変数のばらつきは、成功確率が半々のとき最大になります。

---

# 19. 期待値・分散の計算例

次の確率変数 `X` を考えます。

|X|確率|
|--:|--:|
|0|0.2|
|10|0.5|
|20|0.3|

---

## 期待値

```text
E(X)
= 0 × 0.2 + 10 × 0.5 + 20 × 0.3
= 0 + 5 + 6
= 11
```

期待値は11です。

---

## 分散

まず `E(X²)` を求めます。

```text
E(X²)
= 0² × 0.2 + 10² × 0.5 + 20² × 0.3
= 0 + 100 × 0.5 + 400 × 0.3
= 0 + 50 + 120
= 170
```

次に、

```text
{E(X)}² = 11² = 121
```

したがって、

```text
V(X)
= E(X²) - {E(X)}²
= 170 - 121
= 49
```

標準偏差は、

```text
√49 = 7
```

です。

---

# 20. 期待値・分散の計算手順

離散型確率変数では、この手順で計算すると安定します。

```text
1. X と確率の表を作る
2. x × 確率 を計算する
3. それを足して E(X) を出す
4. x² × 確率 を計算する
5. それを足して E(X²) を出す
6. V(X) = E(X²) - {E(X)}² を使う
7. 必要なら √V(X) で標準偏差を出す
```

この手順は試験でもそのまま使えます。

---

次の確率変数 `X` の期待値を求めよ。

|X|確率|
|--:|--:|
|0|0.4|
|10|0.6|

---

### 解答

```text
E(X)
= 0 × 0.4 + 10 × 0.6
= 6
```

答えは、

```text
6
```

です。

---

## 問題2

問題1の分散を求めよ。

|X|確率|
|--:|--:|
|0|0.4|
|10|0.6|

---

### 解答

まず、

```text
E(X) = 6
```

です。

次に、

```text
E(X²)
= 0² × 0.4 + 10² × 0.6
= 0 + 100 × 0.6
= 60
```

したがって、

```text
V(X)
= E(X²) - {E(X)}²
= 60 - 6²
= 60 - 36
= 24
```

答えは、

```text
24
```

です。

## 問題3

成功確率 `p = 0.2` の0/1変数 `X` がある。

期待値と分散を求めよ。

---

### 解答

0/1変数では、

```text
E(X) = p
V(X) = p(1-p)
```

です。

したがって、

```text
E(X) = 0.2
```

```text
V(X) = 0.2 × 0.8 = 0.16
```

答えは、

```text
期待値 0.2
分散 0.16
```

です。

## 問題4

サイコロを1回振る。  
出る目を `X` とする。

期待値を求めよ。

---

### 解答

```text
E(X)
= (1+2+3+4+5+6)/6
= 21/6
= 3.5
```

答えは、

```text
3.5
```

です。

## 問題5

ある馬券を100円買う。  
的中したら800円払い戻し、外れたら0円。  
的中確率は0.1とする。

払戻金 `X` の期待値と、期待利益を求めよ。

### 解答

払戻金は、

|0|0.9|

```text
E(X)
= 800 × 0.1 + 0 × 0.9
= 80
```

払戻金の期待値は80円。

購入費用は100円なので、期待利益は、

```text

```

# 23. 今日の盲点

今日の最大の盲点はこれです。

```text
期待値が同じでも、リスクは同じではない
```

くじAとくじBのように、

```text
毎回50円
0円か100円
```

は、期待値は同じでも分散が違います。

もう1つは、

```text
期待値プラス = 短期で勝てる
```

ではないことです。

期待値は長期平均です。  
短期では分散に支配されます。

あなたの競馬AI文脈で言えば、期待値を見ているのは正しい。  
でも、分散・ドローダウン・資金制約を無視した期待値戦略は、数学的に勝っていても運用で死にます。
