# 第16回：確率変数

## 今日のゴール

今日のゴールはこれです。

> 確率変数とは、  
> 偶然によって値が決まる変数である。

たとえば、サイコロを1回振る。

出る目は、振る前には分かりません。

でも、出る値の候補は、

```text
1, 2, 3, 4, 5, 6
```

です。

この「サイコロの出る目」を `X` と置くと、`X` は確率変数です。

---

# 1. 変数とは何か

まず普通の変数から確認します。

変数とは、

> 値を入れる箱

のようなものです。

たとえば、

```text
x = 5
```

なら、`x` には5という値が入っています。

でも、確率変数は少し違います。

---

# 2. 確率変数とは何か

確率変数は、

> 偶然によって値が決まる変数

です。

たとえば、サイコロを1回振るとします。

```text
X = サイコロの出る目
```

と置きます。

このとき、`X` はまだ固定された値ではありません。

サイコロを振る前は、

```text
X = 1 かもしれない
X = 2 かもしれない
X = 3 かもしれない
X = 4 かもしれない
X = 5 かもしれない
X = 6 かもしれない
```

という状態です。

そして、それぞれの値に確率があります。

|X|確率|
|--:|--:|
|1|1/6|
|2|1/6|
|3|1/6|
|4|1/6|
|5|1/6|
|6|1/6|

このように、値が確率的に決まるものが確率変数です。

---

# 3. 確率変数は「事象を数値に変換するもの」

ここはかなり大事です。

確率変数は、ただのランダムな数字ではありません。

> 起きた結果を、数値に対応させるもの

です。

たとえば、コインを1回投げる。

結果は、

```text
表
裏
```

です。

このままだと、平均や分散を計算しづらい。

そこで、こう決めます。

```text
表なら X = 1
裏なら X = 0
```

すると、結果を数値として扱えます。

|結果|X|
|---|--:|
|表|1|
|裏|0|

これが確率変数です。

つまり、確率変数とは、

> 偶然の結果を、分析できる数値に変換するルール

でもあります。

---

# 4. なぜ確率変数が必要なのか

確率変数を使う理由は単純です。

> 偶然の出来事を、計算できる形にするため

です。

たとえば、

```text
コインが表か裏か
```

だけでは、平均を計算しづらい。

でも、

```text
表 = 1
裏 = 0
```

とすれば、

```text
表が出る確率
平均的にどれくらい1に近いか
分散はどれくらいか
```

を計算できます。

統計では、現実の出来事を数値化して扱います。

```text
当たり / 外れ
合格 / 不合格
クリックした / しない
馬券が的中した / しない
病気 / 病気でない
```

こういうものを、

```text
1 / 0
```

に変換すると、確率モデルで扱いやすくなります。

---

# 5. 確率変数と確率分布

確率変数を考えると、自然に確率分布が出てきます。

確率分布とは、

> 確率変数が、どの値をどれくらいの確率で取るか

を表したものです。

たとえば、サイコロの出る目 `X` は、

|X|確率|
|--:|--:|
|1|1/6|
|2|1/6|
|3|1/6|
|4|1/6|
|5|1/6|
|6|1/6|

という確率分布を持ちます。

つまり、

```text
確率変数：値が確率的に決まるもの
確率分布：その値ごとの確率を整理したもの
```

です。

---

# 6. 確率変数の書き方

確率変数は、よく大文字で書きます。

```text
X, Y, Z
```

などです。

実際に取った値は、小文字で書くことが多いです。

```text
X = x
```

これは、

> 確率変数Xが、具体的な値xを取る

という意味です。

たとえば、

```text
P(X = 3)
```

は、

> 確率変数Xが3になる確率

という意味です。

サイコロなら、

```text
P(X = 3) = 1/6
```

です。

---

# 7. 離散型確率変数

確率変数には大きく2種類あります。

まず、離散型です。

離散型確率変数とは、

> 取る値が飛び飛びの確率変数

です。

例：

```text
サイコロの目：1, 2, 3, 4, 5, 6
コインを10回投げたときの表の回数：0, 1, 2, ..., 10
1日に来る問い合わせ件数：0, 1, 2, 3, ...
馬券が当たったか：0, 1
```

基本的に、「数えられる値」です。

---

## 離散型では P(X = x) を考える

離散型確率変数では、

```text
P(X = x)
```

を考えます。

たとえば、

```text
X = サイコロの出る目
```

なら、

```text
P(X = 1) = 1/6
P(X = 2) = 1/6
...
P(X = 6) = 1/6
```

です。

---

# 8. 連続型確率変数

次に連続型です。

連続型確率変数とは、

> 取る値が連続している確率変数

です。

例：

```text
身長
体重
時間
気温
売上金額
レースタイム
APIレスポンスタイム
```

これらは、細かく考えると無限に値を取れます。

たとえば身長なら、

```text
170cm
170.1cm
170.01cm
170.001cm
```

のように、どこまでも細かくできます。

---

## 連続型では P(X = x) は基本的に0

ここが最初の大きな壁です。

連続型確率変数では、

```text
P(X = 170)
```

のようなピンポイントの確率は、基本的に0です。

なぜなら、連続型では取りうる値が無限にあるからです。

だから、連続型では範囲で確率を考えます。

```text
P(160 ≦ X ≦ 170)
```

のようにします。

これは、

> Xが160以上170以下になる確率

です。

---

# 9. 離散型と連続型の違い

整理するとこうです。

|種類|値|確率の考え方|例|
|---|---|---|---|
|離散型|飛び飛び|`P(X = x)`|サイコロ、回数、人数|
|連続型|なめらか|`P(a ≦ X ≦ b)`|身長、時間、気温|

この違いは、第19回の正規分布でかなり重要になります。

正規分布は連続型なので、

```text
ちょうど170cmの確率
```

ではなく、

```text
160cm以上170cm以下の確率
```

を考えます。

---

# 10. 確率変数の例1：コイン

コインを1回投げる。

```text
表なら X = 1
裏なら X = 0
```

とします。

公平なコインなら、

|X|確率|
|--:|--:|
|1|1/2|
|0|1/2|

です。

この `X` は確率変数です。

そしてこの分布は、次回以降に出る **ベルヌーイ分布** の基本形です。

---

# 11. 確率変数の例2：サイコロ

サイコロを1回振る。

```text
X = 出た目
```

とします。

|X|確率|
|--:|--:|
|1|1/6|
|2|1/6|
|3|1/6|
|4|1/6|
|5|1/6|
|6|1/6|

この `X` は離散型確率変数です。

---

## P(X ≧ 4) の意味

サイコロで、

```text
P(X ≧ 4)
```

を考えます。

これは、

```text
4以上の目が出る確率
```

です。

該当するのは、

```text
4, 5, 6
```

なので、

```text
P(X ≧ 4) = 3/6 = 1/2
```

です。

---

# 12. 確率変数の例3：当たり外れ

あるくじを1回引く。

```text
当たりなら X = 1
外れなら X = 0
```

とします。

当たり確率が0.2なら、

|X|確率|
|--:|--:|
|1|0.2|
|0|0.8|

です。

このような0/1の確率変数は、実務でも非常によく出ます。

```text
クリックしたか
購入したか
合格したか
離脱したか
馬券が的中したか
```

こういうものは全部0/1で表せます。

---

# 13. 確率変数の例4：払戻金

あなたの文脈に寄せると、馬券の払戻金も確率変数として見られます。

たとえば、ある馬券を100円買う。

```text
的中したら 500円戻る
外れたら 0円
```

とします。

この払戻金を `X` とすると、

|X|確率|
|--:|--:|
|500|p|
|0|1 - p|

です。

この `X` は確率変数です。

ここから期待値を計算すると、

```text
E(X) = 500p + 0 × (1 - p)
```

になります。

次回の期待値でかなり重要になります。

---

# 14. 確率変数とデータの違い

ここは少し抽象的ですが、重要です。

確率変数は、まだ結果が決まる前の考え方です。

データは、実際に観測された後の値です。

たとえば、サイコロを振る前は、

```text
X = サイコロの出る目
```

という確率変数です。

振ったあとに3が出たら、

```text
観測値 = 3
```

です。

つまり、

|段階|内容|
|---|---|
|振る前|確率変数X|
|振った後|観測値x|

統計では、この区別が後で効いてきます。

---

# 15. 確率変数の関数

確率変数は、計算して別の値に変換できます。

たとえば、サイコロの出る目を `X` とします。

```text
Y = X²
```

と置くと、`Y` も確率変数です。

なぜなら、`X` が偶然決まるので、`Y` も偶然決まるからです。

|X|Y = X²|
|--:|--:|
|1|1|
|2|4|
|3|9|
|4|16|
|5|25|
|6|36|

この考え方は、分散で重要になります。

分散では、

```text
(X - 平均)²
```

を扱うからです。

---

# 16. 確率変数を使うと何ができるか

確率変数を使うと、次のようなことができます。

```text
平均的にどれくらいの値になるか
どれくらいばらつくか
ある範囲に入る確率はどれくらいか
複数回やったときにどうなるか
```

つまり、次回以降の、

```text
期待値
分散
二項分布
正規分布
標本分布
推定
検定
```

全部の土台になります。

---

# 17. 記号に慣れる

ここで、最低限の記号を整理します。

|記号|意味|
|---|---|
|`X`|確率変数|
|`x`|Xが取る具体的な値|
|`P(X = x)`|Xがxになる確率|
|`P(X ≦ x)`|Xがx以下になる確率|
|`P(a ≦ X ≦ b)`|Xがa以上b以下になる確率|

---

## 例

```text
X = サイコロの出る目
```

なら、

```text
P(X = 1) = 1/6
```

```text
P(X ≦ 3) = 3/6 = 1/2
```

```text
P(2 ≦ X ≦ 5) = 4/6 = 2/3
```

です。

# 18. 練習問題

## 問題1

サイコロを1回振る。  
`X` を出た目とする。

`X` は確率変数か？

---

## 解答

はい、確率変数です。

理由は、サイコロを振る前には値が決まっておらず、偶然によって、

```text
1, 2, 3, 4, 5, 6
```

のどれかに決まるからです。

---

## 問題2

コインを1回投げる。  
表なら `X = 1`、裏なら `X = 0` とする。

公平なコインのとき、`P(X = 1)` は？

---

## 解答

表が出る確率なので、

```text
P(X = 1) = 1/2
```

です。

---

## 問題3

サイコロを1回振る。  
`X` を出た目とする。

`P(X ≧ 5)` を求めよ。

---

## 解答

5以上の目は、

```text
5, 6
```

の2通りです。

全体は6通りなので、

```text
P(X ≧ 5) = 2/6 = 1/3
```

です。

---

## 問題4

あるくじの当たり確率は0.3である。  
当たりなら `X = 1`、外れなら `X = 0` とする。

`P(X = 0)` を求めよ。

---

## 解答

当たり確率が0.3なので、外れ確率は、

```text
1 - 0.3 = 0.7
```

です。

したがって、

```text
P(X = 0) = 0.7
```

です。

---

## 問題5

身長を `X` とする。  
この `X` は離散型か、連続型か？

---

## 解答

連続型です。

身長は、

```text
170cm
170.1cm
170.01cm
```

のように細かく考えられるからです。

したがって、身長では、

```text
P(X = 170)
```

のようなピンポイントの確率ではなく、

```text
P(160 ≦ X ≦ 170)
```

のように範囲で確率を考えます。

# 20. 今日の盲点

今回の最大の盲点はこれです。

```text
確率変数は、ただの変数ではなく、偶然の結果を数値化するルールである
```

たとえば、コインの表裏そのものは数値ではありません。

でも、

```text
表 = 1
裏 = 0
```

と決めることで、計算できる対象になります。

もう1つ危ないのは、

```text
離散型と連続型をごちゃ混ぜにすること
```

です。

離散型では、

```text
P(X = x)
```

を考えます。

連続型では、基本的に、

```text
P(a ≦ X ≦ b)
```

を考えます。

ここを今のうちに分けておくと、正規分布がかなり楽になります。
