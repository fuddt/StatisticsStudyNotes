# 第14回：標本分布

## 今日のゴール

今日のゴールはこれ。

> 標本から計算した平均や割合は、毎回ブレる。  
> そのブレ方にも確率分布がある。  
> それが標本分布。

ここを理解すると、次の単元がつながる。

```text
標本分布
↓
推定
↓
仮説検定
↓
t分布・カイ二乗分布・F分布
```

逆に言うと、標本分布を曖昧にしたまま推定・検定へ進むと、公式暗記ゲーになる。  
それは危ない。

---

# 1. そもそも標本とは何か

まず母集団と標本を整理する。

## 母集団

母集団とは、

> 本当に知りたい対象全体

のこと。

例：

```text
全国の高校3年生全員
ある商品の購入者全員
JRAの全レース
あるWebサービスの全ユーザー
```

---

## 標本

標本とは、

> 母集団から一部だけ取り出したデータ

のこと。

例：

```text
全国の高校3年生から選んだ1000人
購入者のうちアンケートに答えた500人
JRA全レースのうち2025年のレース
Webユーザーのうち直近1週間のアクセスログ
```

---

## なぜ標本を使うのか

本当は母集団全体を調べたい。

でも現実には、

```text
人数が多すぎる
時間がかかりすぎる
お金がかかりすぎる
そもそも全部は観測できない
```

だから、一部の標本から母集団を推測する。

これが統計的推測の基本。

---

# 2. 標本平均とは何か

たとえば、全国の高校3年生の平均身長を知りたいとする。

でも全員を測るのは難しい。

そこで、100人を選んで平均身長を計算する。

この平均を **標本平均** という。

記号ではこう書く。

```text
標本平均 = X̄
```

読み方は「エックスバー」。

---

# 3. 標本平均は毎回変わる

ここが標本分布の核心。

母集団から100人を選んで平均身長を計算する。

1回目：

```text
X̄ = 170.2cm
```

2回目：

```text
X̄ = 169.7cm
```

3回目：

```text
X̄ = 170.5cm
```

同じ母集団から取っているのに、標本平均は毎回少し変わる。

なぜか？

選ばれる人が毎回違うから。

---

# 4. 標本平均も確率変数である

ここがかなり大事。

1人の身長 `X` が確率変数なら、  
100人の平均である `X̄` も確率変数になる。

つまり、

> 標本平均は、標本の取り方によって値が変わる

ということ。

だから標本平均にも分布がある。

これが **標本平均の分布**。

---

# 5. 標本分布とは何か

標本分布とは、

> 標本から計算される統計量の確率分布

です。

ここでいう統計量とは、標本から計算される値のこと。

例：

```text
標本平均
標本分散
標本比率
最大値
中央値
相関係数
```

これらは標本の取り方によって変わる。

だから、それぞれに分布がある。

---

## 重要な言い換え

かなり噛み砕くと、標本分布とはこう。

```text
標本を取る
↓
平均などを計算する
↓
もう一回標本を取る
↓
また平均を計算する
↓
これを何度も繰り返す
↓
平均たちがどんなふうに散らばるかを見る
```

この「平均たちの散らばり」が標本分布。

---

# 6. データそのものの分布と標本分布は違う

ここはよく混乱する。

## データそのものの分布

たとえば、100人の身長データがある。

```text
165, 172, 168, 180, 159, ...
```

この分布は、

> 個人の身長がどう散らばっているか

を表す。

---

## 標本平均の分布

一方、100人ずつ何回も標本を取って、その平均を並べる。

```text
170.2, 169.7, 170.5, 169.9, ...
```

これは、

> 標本平均がどう散らばるか

を表す。

---

## 違い

|種類|何の分布か|
|---|---|
|元データの分布|個々のデータの分布|
|標本分布|標本から計算した統計量の分布|

ここを混ぜると、推定と検定で事故る。

---

# 7. 標本平均の期待値

母集団の平均を `μ` とする。

このとき、標本平均 `X̄` の期待値は、

```text
E(X̄) = μ
```

です。

意味は、

> 標本平均は毎回ブレるけど、平均的には母平均μの周りにいる

ということ。

これは直感的にも自然。

ちゃんとランダムに標本を取れば、標本平均は母平均を中心にブレる。

---

# 8. 標本平均の分散

母集団の分散を `σ²` とする。  
標本サイズを `n` とする。

このとき、標本平均 `X̄` の分散は、

```text
V(X̄) = σ² / n
```

です。

かなり重要。

意味は、

> 標本サイズnが大きくなるほど、標本平均のブレは小さくなる

ということ。

---

## 標本平均の標準偏差

分散が、

```text
σ² / n
```

なら、標準偏差は、

```text
σ / √n
```

です。

この `σ / √n` を **標準誤差** という。

---

# 9. 標準誤差とは何か

標準誤差は、

> 標本平均がどれくらいブレるか

を表す値。

標準偏差と似ているけど、意味が違う。

|用語|何のばらつきか|
|---|---|
|標準偏差|個々のデータのばらつき|
|標準誤差|標本平均のばらつき|

---

## 例

母集団の標準偏差が `σ = 20` だとする。

標本サイズが `n = 100` なら、

```text
標準誤差 = σ / √n
         = 20 / √100
         = 20 / 10
         = 2
```

つまり、個々のデータは標準偏差20くらいバラつく。

でも、100人の平均は標準誤差2くらいのブレに収まる。

---

# 10. ここが統計の強さ

個人のデータはかなりバラつく。

でも、平均を取るとブレが小さくなる。

しかも標本サイズを増やすと、

```text
標準誤差 = σ / √n
```

なので、さらにブレが小さくなる。

ただし、ここで甘く見てはいけない。

nを4倍にしても、標準誤差は半分にしかならない。

なぜなら、

```text
√4 = 2
```

だから。

つまり、精度を2倍にしたければ、データ数は4倍必要。

これはかなり実務的にも重要。

---

# 11. 中心極限定理

ここで最重要概念が出る。

**中心極限定理**。

名前は重いけど、意味はかなり強い。

## 中心極限定理とは

ざっくり言うと、

> 母集団の分布がどんな形でも、標本サイズnが十分大きければ、標本平均の分布は正規分布に近づく

という定理。

---

## つまり何が嬉しいのか

元データが正規分布でなくても、標本平均は正規分布っぽくなる。

たとえば、元データが偏っていても、

```text
売上
待ち時間
アクセス数
競馬の払戻金
```

みたいに歪んだ分布でも、標本平均を考えると正規分布に近づいていく。

だから、正規分布を使って推定や検定ができる。

ここが強い。

---

# 12. 中心極限定理のイメージ

元データがこんなに歪んでいるとする。

```text
低い値が多く、高い値が少ない分布

█
██
███
████
██████
████████
████████████
```

でも、標本をたくさん取って平均を計算する。

その標本平均をたくさん集めると、こうなる。

```text
標本平均の分布

        █
      ███
    █████
  ███████
█████████
  ███████
    █████
      ███
        █
```

山型に近づく。

これが中心極限定理の感覚。

---

# 13. 標本平均の分布まとめ

母平均 `μ`、母分散 `σ²` の母集団から、サイズ `n` の標本を取る。

このとき、標本平均 `X̄` について、

```text
E(X̄) = μ
V(X̄) = σ² / n
標準誤差 = σ / √n
```

さらに、nが十分大きければ、

```text
X̄ はだいたい N(μ, σ²/n) に従う
```

と考えられる。

ここは本当に重要。

---

# 14. 正規分布との接続

前回やった正規分布を思い出す。

```text
X ~ N(μ, σ²)
```

これは、個々のデータ `X` が正規分布に従うという意味。

でも今回は、

```text
X̄ ~ N(μ, σ²/n)
```

です。

これは、標本平均 `X̄` が正規分布に従うという意味。

この違いが大事。

---

## 比較

|分布|意味|
|---|---|
|`X ~ N(μ, σ²)`|個々のデータXの分布|
|`X̄ ~ N(μ, σ²/n)`|標本平均X̄の分布|

ここを混ぜると、標準化で間違える。

---

# 15. 標本平均の標準化

標本平均を標準化するときは、

```text
Z = (X̄ - μ) / (σ / √n)
```

です。

前回の正規分布では、

```text
Z = (X - μ) / σ
```

だった。

今回は `X` ではなく `X̄` だから、割るものも変わる。

```text
個々の値を標準化する → σで割る
標本平均を標準化する → σ/√nで割る
```

ここは危ない。

---

# 16. 例題1：標本平均の標準誤差

あるテストの点数について、母平均は60点、母標準偏差は20点だとする。  
この母集団から36人を無作為に選び、標本平均を計算する。

このとき、標本平均の標準誤差は？

---

## 解答

```text
σ = 20
n = 36
```

標準誤差は、

```text
σ / √n = 20 / √36
       = 20 / 6
       = 3.33...
```

したがって、

```text
標準誤差 ≒ 3.33
```

---

# 17. 例題2：標本平均の確率

あるテストの点数について、母平均は60点、母標準偏差は20点だとする。  
36人を選んだとき、標本平均が65点以上になる確率を求めたい。

標本平均を `X̄` とする。

---

## 解き方

まず、標本平均の分布を考える。

```text
μ = 60
σ = 20
n = 36
```

標準誤差は、

```text
σ / √n = 20 / 6 = 3.33...
```

したがって、

```text
X̄ はだいたい N(60, 20²/36) に従う
```

標準化する。

```text
Z = (X̄ - μ) / (σ / √n)
```

今回は `X̄ = 65` なので、

```text
Z = (65 - 60) / (20 / 6)
  = 5 / 3.33...
  = 1.5
```

つまり、

```text
P(X̄ >= 65) = P(Z >= 1.5)
```

標準正規分布表を使うと、

```text
P(Z >= 1.5) ≒ 0.0668
```

つまり、約6.68%。

---

# 18. ここでの重要ポイント

この問題でやっていることは、単なる計算ではない。

流れはこう。

```text
母集団の情報を見る
↓
標本サイズnを見る
↓
標本平均の分布を作る
↓
標準誤差を計算する
↓
標準化する
↓
標準正規分布表を見る
```

この流れが、次の「推定」「仮説検定」の基本形になる。

---

# 19. 標本比率の分布

標本平均だけではなく、標本比率も重要。

たとえば、

```text
支持率
合格率
不良品率
クリック率
的中率
```

こういう「割合」を扱うとき。

---

## 母比率と標本比率

母集団全体の比率を `p` とする。

標本から計算した比率を `p̂` とする。

読み方は「ピーハット」。

たとえば、100人中30人が支持したら、

```text
p̂ = 30 / 100 = 0.3
```

です。

---

## 標本比率の期待値と分散

標本サイズを `n`、母比率を `p` とすると、

```text
E(p̂) = p
V(p̂) = p(1 - p) / n
```

標準誤差は、

```text
√{p(1 - p) / n}
```

です。

---

# 20. 標本比率も正規分布に近づく

nが十分大きいとき、標本比率 `p̂` はだいたい正規分布に従う。

```text
p̂ ≒ N(p, p(1-p)/n)
```

標準化すると、

```text
Z = (p̂ - p) / √{p(1-p)/n}
```

これも統計検定2級で重要。

---

# 21. 例題3：標本比率

ある商品の不良率は5%である。  
400個を検査したとき、不良品率 `p̂` の標準誤差を求めよ。

---

## 解答

```text
p = 0.05
n = 400
```

標準誤差は、

```text
√{p(1-p)/n}
= √{0.05 × 0.95 / 400}
= √{0.0475 / 400}
= √0.00011875
≒ 0.0109
```

つまり、

```text
標準誤差 ≒ 0.0109
```

割合でいうと、約1.09%。

---

# 22. 標本分布で出てくる4つの分布

ここから今後の予告。

標本分布では、次の4つが重要になる。

|分布|主な用途|
|---|---|
|正規分布|母分散が既知、または標本サイズが大きいとき|
|t分布|母分散が未知で、標本サイズが小さいとき|
|カイ二乗分布|母分散の推定・検定|
|F分布|分散比、分散分析、回帰の検定|

前に気にしていた **カイ二乗分布・F分布** はここから出てくる。

つまり、位置づけはこう。

```text
確率分布
↓
標本分布
↓
t分布・カイ二乗分布・F分布
↓
推定・検定・分散分析・回帰
```

---

# 23. ここで全部理解しようとしなくていい

ただし、今日の段階で t分布・カイ二乗分布・F分布を全部細かくやると重い。

今日の時点では、まずこれだけでいい。

```text
標本平均の分布 → 正規分布・t分布につながる
標本分散の分布 → カイ二乗分布につながる
分散の比の分布 → F分布につながる
```

この地図を持っておけば十分。

# 24. 今日のまとめ

今日の内容を整理する。

|用語|意味|
|---|---|
|母集団|本当に知りたい対象全体|
|標本|母集団から取り出した一部|
|統計量|標本から計算される値|
|標本分布|統計量の確率分布|
|標本平均|標本から計算した平均|
|標準誤差|標本平均などのブレの大きさ|
|中心極限定理|nが大きいと標本平均の分布は正規分布に近づく|

---

## 重要公式

```text
E(X̄) = μ
```

```text
V(X̄) = σ² / n
```

```text
標準誤差 = σ / √n
```

```text
Z = (X̄ - μ) / (σ / √n)
```

標本比率では、

```text
E(p̂) = p
```

```text
V(p̂) = p(1-p) / n
```

```text
標準誤差 = √{p(1-p)/n}
```

# 25. 今日の盲点

率直に言うと、標本分布で一番危ないのはこれ。

> 元データの分布と、標本平均の分布を混同すること。

たとえば、

```text
X ~ N(μ, σ²)
```

と、

```text
X̄ ~ N(μ, σ²/n)
```

は別物。

標準化でも、

```text
個々の値X → σで割る
標本平均X̄ → σ/√nで割る
```

ここを混ぜると、推定・検定で一気に崩れる。

---

# 26. 最低限の暗記

今日の内容で、最低限これだけは暗記していい。

```text
標本分布 = 標本から計算した統計量の分布
```

```text
標本平均の期待値 = 母平均
```

```text
標本平均の標準誤差 = σ / √n
```

```text
標本サイズが大きいと、標本平均は正規分布に近づく
```

```text
標本平均を標準化するときは、σではなく σ/√n で割る
```

ここまで入れば、第15回の **推定** に進める。
