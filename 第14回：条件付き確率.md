
## 今日のゴール

今日のゴールはこれです。

> 条件付き確率とは、  
> 「ある条件がすでに分かっているときの確率」  
> である。

たとえば、普通にサイコロを振ったとき、

```text
6が出る確率 = 1/6
```

です。

でも、

```text
偶数が出たことは分かっている。
そのうえで、6である確率は？
```

と聞かれたら、話が変わります。

偶数は、

```text
{2, 4, 6}
```

なので、その中で6は1つ。

だから、

```text
確率 = 1/3
```

になります。

これが条件付き確率です。

---

# 1. 条件が変わると「全体」が変わる

条件付き確率で一番大事なのはこれです。

> 条件が与えられると、分母になる「全体」が変わる。

通常のサイコロなら、全体は、

```text
{1, 2, 3, 4, 5, 6}
```

です。

でも、「偶数が出た」という条件があるなら、全体は、

```text
{2, 4, 6}
```

になります。

つまり、世界が狭くなる。

---

## 例

サイコロを1回振る。

```text
A = 6が出る
B = 偶数が出る
```

とします。

普通に6が出る確率は、

```text
P(A) = 1/6
```

です。

でも、偶数が出たと分かっているときに6が出る確率は、

```text
1/3
```

です。

これは、

```text
P(A | B)
```

と書きます。

読み方は、

```text
Bが起きたもとでAが起きる確率
```

です。

---

# 2. 条件付き確率の記号

条件付き確率はこう書きます。

```text
P(A | B)
```

読み方は、

```text
Bが起こったという条件のもとで、Aが起こる確率
```

です。

順番がかなり大事です。

|記号|意味|
|---|---|
|`P(A｜B)`|Bが起きたと分かっているとき、Aが起こる確率|
|`P(B｜A)`|Aが起きたと分かっているとき、Bが起こる確率|

この2つは普通、同じではありません。

---

# 3. 条件付き確率の公式

条件付き確率の公式はこれです。

```text
P(A | B) = P(A ∩ B) / P(B)
```

ただし、

```text
P(B) > 0
```

です。

意味はこうです。

```text
Bが起きた世界の中で、
Aも同時に起きている部分がどれくらいあるか
```

つまり、

```text
条件になる事象Bが分母
AとBが両方起きる部分が分子
```

です。

---

## 図で見る

```text
全体 Ω
┌────────────────────┐
│        A           │
│    ┌───────┐       │
│    │ A∩B  │ B     │
│    └───────┘       │
└────────────────────┘
```

`P(A | B)` では、全体を `Ω` ではなく `B` として見ます。

だから、

```text
P(A | B) = Bの中にあるAの割合
```

です。

---

# 4. サイコロで計算してみる

サイコロを1回振る。

```text
A = 6が出る = {6}
B = 偶数が出る = {2, 4, 6}
```

このとき、

```text
A ∩ B = {6}
```

です。

したがって、

```text
P(A ∩ B) = 1/6
P(B) = 3/6
```

条件付き確率は、

```text
P(A | B)
= P(A ∩ B) / P(B)
= (1/6) / (3/6)
= 1/3
```

です。

さっきの直感と一致します。

---

# 5. P(A｜B) と P(B｜A) は違う

ここはかなり危ないです。

同じサイコロで、

```text
A = 6が出る
B = 偶数が出る
```

とします。

さっきは、

```text
P(A | B) = 1/3
```

でした。

これは、

```text
偶数が出たと分かっているとき、6である確率
```

です。

一方、

```text
P(B | A)
```

は、

```text
6が出たと分かっているとき、偶数である確率
```

です。

6は必ず偶数なので、

```text
P(B | A) = 1
```

です。

つまり、

```text
P(A | B) = 1/3
P(B | A) = 1
```

同じではありません。

---

# 6. 具体例：カード

トランプ52枚から1枚引くとします。

```text
A = エースを引く
B = スペードを引く
```

このとき、

```text
P(A | B)
```

は、

```text
スペードだと分かっているとき、エースである確率
```

です。

スペードは13枚。  
その中でエースは1枚。

だから、

```text
P(A | B) = 1/13
```

です。

---

一方、

```text
P(B | A)
```

は、

```text
エースだと分かっているとき、スペードである確率
```

です。

エースは4枚。  
その中でスペードのエースは1枚。

だから、

```text
P(B | A) = 1/4
```

です。

やはり違います。

---

# 7. 条件付き確率を表で見る

条件付き確率は、表で見るとかなり分かりやすいです。

ある学校の100人について、運動部かどうか、男子か女子かを調べたとします。

||運動部|運動部でない|合計|
|---|--:|--:|--:|
|男子|30|20|50|
|女子|10|40|50|
|合計|40|60|100|

---

## 男子である確率

全体100人のうち男子は50人なので、

```text
P(男子) = 50/100 = 0.5
```

です。

---

## 運動部である確率

全体100人のうち運動部は40人なので、

```text
P(運動部) = 40/100 = 0.4
```

です。

---

## 男子であると分かっているとき、運動部である確率

これは、

```text
P(運動部 | 男子)
```

です。

条件が「男子」なので、分母は男子50人です。

男子50人のうち、運動部は30人。

したがって、

```text
P(運動部 | 男子) = 30/50 = 0.6
```

です。

---

## 運動部であると分かっているとき、男子である確率

これは、

```text
P(男子 | 運動部)
```

です。

条件が「運動部」なので、分母は運動部40人です。

運動部40人のうち、男子は30人。

したがって、

```text
P(男子 | 運動部) = 30/40 = 0.75
```

です。

---

## 違い

```text
P(運動部 | 男子) = 0.6
P(男子 | 運動部) = 0.75
```

違います。

これは、

```text
男子の中で運動部の割合
運動部の中で男子の割合
```

を比べているからです。

分母が違います。

---

# 8. 条件付き確率の本質

条件付き確率の本質は、

> 分母を間違えないこと

です。

普通の確率では、分母は全体です。

```text
全体100人中、何人か
```

でも条件付き確率では、分母は条件です。

```text
男子50人中、何人か
運動部40人中、何人か
スペード13枚中、何枚か
偶数3通り中、何通りか
```

これが一番重要です。

---

# 9. 条件付き確率から積の公式へ

条件付き確率の公式は、

```text
P(A | B) = P(A ∩ B) / P(B)
```

でした。

これを変形すると、

```text
P(A ∩ B) = P(B)P(A | B)
```

になります。

これはかなり重要です。

意味は、

```text
Bが起こる確率
×
Bが起こったもとでAが起こる確率
=
AとBが両方起こる確率
```

です。

---

## 順番を逆にしてもいい

同じように、

```text
P(B | A) = P(A ∩ B) / P(A)
```

なので、

```text
P(A ∩ B) = P(A)P(B | A)
```

です。

つまり、

```text
P(A ∩ B) = P(A)P(B | A) = P(B)P(A | B)
```

です。

---

# 10. 例：袋から玉を2個引く

袋に赤玉3個、青玉2個があります。  
合計5個です。

この袋から、玉を戻さずに2個引きます。

```text
1個目が赤
2個目も赤
```

となる確率を求めます。

---

## Step 1：1個目が赤の確率

赤玉は3個、全体は5個なので、

```text
P(1個目赤) = 3/5
```

です。

---

## Step 2：1個目が赤だった条件のもとで、2個目も赤の確率

1個目で赤を引いたので、袋の中は、

```text
赤玉：2個
青玉：2個
合計：4個
```

になります。

したがって、

```text
P(2個目赤 | 1個目赤) = 2/4 = 1/2
```

です。

---

## Step 3：両方赤の確率

積の公式より、

```text
P(1個目赤 ∩ 2個目赤)
= P(1個目赤) × P(2個目赤 | 1個目赤)
= 3/5 × 2/4
= 6/20
= 3/10
```

です。

---

# 11. 「戻す」と「戻さない」の違い

ここも重要です。

## 戻す場合

袋に赤玉3個、青玉2個。  
1個引いたあと、袋に戻してから2個目を引く。

この場合、1回目の結果は2回目に影響しません。

だから、

```text
P(1個目赤) = 3/5
P(2個目赤) = 3/5
```

です。

両方赤の確率は、

```text
3/5 × 3/5 = 9/25
```

です。

---

## 戻さない場合

1個引いたあと、戻さずに2個目を引く。

1回目で赤を引くと、袋の中の赤玉が減ります。

だから、

```text
P(1個目赤) = 3/5
P(2個目赤 | 1個目赤) = 2/4
```

です。

両方赤の確率は、

```text
3/5 × 2/4 = 3/10
```

です。

---

## 違い

|条件|両方赤の確率|
|---|--:|
|戻す|`9/25 = 0.36`|
|戻さない|`3/10 = 0.30`|

戻さない場合は、1回目の結果が2回目に影響します。

つまり、独立ではありません。

---

# 12. 独立とは何か

ここで独立をもう少し正確に扱います。

AとBが独立とは、

> Aが起こっても、Bの確率が変わらない

ということです。

式で書くと、

```text
P(B | A) = P(B)
```

です。

または、

```text
P(A | B) = P(A)
```

です。

---

## 独立なら積の公式が簡単になる

一般には、

```text
P(A ∩ B) = P(A)P(B | A)
```

です。

でも独立なら、

```text
P(B | A) = P(B)
```

なので、

```text
P(A ∩ B) = P(A)P(B)
```

になります。

これが独立の積の公式です。

---

# 13. 独立の例

サイコロを2回振る。

```text
A = 1回目に6が出る
B = 2回目に6が出る
```

1回目に6が出ても、2回目に6が出る確率は変わりません。

```text
P(B) = 1/6
P(B | A) = 1/6
```

だから独立です。

両方6が出る確率は、

```text
P(A ∩ B)
= P(A)P(B)
= 1/6 × 1/6
= 1/36
```

です。

---

# 14. 独立でない例

さっきの玉の例です。

袋に赤玉3個、青玉2個。  
戻さずに2個引く。

```text
A = 1個目が赤
B = 2個目が赤
```

普通に2個目が赤である確率だけを見ると、対称性から、

```text
P(B) = 3/5
```

です。

でも、1個目が赤だったと分かっていると、

```text
P(B | A) = 2/4 = 1/2
```

です。

```text
P(B) = 3/5
P(B | A) = 1/2
```

違います。

だから独立ではありません。

---

# 15. 排反と独立の再整理

前回も触れましたが、ここは危険なので再整理します。

## 排反

排反は、

```text
A ∩ B = ∅
```

です。

つまり、同時に起こらない。

例：

```text
1回のサイコロで1が出る
1回のサイコロで2が出る
```

---

## 独立

独立は、

```text
P(B | A) = P(B)
```

です。

つまり、一方が起こっても、もう一方の確率が変わらない。

例：

```text
1回目のサイコロで6が出る
2回目のサイコロで6が出る
```

---

## 排反と独立はほぼ逆方向

AとBが排反で、かつ `P(A)>0`, `P(B)>0` なら、Aが起こった時点でBは絶対に起こりません。

つまり、

```text
P(B | A) = 0
```

です。

でも元々、

```text
P(B) > 0
```

です。

だから、

```text
P(B | A) ≠ P(B)
```

となり、独立ではありません。

つまり、

> 確率が0でない排反事象は、独立ではない。

ここは試験でも実務でも危ないです。

---

# 16. 条件付き確率と情報更新

条件付き確率は、単なる計算ではありません。

本質は、

> 情報が増えると、確率が更新される

ということです。

たとえば、

```text
ある人が運動部である確率 = 0.4
```

だったとします。

でも、

```text
その人は男子である
```

という情報が追加されると、

```text
男子である条件のもとで運動部である確率 = 0.6
```

に変わるかもしれません。

これは、情報によって確率が変わったということです。

---

# 17. 次回のベイズの定理との接続

次回やるベイズの定理は、条件付き確率の応用です。

ざっくり言うと、ベイズの定理は、

```text
P(A | B) から P(B | A) をつなぐ道具
```

です。

または、

```text
結果を見て、原因の確率を更新する道具
```

です。

たとえば、

```text
病気の人が陽性になる確率
```

から、

```text
陽性だった人が本当に病気である確率
```

を求める。

こういうのがベイズの定理です。

だから、今日の `P(A | B)` と `P(B | A)` の違いは、かなり重要です。

# 18. 練習問題

## 問題1

サイコロを1回振る。

```text
A = 4以上が出る
B = 偶数が出る
```

このとき、`P(A | B)` を求めよ。

---

## 解答

まず、

```text
A = {4, 5, 6}
B = {2, 4, 6}
```

です。

条件はBなので、分母はBです。

Bの中身は、

```text
{2, 4, 6}
```

そのうちAにも入っているのは、

```text
{4, 6}
```

です。

したがって、

```text
P(A | B) = 2/3
```

です。

---

## 問題2

同じ条件で、`P(B | A)` を求めよ。

```text
A = 4以上が出る
B = 偶数が出る
```

---

## 解答

条件はAなので、分母はAです。

```text
A = {4, 5, 6}
```

そのうちBにも入っているのは、

```text
{4, 6}
```

です。

したがって、

```text
P(B | A) = 2/3
```

です。

この例ではたまたま同じですが、いつも同じとは限りません。

---

## 問題3

100人のうち、喫煙者が30人、非喫煙者が70人。  
喫煙者30人のうち、運動習慣がある人は6人。  
非喫煙者70人のうち、運動習慣がある人は35人。

喫煙者であると分かっているとき、運動習慣がある確率を求めよ。

---

## 解答

求めるのは、

```text
P(運動習慣あり | 喫煙者)
```

です。

条件は喫煙者なので、分母は喫煙者30人。

そのうち運動習慣がある人は6人。

したがって、

```text
P(運動習慣あり | 喫煙者)
= 6/30
= 0.2
```

答えは20%です。

---

## 問題4

問題3と同じ条件で、運動習慣があると分かっているとき、喫煙者である確率を求めよ。

---

## 解答

求めるのは、

```text
P(喫煙者 | 運動習慣あり)
```

です。

運動習慣がある人は、

```text
喫煙者の中に6人
非喫煙者の中に35人
合計41人
```

です。

そのうち喫煙者は6人。

したがって、

```text
P(喫煙者 | 運動習慣あり)
= 6/41
≒ 0.146
```

答えは約14.6%です。

---

## 問題5

袋に赤玉4個、青玉6個がある。  
玉を戻さずに2個引く。

1個目も2個目も赤である確率を求めよ。

---

## 解答

1個目が赤である確率は、

```text
4/10
```

1個目が赤だった条件のもとで、2個目も赤である確率は、

```text
3/9
```

したがって、

```text
4/10 × 3/9
= 12/90
= 2/15
```

答えは、

```text
2/15
```

です。

# 19. 今日のまとめ

|用語|意味|
|---|---|
|条件付き確率|ある条件が分かっているときの確率|
|`P(A｜B)`|Bが起きた条件のもとでAが起きる確率|
|条件|分母になる範囲|
|独立|条件が加わっても確率が変わらないこと|
|排反|同時に起こらないこと|

---

## 重要公式

```text
P(A | B) = P(A ∩ B) / P(B)
```

```text
P(A ∩ B) = P(B)P(A | B)
```

```text
P(A ∩ B) = P(A)P(B | A)
```

独立なら、

```text
P(A ∩ B) = P(A)P(B)
```

---

# 20. 今日の盲点

今日の最大の盲点はこれです。

```text
P(A | B) と P(B | A) を混同する
```

これは本当に危ないです。

```text
P(病気 | 陽性)
P(陽性 | 病気)
```

この2つはまったく違います。

次回のベイズの定理では、この違いが中心になります。

もう1つ危ないのは、

```text
条件付き確率では、分母が全体ではなく条件になる
```

という点です。

条件が「男子」なら分母は男子。  
条件が「運動部」なら分母は運動部。  
条件が「陽性」なら分母は陽性。

ここを外すと全部ズレます。

---

# 21. 最低限の暗記

今回、最低限暗記するならこれです。

```text
条件付き確率 = 条件で世界を狭めたあとの確率
```

```text
P(A | B) = Bの中でAが起こる割合
```

```text
P(A | B) = P(A ∩ B) / P(B)
```

```text
P(A | B) と P(B | A) は普通は違う
```

```text
独立 = 条件が加わっても確率が変わらない
```

次回は **第15回：ベイズの定理** です。  
今日の条件付き確率を使って、「結果から原因を逆算する」話に入ります。
