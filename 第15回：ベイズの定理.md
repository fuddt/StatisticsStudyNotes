了解。  
第15回は **ベイズの定理** です。

前回の条件付き確率の続きです。  
今回は、

```text
P(A | B) と P(B | A) をつなぐ
```

という話です。

---

## 今日のゴール

今日のゴールはこれです。

> ベイズの定理とは、  
> 「結果が分かったあとで、原因の確率を更新する公式」  
> である。

たとえば、病気の検査で考えると、

```text
病気の人が陽性になる確率
```

と、

```text
陽性だった人が本当に病気である確率
```

は違います。

この違いを扱うのがベイズの定理です。

---

# 1. まず前回の復習

前回、条件付き確率をやりました。

```text
P(A | B)
```

これは、

```text
Bが起きたという条件のもとで、Aが起きる確率
```

でした。

公式は、

```text
P(A | B) = P(A ∩ B) / P(B)
```

です。

---

# 2. P(A|B) と P(B|A) は違う

ここがベイズの定理の入口です。

たとえば、

```text
A = 病気である
B = 検査で陽性になる
```

とします。

このとき、

```text
P(B | A)
```

は、

```text
病気の人が陽性になる確率
```

です。

一方、

```text
P(A | B)
```

は、

```text
陽性だった人が本当に病気である確率
```

です。

この2つは別物です。

---

## 具体的に言うと

検査の性能として、

```text
病気の人が陽性になる確率は99%
```

と聞くと、

```text
陽性なら99%病気なのでは？
```

と思いやすい。

でもこれは危険です。

なぜなら、病気の人がそもそも少ない場合、陽性者の中には「病気ではないのに陽性になった人」もかなり混ざるからです。

ここでベイズの定理が必要になります。

---

# 3. ベイズの定理の公式

ベイズの定理はこれです。

```text
P(A | B) = P(B | A)P(A) / P(B)
```

記号だけ見ると硬いですが、意味はこうです。

```text
Aが原因
Bが結果
```

と考えると、

```text
結果Bが分かったあとで、
原因Aだった確率を求める
```

公式です。

---

## 各パーツの意味

|記号|意味|
|---|---|
|`P(A)`|Aが起こる元々の確率|
|`P(B｜A)`|AのときBが起こる確率|
|`P(B)`|Bが起こる全体の確率|
|`P(A｜B)`|Bが起きたあとでAだった確率|

---

# 4. 事前確率・尤度・事後確率

ベイズの定理では、よくこの言葉が出ます。

|用語|意味|
|---|---|
|事前確率|データを見る前の確率|
|尤度|その原因なら、その結果がどれくらい起こりやすいか|
|事後確率|データを見た後の確率|

---

## 病気検査で言うと

```text
A = 病気
B = 陽性
```

なら、

|用語|内容|
|---|---|
|事前確率 `P(A)`|検査前に病気である確率|
|尤度 `P(B｜A)`|病気の人が陽性になる確率|
|事後確率 `P(A｜B)`|陽性後に本当に病気である確率|

つまり、ベイズの定理は、

```text
事前確率を、結果を見て、事後確率に更新する
```

公式です。

---

# 5. なぜ P(B) が分母にあるのか

公式は、

```text
P(A | B) = P(B | A)P(A) / P(B)
```

でした。

分母の `P(B)` は、

```text
Bが起こる全体の確率
```

です。

病気検査なら、

```text
陽性になる全体の確率
```

です。

陽性になる人には2種類います。

```text
病気で陽性になる人
病気でないのに陽性になる人
```

だから、

```text
P(B)
```

は、この両方を合わせたものです。

---

# 6. 全確率の公式

分母の `P(B)` を求めるときに使うのが、全確率の公式です。

病気か病気でないかで分けると、

```text
P(B) = P(B | A)P(A) + P(B | Aᶜ)P(Aᶜ)
```

です。

意味は、

```text
Bが起こる確率
=
AでBが起こる確率
+
AでないのにBが起こる確率
```

です。

病気検査なら、

```text
陽性になる確率
=
病気で陽性になる確率
+
病気でないのに陽性になる確率
```

です。

---

# 7. 病気検査の例

ここから具体例でやります。

ある病気について、次の条件があるとします。

```text
病気である確率：1%
病気の人が陽性になる確率：99%
病気でない人が陽性になる確率：5%
```

つまり、

```text
P(病気) = 0.01
P(陽性 | 病気) = 0.99
P(陽性 | 病気でない) = 0.05
```

求めたいのは、

```text
P(病気 | 陽性)
```

です。

つまり、

```text
陽性だった人が本当に病気である確率
```

です。

---

# 8. Step 1：記号を置く

```text
A = 病気
B = 陽性
```

とします。

求めたいのは、

```text
P(A | B)
```

です。

ベイズの定理は、

```text
P(A | B) = P(B | A)P(A) / P(B)
```

です。

---

# 9. Step 2：分子を計算する

分子は、

```text
P(B | A)P(A)
```

です。

値を入れると、

```text
P(B | A)P(A)
= 0.99 × 0.01
= 0.0099
```

これは、

```text
病気で、かつ陽性になる確率
```

です。

---

# 10. Step 3：分母を計算する

分母は、

```text
P(B)
```

つまり、陽性になる全体の確率です。

陽性になるパターンは2つです。

```text
病気で陽性
病気でないのに陽性
```

なので、

```text
P(B)
= P(B | A)P(A) + P(B | Aᶜ)P(Aᶜ)
```

です。

値を入れます。

```text
P(B)
= 0.99 × 0.01 + 0.05 × 0.99
= 0.0099 + 0.0495
= 0.0594
```

---

# 11. Step 4：ベイズの定理に入れる

```text
P(A | B)
= 0.0099 / 0.0594
≒ 0.1667
```

つまり、

```text
約16.7%
```

です。

---

# 12. 結論

検査の性能はかなり高く見えます。

```text
病気の人が陽性になる確率 = 99%
```

でも、陽性だった人が本当に病気である確率は、

```text
約16.7%
```

です。

これは直感に反します。

でも理由は明確です。

```text
病気の人が少ない
病気でない人が多い
病気でない人の5%が陽性になる
```

だから、陽性者の中に偽陽性がかなり混ざるのです。

---

# 13. 10000人で考える

ベイズは、確率だけで見ると混乱しやすいです。  
人数で見るとかなり分かりやすい。

10000人いるとします。

## 病気の人

病気である確率は1%なので、

```text
病気の人 = 10000 × 0.01 = 100人
```

このうち99%が陽性なので、

```text
病気で陽性 = 100 × 0.99 = 99人
```

---

## 病気でない人

病気でない人は、

```text
9900人
```

このうち5%が陽性になるので、

```text
病気でないのに陽性 = 9900 × 0.05 = 495人
```

---

## 陽性者全体

陽性者は、

```text
病気で陽性 99人
+
病気でないのに陽性 495人
=
594人
```

そのうち本当に病気なのは99人。

だから、

```text
P(病気 | 陽性)
= 99 / 594
≒ 0.1667
```

つまり約16.7%です。

---

# 14. ここでの本質

この例で分かることは、

```text
検査の精度だけ見てもダメ
```

ということです。

必要なのは、

```text
病気の人が元々どれくらいいるか
```

です。

つまり、事前確率です。

ベイズの定理では、事前確率が非常に重要です。

---

# 15. ベイズの定理を表で見る

||陽性|陰性|合計|
|---|--:|--:|--:|
|病気|99|1|100|
|病気でない|495|9405|9900|
|合計|594|9406|10000|

求めたいのは、

```text
P(病気 | 陽性)
```

です。

条件は「陽性」なので、分母は陽性者594人。

そのうち病気は99人。

```text
99 / 594 ≒ 0.1667
```

です。

条件付き確率の考え方そのものです。

---

# 16. ベイズの定理の見方

ベイズの定理は、こう見るとよいです。

```text
原因 → 結果
```

は分かりやすい。

たとえば、

```text
病気 → 陽性
```

です。

でも知りたいのは逆です。

```text
陽性 → 病気？
```

つまり、

```text
結果から原因を推測したい
```

ベイズの定理は、この逆向き推論の道具です。

---

# 17. 競馬でたとえる

競馬でも似たことが起きます。

たとえば、

```text
強い馬は人気になりやすい
```

これは、

```text
P(人気 | 強い)
```

に近いです。

でも、実際に知りたいのは、

```text
人気馬は本当に強いのか？
```

です。

これは、

```text
P(強い | 人気)
```

です。

この2つは違います。

```text
P(人気 | 強い) と P(強い | 人気) は違う
```

ここを混同すると、市場分析でかなり危険です。

---

## もう少し具体的に

仮に、

```text
強い馬は80%の確率で人気になる
```

としても、

```text
人気馬の80%が強い
```

とは限りません。

なぜなら、

```text
強くないのに人気になる馬
```

もいるからです。

騎手人気、血統人気、前走派手、過剰投票などです。

だから、人気を見たあとに強さを推測するには、ベイズ的な見方が必要です。

---

# 18. ベイズの定理の一般形

事象AとBについて、

```text
P(A | B) = P(B | A)P(A) / P(B)
```

さらに、BがAまたはAでない場合に起こるなら、

```text
P(B) = P(B | A)P(A) + P(B | Aᶜ)P(Aᶜ)
```

なので、

```text
P(A | B)
=
P(B | A)P(A)
/
{P(B | A)P(A) + P(B | Aᶜ)P(Aᶜ)}
```

この形はかなりよく使います。

---

# 19. ベイズの定理でよくあるミス

## ミス1：P(A|B) と P(B|A) を逆にする

これは最大のミスです。

```text
P(病気 | 陽性)
```

と、

```text
P(陽性 | 病気)
```

は違います。

前者は「陽性者の中で病気の割合」。  
後者は「病気の人の中で陽性の割合」。

分母が違います。

---

## ミス2：事前確率を無視する

検査精度だけ見て、

```text
陽性ならほぼ病気
```

と考えるのは危険です。

病気がそもそも少なければ、偽陽性が多くなります。

---

## ミス3：分母の P(B) を雑に扱う

`P(B)` は、ただの飾りではありません。

```text
Bが起こる全体の確率
```

です。

病気検査なら、

```text
陽性になる全体の確率
```

です。

この中には、

```text
病気で陽性
病気でないのに陽性
```

の両方が入ります。

---

# 20. 練習問題

## 問題1

ある病気の有病率は2%である。  
病気の人が陽性になる確率は90%。  
病気でない人が陽性になる確率は10%。

陽性だった人が本当に病気である確率を求めよ。

---

## 解答

```text
A = 病気
B = 陽性
```

与えられているのは、

```text
P(A) = 0.02
P(B | A) = 0.90
P(B | Aᶜ) = 0.10
```

求めたいのは、

```text
P(A | B)
```

です。

ベイズの定理より、

```text
P(A | B)
=
P(B | A)P(A)
/
{P(B | A)P(A) + P(B | Aᶜ)P(Aᶜ)}
```

値を入れます。

```text
P(A | B)
=
0.90 × 0.02
/
{0.90 × 0.02 + 0.10 × 0.98}
```

分子は、

```text
0.90 × 0.02 = 0.018
```

分母は、

```text
0.018 + 0.098 = 0.116
```

したがって、

```text
P(A | B) = 0.018 / 0.116 ≒ 0.155
```

答えは、

```text
約15.5%
```

です。

---

## 問題2

あるメールのうち、迷惑メールは20%である。  
迷惑メールには「無料」という単語が含まれる確率が60%。  
通常メールには「無料」という単語が含まれる確率が10%。

「無料」という単語が含まれているメールが迷惑メールである確率を求めよ。

---

## 解答

```text
A = 迷惑メール
B = 「無料」を含む
```

与えられているのは、

```text
P(A) = 0.20
P(B | A) = 0.60
P(B | Aᶜ) = 0.10
```

求めたいのは、

```text
P(A | B)
```

です。

```text
P(A | B)
=
0.60 × 0.20
/
{0.60 × 0.20 + 0.10 × 0.80}
```

分子は、

```text
0.60 × 0.20 = 0.12
```

分母は、

```text
0.12 + 0.08 = 0.20
```

したがって、

```text
P(A | B) = 0.12 / 0.20 = 0.60
```

答えは、

```text
60%
```

です。

## 問題3

ある工場で、製品の5%が不良品である。  
不良品が検査で不合格になる確率は95%。  
良品が検査で不合格になる確率は2%。

検査で不合格だった製品が本当に不良品である確率を求めよ。

---

## 解答

```text
A = 不良品
B = 不合格
```

与えられているのは、

```text
P(A) = 0.05
P(B | A) = 0.95
P(B | Aᶜ) = 0.02
```

求めたいのは、

```text
P(A | B)
```

です。

```text
P(A | B)
=
0.95 × 0.05
/
{0.95 × 0.05 + 0.02 × 0.95}
```

分子は、

```text
0.95 × 0.05 = 0.0475
```

分母は、

```text
0.0475 + 0.019 = 0.0665
```

したがって、

```text
P(A | B)
= 0.0475 / 0.0665
≒ 0.714
```

答えは、

```text
約71.4%
```

です。
