了解。  
これがこの確率・分布・推測統計パートのラスト、**第26回：回帰分析** です。

# 第26回：回帰分析

## 今日のゴール

今日のゴールはこれです。

> 回帰分析とは、  
> ある変数を、別の変数で説明・予測するための方法。

たとえば、

```text
勉強時間が長いほど、テスト点数は高くなるのか
広告費を増やすと、売上は増えるのか
気温が高いと、アイスの売上は増えるのか
オッズ乖離が大きいと、回収率は変わるのか
```

こういう関係を、数式で表すのが回帰分析です。

---

# 1. 回帰分析とは何か

回帰分析では、主に2種類の変数を使います。

|用語|意味|
|---|---|
|目的変数|説明・予測したい変数|
|説明変数|目的変数を説明するために使う変数|

たとえば、

```text
勉強時間からテスト点数を予測する
```

なら、

|変数|内容|
|---|---|
|目的変数|テスト点数|
|説明変数|勉強時間|

です。

英語では、目的変数を `Y`、説明変数を `X` と置くことが多いです。

---

# 2. 単回帰分析

説明変数が1つだけの回帰分析を、**単回帰分析** といいます。

たとえば、

```text
Y = テスト点数
X = 勉強時間
```

のように、1つの説明変数で1つの目的変数を説明します。

単回帰分析の基本式はこれです。

```text
Y = β0 + β1X + ε
```

それぞれの意味はこうです。

|記号|意味|
|---|---|
|`Y`|目的変数|
|`X`|説明変数|
|`β0`|切片|
|`β1`|回帰係数・傾き|
|`ε`|誤差|

---

# 3. 回帰直線

実際のデータに対して、次のような直線を引くイメージです。

```text
テスト点数
  ^
90|                    *
80|              *   *
70|          *  *
60|      * *
50|   *
  +--------------------------> 勉強時間
```

この点の集まりに対して、

```text
Y = β0 + β1X
```

という直線を引きます。

この直線が **回帰直線** です。

---

# 4. 切片と傾き

単回帰式をもう一度見ます。

```text
Y = β0 + β1X
```

## 切片 `β0`

切片は、

> Xが0のときのYの値

です。

たとえば、

```text
Y = 50 + 5X
```

なら、`X = 0` のとき、

```text
Y = 50
```

です。

つまり、勉強時間が0時間のとき、予測点数は50点です。

---

## 傾き `β1`

傾きは、

> Xが1増えたとき、Yが平均的にどれくらい変わるか

です。

```text
Y = 50 + 5X
```

なら、`β1 = 5` です。

つまり、

```text
勉強時間が1時間増えると、点数は平均的に5点増える
```

という意味です。

---

# 5. 回帰係数の解釈

回帰分析で一番重要なのは、係数の解釈です。

たとえば、

```text
売上 = 100 + 3 × 広告費
```

という回帰式があるとします。

このとき、広告費の係数は3です。

これは、

```text
広告費が1単位増えると、売上は平均的に3単位増える
```

という意味です。

ただし、ここで注意です。

> 回帰係数は、必ず因果効果を表すわけではありません。

広告費と売上に関係があっても、それだけで、

```text
広告費を増やしたから売上が増えた
```

とは言えません。

因果を言うには、実験設計、交絡の制御、時系列構造などが必要です。

---

# 6. 誤差 `ε`

現実のデータは、完全に直線上には乗りません。

たとえば、同じ勉強時間でも、点数は人によって違います。

```text
Y = β0 + β1X + ε
```

の `ε` は、このズレです。

つまり、

```text
実際の値 = 回帰式で説明できる部分 + 説明できないズレ
```

です。

この `ε` を **誤差項** といいます。

---

# 7. 予測値と残差

回帰分析では、実際の値と予測値を区別します。

|用語|意味|
|---|---|
|実測値|実際に観測されたY|
|予測値|回帰式から予測されたY|
|残差|実測値 − 予測値|

予測値は、よく `Ŷ` と書きます。

読み方は「ワイハット」です。

```text
Ŷ = b0 + b1X
```

残差は、

```text
e = Y - Ŷ
```

です。

---

# 8. 残差とは何か

残差は、

> 回帰式で説明しきれなかったズレ

です。

たとえば、ある人の勉強時間が4時間で、実際の点数が80点だったとします。

回帰式による予測値が75点なら、

```text
残差 = 実測値 - 予測値
     = 80 - 75
     = 5
```

です。

つまり、この人は回帰式の予測より5点高かった、ということです。

---

# 9. 最小二乗法

回帰直線は、どうやって決めるのでしょうか。

基本は **最小二乗法** です。

最小二乗法とは、

> 残差の二乗和が最小になるように直線を決める方法

です。

残差をそのまま足すと、プラスとマイナスが打ち消し合います。

だから二乗します。

```text
残差平方和 = Σ(Y - Ŷ)²
```

この値が一番小さくなる直線を選びます。

---

# 10. なぜ二乗するのか

これは分散・カイ二乗・分散分析と同じ構造です。

```text
ズレを測る
↓
プラスとマイナスが打ち消さないように二乗する
↓
ズレの大きさを合計する
```

回帰分析も、結局は「ズレ」を扱っています。

ここまで学んできた内容とかなりつながっています。

---

# 11. 回帰係数の推定値

理論上の回帰式は、

```text
Y = β0 + β1X + ε
```

です。

でも、母集団の本当の `β0` や `β1` は分かりません。

だから標本データから推定します。

推定した係数を、

```text
b0, b1
```

と書くことがあります。

回帰式は、

```text
Ŷ = b0 + b1X
```

になります。

|理論上|標本からの推定|
|---|---|
|`β0`|`b0`|
|`β1`|`b1`|

---

# 12. 回帰係数と相関係数の違い

ここは混乱しやすいです。

|指標|意味|
|---|---|
|相関係数|2変数の直線的な関係の強さ|
|回帰係数|Xが1増えたときYがどれくらい変わるか|

相関係数は、-1から1の範囲です。

一方、回帰係数は単位に依存します。

たとえば、

```text
身長(cm) と 体重(kg)
```

の回帰係数と、

```text
身長(m) と 体重(kg)
```

の回帰係数は変わります。

なぜなら、Xの単位が変わるからです。

---

# 13. 決定係数 R²

回帰分析でよく出るのが **決定係数** です。

記号では、

```text
R²
```

と書きます。

決定係数は、

> 目的変数Yのばらつきのうち、回帰モデルで説明できた割合

です。

たとえば、

```text
R² = 0.64
```

なら、

```text
Yのばらつきの64%を、このモデルで説明できている
```

という意味です。

---

# 14. 決定係数の範囲

単回帰で切片を含む通常の回帰では、決定係数はだいたい、

```text
0 ≦ R² ≦ 1
```

です。

|R²|解釈|
|--:|---|
|0に近い|ほとんど説明できていない|
|1に近い|よく説明できている|
|0.5|ばらつきの約50%を説明|

ただし、R²が高いから良いモデルとは限りません。

過学習している場合もあります。

---

# 15. 決定係数の注意

R²には注意点があります。

## 注意1：因果を示さない

```text
R²が高い
```

ということは、

```text
よく説明できている
```

という意味ではあります。

でも、

```text
XがYの原因である
```

とは限りません。

---

## 注意2：予測性能を保証しない

学習データでR²が高くても、未知データで当たるとは限りません。

実務では、

```text
訓練データ
検証データ
テストデータ
時系列外検証
```

を分けて見る必要があります。

これは競馬AIでも同じです。

---

## 注意3：説明変数を増やすとR²は上がりやすい

重回帰分析では、説明変数を増やすと、R²は基本的に下がりにくいです。

だから、説明変数を増やしただけでR²が上がっても、良いモデルとは限りません。

この問題に対応するために、**自由度調整済み決定係数** が使われます。

---

# 16. 回帰係数の検定

回帰分析では、係数が本当に意味を持つかを検定します。

たとえば、

```text
Y = β0 + β1X + ε
```

で、`β1` が0ならどうでしょうか。

```text
Y = β0 + ε
```

となり、XはYを説明していないことになります。

だから、係数について次の仮説を立てます。

```text
H0：β1 = 0
H1：β1 ≠ 0
```

---

## 係数のt検定

回帰係数の検定では、t検定を使います。

検定統計量はざっくり言うと、

```text
t = 推定された係数 / その標準誤差
```

です。

つまり、

```text
係数が、推定誤差に比べて十分大きいか
```

を見ています。

---

# 17. p値の解釈

回帰出力では、係数ごとにp値が出ます。

たとえば、

|変数|係数|p値|
|---|--:|--:|
|勉強時間|5.2|0.003|

この場合、

```text
H0：勉強時間の係数 = 0
```

を検定して、p値が0.003だったということです。

有意水準5%なら、

```text
0.003 < 0.05
```

なので、係数は有意です。

つまり、

```text
勉強時間と点数には、統計的に有意な線形関係がある
```

と言えます。

ただし、

```text
勉強時間を増やせば必ず点数が上がる
```

と断言するのは危険です。

---

# 18. 回帰分析のF検定

回帰分析では、モデル全体の検定も行います。

これはF検定です。

単回帰なら、

```text
H0：β1 = 0
H1：β1 ≠ 0
```

と係数のt検定に近い意味になります。

重回帰では、

```text
H0：すべての説明変数の係数が0
H1：少なくとも1つの係数は0でない
```

というモデル全体の検定になります。

---

## F検定の意味

回帰のF検定は、ざっくり言うと、

```text
モデルで説明できるばらつき
÷
モデルで説明できないばらつき
```

を見ています。

F値が大きいほど、

```text
このモデルは、Yのばらつきをある程度説明できていそう
```

となります。

第23回のF分布、第25回の分散分析と同じ構造です。

---

# 19. 回帰分析と分散分析のつながり

回帰分析でも、ばらつきを分解します。

```text
全体のばらつき
=
回帰で説明できるばらつき
+
残差のばらつき
```

記号では、

```text
SST = SSR + SSE
```

と書きます。

|記号|意味|
|---|---|
|SST|全体平方和|
|SSR|回帰平方和、モデルで説明できた部分|
|SSE|残差平方和、説明できなかった部分|

分散分析の、

```text
SST = SSB + SSW
```

と構造が似ています。

つまり、回帰分析も広い意味では「ばらつきの分解」です。

---

# 20. 重回帰分析

説明変数が2つ以上ある回帰分析を、**重回帰分析** といいます。

式はこうです。

```text
Y = β0 + β1X1 + β2X2 + ... + βpXp + ε
```

たとえば、テスト点数を、

```text
勉強時間
睡眠時間
出席率
```

で説明するなら、

```text
点数 = β0 + β1×勉強時間 + β2×睡眠時間 + β3×出席率 + ε
```

です。

---

# 21. 重回帰係数の解釈

重回帰の係数は、単回帰より少し注意が必要です。

たとえば、

```text
点数 = β0 + β1×勉強時間 + β2×睡眠時間 + ε
```

で、`β1 = 4` だったとします。

これは、

> 睡眠時間を一定にしたとき、勉強時間が1時間増えると、点数が平均的に4点増える

という意味です。

つまり、重回帰では、

```text
他の説明変数を一定にしたうえで
```

という条件が入ります。

ここを忘れると解釈を間違えます。

---

# 22. 多重共線性

重回帰で危険なのが **多重共線性** です。

多重共線性とは、

> 説明変数同士が強く相関している状態

です。

たとえば、

```text
勉強時間
問題演習時間
机に向かっていた時間
```

は、かなり似た情報を持っているかもしれません。

このような変数を同時に入れると、係数が不安定になります。

---

## 多重共線性があると何が困るか

```text
係数の符号が変になる
係数の標準誤差が大きくなる
p値が大きくなる
解釈が不安定になる
```

つまり、モデル全体としては予測できていても、

```text
どの変数がどれくらい効いているか
```

の解釈が難しくなります。

---

# 23. 外れ値と影響点

回帰分析は外れ値に弱いです。

特に、説明変数Xの端にある点が大きく外れていると、回帰直線を強く引っ張ります。

```text
Y
^
|          *
|       *
|    *
| *
|                         *
+----------------------------> X
```

右端の点が回帰直線に強く影響することがあります。

こういう点を **影響点** といいます。

---

# 24. 回帰分析の前提

統計検定2級では、回帰分析の前提も重要です。

代表的には次の4つです。

|前提|意味|
|---|---|
|線形性|XとYの関係が直線で近似できる|
|独立性|誤差が互いに独立|
|等分散性|誤差の分散が一定|
|正規性|誤差が正規分布に近い|

---

## 線形性

回帰分析は、基本的に直線関係を仮定します。

もし関係が曲線なら、単純な線形回帰ではうまく表せません。

```text
Y
^
|           *
|       *
|    *
|  *
| *
+----------------> X
```

このような曲線関係なら、変数変換や非線形モデルを考える必要があります。

---

## 独立性

誤差が互いに独立であることです。

時系列データでは、誤差が連続して似た動きをすることがあります。

これを自己相関といいます。

ログ分析や競馬の時系列検証では、独立性を雑に仮定すると危険です。

---

## 等分散性

Xの値によって誤差のばらつきが変わらないことです。

たとえば、Xが大きくなるほどYのばらつきも大きくなる場合、等分散性が崩れます。

```text
Y
^
|             *     *
|        *       *
|    *     *
| * 
+----------------------> X
```

この場合、通常の回帰の標準誤差やp値が怪しくなることがあります。

---

## 正規性

係数の検定や信頼区間では、誤差の正規性が関係します。

ただし、サンプルサイズが大きいと中心極限定理によりある程度頑健になることもあります。

でも、強い外れ値や裾の厚い分布では注意が必要です。

---

# 25. 回帰出力の読み方

統計検定2級では、回帰出力表を読ませる問題があり得ます。

たとえば、次のような表です。

|変数|係数|標準誤差|t値|p値|
|---|--:|--:|--:|--:|
|切片|50.0|3.0|16.67|0.000|
|勉強時間|5.0|1.2|4.17|0.002|

ここで見るべきものは、

```text
係数
p値
符号
解釈
```

です。

勉強時間の係数が5.0なので、

```text
勉強時間が1時間増えると、点数は平均的に5点増える
```

と読みます。

p値が0.002なので、有意水準5%では有意です。

---

# 26. 回帰分析でやってはいけない解釈

## NG1：相関・回帰を因果と読む

```text
XとYに関係がある
```

ことは、

```text
XがYの原因である
```

を意味しません。

交絡変数があるかもしれません。

たとえば、

```text
アイスの売上と水難事故が相関する
```

としても、アイスが水難事故を起こしているわけではありません。

共通原因として気温があります。

---

## NG2：p値だけでモデルを評価する

p値が小さいから良いモデル、ではありません。

特にデータ数が多いと、実務的には小さい効果でも有意になります。

見るべきは、

```text
係数の大きさ
信頼区間
R²
残差
外れ値
予測性能
実務的意味
```

です。

---

## NG3：R²だけでモデルを評価する

R²が高くても、過学習しているかもしれません。

特に説明変数を大量に入れると、学習データではR²が高くなりやすいです。

未知データでの性能を確認する必要があります。

---

# 27. 競馬AI文脈での注意

あなたの文脈では、回帰分析はかなり使えます。

たとえば、

```text
市場心理スコアからROIを説明する
オッズ乖離から回収率を説明する
人気帯・馬場・距離からモデル誤差を説明する
市場バイアスクラスタから期待値誤差を説明する
```

ただし、ここは率直に言います。

競馬のROIや払戻を通常の線形回帰で雑に扱うのは危険です。

理由はこれです。

```text
ゼロが多い
外れ値が強い
右裾が長い
分散が一定ではない
同一レース内で依存がある
時系列で市場構造が変わる
```

つまり、線形回帰を使うなら、

```text
説明・診断用
```

として使うのはよいです。

でも、

```text
これで勝てるモデルができた
```

と短絡するのは危険です。

実運用では、回帰係数よりも、

```text
アウトオブサンプル性能
期間分割
ドローダウン
取引制約
外れ値依存
再現性
```

の方がはるかに重要です。

---

# 28. 統計検定2級での回帰分析

統計検定2級では、最低限これを押さえます。

```text
1. 目的変数と説明変数
2. 単回帰式 Y = β0 + β1X + ε
3. 切片と傾きの意味
4. 予測値と残差
5. 最小二乗法
6. 決定係数 R²
7. 回帰係数のt検定
8. モデル全体のF検定
9. 重回帰係数の解釈
10. 多重共線性
11. 回帰の前提
```

```text
α = 0.05
```

## 重要公式

単回帰モデル：

```text
Y = β0 + β1X + ε
```

推定された回帰式：

```text
Ŷ = b0 + b1X
```

残差：

```text
e = Y - Ŷ
```

残差平方和：

```text
SSE = Σ(Y - Ŷ)²
```

ばらつきの分解：

```text
SST = SSR + SSE
```

決定係数：

```text
R² = SSR / SST
```

回帰のF検定：

```text
F = モデルで説明できるばらつき / 説明できないばらつき
```

# 確率・分布・推測統計パートの総括

これで、第13回から第26回までの大きな流れが一通り終わりました。

このパートで本当に大事だったのは、公式を暗記することではありません。

本質はこれです。

```text
確率：不確実な出来事を割合・面積として扱う
確率変数：偶然の結果を数値化する
期待値・分散：確率分布の中心とばらつきを見る
確率分布：値の出方のルールを見る
標本分布：統計量のブレ方を見る
推定：標本から母数を幅つきで推測する
検定：仮説のもとで観測結果が珍しすぎるかを見る
回帰：変数間の関係を数式化する
```

かなり率直に言うと、ここまでを一度で完全理解するのは無理です。  
ただ、全体地図はかなりできています。

次にやるべきことは、講義を進めることではなく、**問題演習で穴を見つけること**です。  
特に弱点になりやすいのは、

```text
条件付き確率・ベイズ
標準化
標本分布
t / χ² / F の使い分け
信頼区間と検定の解釈
カイ二乗検定の期待度数
分散分析表
回帰出力の読み取り
```

ここです。

次は、ここまでの範囲を **演習セット形式** で固めるのが一番効率的です。
