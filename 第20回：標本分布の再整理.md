## 今日のゴール

今日のゴールはこれです。

> 標本分布とは、  
> 標本から計算した統計量の確率分布である。

もう少し噛み砕くと、

> 標本を何度も取り直したとき、  
> 標本平均や標本比率がどのようにブレるかを見る分布

です。

ここを理解すると、推定・仮説検定の意味がかなり見えます。

---

# 1. 母集団と標本

まず言葉を整理します。

## 母集団

母集団とは、

> 本当に知りたい対象全体

です。

例：

```text
全国の高校3年生全員
あるWebサイトの全アクセス
JRAの全レース
ある工場で作られる全製品
```

---

## 標本

標本とは、

> 母集団から取り出した一部のデータ

です。

例：

```text
高校3年生から選んだ100人
Webサイトの直近1万アクセス
JRA全レースのうち2025年のレース
工場で今日検査した50個の製品
```

現実には、母集団全体を調べられないことが多いです。

だから、標本から母集団を推測します。

---

# 2. 母数と統計量

次に、母数と統計量です。

## 母数

母数とは、

> 母集団の特徴を表す値

です。

代表例はこれです。

|母数|意味|
|---|---|
|母平均 `μ`|母集団全体の平均|
|母分散 `σ²`|母集団全体の分散|
|母比率 `p`|母集団全体の割合|

母数は、本当は知りたいけど、普通は直接わかりません。

---

## 統計量

統計量とは、

> 標本から計算した値

です。

代表例はこれです。

|統計量|意味|
|---|---|
|標本平均 `X̄`|標本から計算した平均|
|標本分散 `s²`|標本から計算した分散|
|標本比率 `p̂`|標本から計算した割合|

統計量は、実際に標本データから計算できます。

---

## 対応関係

|知りたい母数|使う統計量|
|---|---|
|母平均 `μ`|標本平均 `X̄`|
|母分散 `σ²`|標本分散 `s²`|
|母比率 `p`|標本比率 `p̂`|

つまり、統計では、

```text
母数は直接わからない
↓
標本から統計量を計算する
↓
統計量を使って母数を推測する
```

という流れになります。

---

# 3. 標本平均は毎回変わる

ここが標本分布の入口です。

たとえば、全国の高校3年生の平均身長を知りたいとします。

でも全員を測るのは大変なので、100人だけ選びます。

1回目に選んだ100人の平均：

```text
X̄ = 170.2cm
```

別の100人を選ぶと：

```text
X̄ = 169.7cm
```

さらに別の100人を選ぶと：

```text
X̄ = 170.5cm
```

このように、同じ母集団から選んでいるのに、標本平均は毎回少し変わります。

なぜなら、選ばれる人が毎回違うからです。

---

# 4. 標本平均も確率変数である

ここが重要です。

1人の身長 `X` が確率変数なら、  
100人の平均 `X̄` も確率変数です。

なぜなら、標本の取り方によって `X̄` の値が変わるからです。

つまり、

```text
標本平均 X̄ は、固定値ではない
標本を取るたびに変わる
だから確率変数である
```

ということです。

---

# 5. 標本分布とは何か

標本分布とは、

> 統計量の確率分布

です。

つまり、標本平均 `X̄` や標本比率 `p̂` などが、どのようにブレるかを表す分布です。

---

## イメージ

標本を何度も取り直します。

```text
1回目の標本 → X̄ = 170.2
2回目の標本 → X̄ = 169.7
3回目の標本 → X̄ = 170.5
4回目の標本 → X̄ = 169.9
...
```

この `X̄` たちを集めて分布を作る。

これが標本平均の標本分布です。

---

# 6. 元データの分布と標本分布は違う

ここはかなり混乱しやすいです。

## 元データの分布

元データの分布は、

> 個々のデータがどう散らばっているか

です。

たとえば、個人の身長。

```text
165, 172, 168, 180, 159, ...
```

これは、個人の身長の分布です。

---

## 標本平均の分布

標本平均の分布は、

> 標本平均がどう散らばるか

です。

```text
170.2, 169.7, 170.5, 169.9, ...
```

これは、個人の身長ではなく、平均値たちの分布です。

---

## 違い

|分布|何の分布か|
|---|---|
|元データの分布|個々のデータ `X` の分布|
|標本平均の分布|標本平均 `X̄` の分布|

前回の正規分布でいうと、

```text
X ~ N(μ, σ²)
```

と、

```text
X̄ ~ N(μ, σ²/n)
```

は違います。

ここは絶対に混ぜない方がいいです。

---

# 7. 標本平均の期待値

母平均を `μ` とします。

このとき、標本平均 `X̄` の期待値は、

```text
E(X̄) = μ
```

です。

意味は、

> 標本平均は毎回ブレるが、平均的には母平均を中心にしてブレる

ということです。

---

## 直感

ちゃんとランダムに標本を取れば、標本平均は母平均の周りに出ます。

もちろん1回1回はズレます。

でも、標本を何度も取り直して標本平均を集めると、その中心は母平均になります。

だから、

```text
標本平均は母平均を狙っている
```

と言えます。

---

# 8. 標本平均の分散

母分散を `σ²`、標本サイズを `n` とします。

このとき、標本平均 `X̄` の分散は、

```text
V(X̄) = σ² / n
```

です。

これは超重要です。

意味は、

> 標本サイズが大きくなるほど、標本平均のブレは小さくなる

です。

---

## 標本平均の標準偏差

分散が、

```text
σ² / n
```

なら、標準偏差は、

```text
σ / √n
```

です。

この `σ / √n` を **標準誤差** といいます。

---

# 9. 標準誤差とは何か

標準誤差とは、

> 統計量のブレの大きさ

です。

今回なら、

> 標本平均がどれくらいブレるか

を表します。

---

## 標準偏差と標準誤差の違い

|用語|何のばらつきか|
|---|---|
|標準偏差 `σ`|個々のデータのばらつき|
|標準誤差 `σ/√n`|標本平均のばらつき|

これが非常に重要です。

---

## 例

母集団の標準偏差が、

```text
σ = 20
```

だとします。

標本サイズが、

```text
n = 100
```

なら、

```text
標準誤差 = σ / √n
= 20 / √100
= 20 / 10
= 2
```

です。

つまり、

```text
個々のデータは標準偏差20くらいブレる
でも100個の平均は標準誤差2くらいしかブレない
```

ということです。

---

# 10. nを増やすと精度は上がるが、√で効く

標準誤差は、

```text
σ / √n
```

です。

だから、標本サイズ `n` を増やすと標準誤差は小さくなります。

ただし、効き方は平方根です。

たとえば、nを4倍にすると、

```text
√4 = 2
```

なので、標準誤差は半分になります。

nを100倍にすると、

```text
√100 = 10
```

なので、標準誤差は10分の1になります。

---

## ここは実務で大事

「データを増やせば精度が上がる」は正しいです。

でも、

```text
精度を2倍にしたいなら、データ数は4倍必要
精度を10倍にしたいなら、データ数は100倍必要
```

です。

データ量の増加には限界効用があります。

ここを甘く見ると、データを増やせば全部解決するという雑な判断になります。

---

# 11. 標本平均の分布

母集団が正規分布に従う場合、

```text
X ~ N(μ, σ²)
```

なら、標本平均は、

```text
X̄ ~ N(μ, σ²/n)
```

に従います。

つまり、

```text
標本平均の平均 = μ
標本平均の分散 = σ²/n
標本平均の標準偏差 = σ/√n
```

です。

---

# 12. 中心極限定理

ここで超重要な定理が出ます。

**中心極限定理**です。

## 中心極限定理とは

ざっくり言うと、

> 母集団の分布が正規分布でなくても、  
> 標本サイズnが十分大きければ、  
> 標本平均の分布は正規分布に近づく

という定理です。

---

## 何がすごいのか

元データが歪んでいても、標本平均は正規分布に近づきます。

たとえば、元データがこんな偏った分布でも、

```text
元データの分布

██████████
██████
███
█
```

標本平均を何度も取ると、

```text
標本平均の分布

        █
      ███
    █████
  ███████
███████████
  ███████
    █████
      ███
        █
```

のように、山型に近づいていきます。

---

# 13. 中心極限定理の意味

中心極限定理があるから、正規分布がとても重要になります。

なぜなら、現実のデータが正規分布でなくても、標本平均については正規分布を使って近似できることが多いからです。

だから、推定や検定で正規分布が何度も出てきます。

```text
元データが正規分布
→ 標本平均も正規分布

元データが正規分布でない
→ nが大きければ、標本平均はだいたい正規分布
```

この2段階で理解するとよいです。

---

# 14. 標本平均の標準化

標本平均を標準化するときは、普通の `X` とは少し違います。

個々の値 `X` を標準化するときは、

```text
Z = (X - μ) / σ
```

でした。

でも、標本平均 `X̄` を標準化するときは、

```text
Z = (X̄ - μ) / (σ / √n)
```

です。

---

## ここが危ない

割るものが違います。

|対象|標準化|
|---|---|
|個々の値 `X`|`(X - μ) / σ`|
|標本平均 `X̄`|`(X̄ - μ) / (σ/√n)`|

標本平均は、個々の値よりブレが小さい。  
だから、標準誤差 `σ/√n` で割ります。

---

# 15. 例題1：標準誤差

あるテストの点数について、母平均は60点、母標準偏差は20点である。  
この母集団から36人を無作為に選び、標本平均を計算する。

標本平均の標準誤差を求めよ。

---

## 解答

与えられているのは、

```text
σ = 20
n = 36
```

標準誤差は、

```text
σ / √n
= 20 / √36
= 20 / 6
= 3.333...
```

答えは、

```text
約3.33点
```

です。

---

# 16. 例題2：標本平均の確率

あるテストの点数について、母平均は60点、母標準偏差は20点である。  
36人を無作為に選んだとき、標本平均が65点以上になる確率を求めよ。

---

## Step 1：標本平均の分布を考える

```text
μ = 60
σ = 20
n = 36
```

標本平均の標準誤差は、

```text
σ / √n = 20 / 6 = 3.333...
```

したがって、

```text
X̄ ~ N(60, 20²/36)
```

です。

---

## Step 2：標準化する

求めたいのは、

```text
P(X̄ ≧ 65)
```

です。

標準化すると、

```text
Z = (X̄ - μ) / (σ / √n)
```

なので、

```text
Z = (65 - 60) / (20 / 6)
= 5 / 3.333...
= 1.5
```

したがって、

```text
P(X̄ ≧ 65) = P(Z ≧ 1.5)
```

---

## Step 3：正規分布表を使う

標準正規分布表より、

```text
P(Z ≦ 1.5) ≒ 0.9332
```

なので、

```text
P(Z ≧ 1.5)
= 1 - 0.9332
= 0.0668
```

答えは、

```text
約0.0668
```

つまり約6.68%です。

---

# 17. 個々の値と標本平均の比較

同じ設定で比較します。

```text
母平均 μ = 60
母標準偏差 σ = 20
標本サイズ n = 36
```

## 個人が65点以上を取る確率

```text
Z = (65 - 60) / 20
= 0.25
```

これはそれほど珍しくありません。

---

## 36人の平均が65点以上になる確率

```text
Z = (65 - 60) / (20/√36)
= 1.5
```

これはかなり珍しくなります。

---

## なぜか

個人の点数はかなりブレます。

でも36人の平均はそこまでブレません。

だから、平均が5点ズレることは、個人が5点ズレるよりずっと珍しいです。

ここが標本分布の核心です。

---

# 18. 標本比率の分布

標本平均だけでなく、標本比率も重要です。

たとえば、

```text
支持率
合格率
不良品率
クリック率
的中率
```

などです。

母比率を `p`、標本比率を `p̂` とします。

---

## 標本比率とは

たとえば、400人にアンケートを取り、120人が賛成したとします。

このとき、標本比率は、

```text
p̂ = 120 / 400 = 0.3
```

です。

---

## 標本比率の期待値と分散

母比率が `p`、標本サイズが `n` のとき、

```text
E(p̂) = p
```

```text
V(p̂) = p(1-p) / n
```

標準誤差は、

```text
√{p(1-p)/n}
```

です。

---

# 19. 標本比率の正規近似

標本サイズが十分大きいとき、標本比率 `p̂` はだいたい正規分布に従います。

```text
p̂ ≒ N(p, p(1-p)/n)
```

標準化すると、

```text
Z = (p̂ - p) / √{p(1-p)/n}
```

です。

これは母比率の推定や検定で使います。

---

# 20. 例題3：標本比率の標準誤差

ある商品の不良率は5%である。  
400個を検査したとき、不良品率 `p̂` の標準誤差を求めよ。

---

## 解答

```text
p = 0.05
n = 400
```

標準誤差は、

```text
√{p(1-p)/n}
= √{0.05 × 0.95 / 400}
= √{0.0475 / 400}
= √0.00011875
≒ 0.0109
```

答えは、

```text
約0.0109
```

つまり約1.09%です。

---

# 21. 標本分布と推定の接続

推定では、標本から母数を推測します。

たとえば、母平均 `μ` を知りたい。

でも、観測できるのは標本平均 `X̄` だけです。

ここで標本分布が必要になります。

```text
標本平均X̄は、母平均μの周りにどれくらいブレるか？
```

これがわかるから、

```text
X̄ ± 誤差幅
```

という信頼区間が作れます。

---

## 信頼区間の元ネタ

たとえば、母標準偏差 `σ` が既知で、標本サイズが `n` のとき、

```text
X̄ は μ の周りに σ/√n くらいブレる
```

そして正規分布では、

```text
±1.96標準誤差の範囲に約95%
```

が入ります。

だから、

```text
X̄ ± 1.96 × σ/√n
```

という95%信頼区間が出てきます。

---

# 22. 標本分布と仮説検定の接続

仮説検定でも標本分布を使います。

たとえば、

```text
H₀：μ = 60
```

と仮定します。

このとき、

```text
もし本当にμ=60なら、
標本平均X̄はどれくらいブレるはずか？
```

を考えます。

そのブレ方が標本分布です。

もし観測された `X̄` が、標本分布から見てかなり珍しい位置にあれば、

```text
H₀：μ = 60 は怪しい
```

と判断します。

つまり、仮説検定は、

> 帰無仮説のもとでの標本分布を使って、観測値が珍しすぎるかを見る方法

です。

---

# 23. 標本分布の全体地図

ここまでを整理すると、こうです。

```text
母集団
  ↓ 標本を取る
標本
  ↓ 統計量を計算する
標本平均 X̄ / 標本比率 p̂
  ↓ 標本を取り直すと値がブレる
標本分布
  ↓
推定・仮説検定に使う
```

もう少し具体的には、

```text
個々のデータ X の分布
↓
標本平均 X̄ の分布
↓
標準誤差 σ/√n
↓
信頼区間・検定統計量
```

です。

---

# 24. ここで t分布が出てくる理由

今日の話では、母標準偏差 `σ` がわかっている前提が多かったです。

でも現実には、母標準偏差 `σ` は普通わかりません。

そのときは、標本標準偏差 `s` を使います。

```text
σ の代わりに s を使う
```

すると、不確実性が増えます。

このとき出てくるのが、次回の **t分布** です。

---

## 位置づけ

```text
母標準偏差σが既知
→ 標本平均を正規分布で扱う

母標準偏差σが未知
→ sで代用する
→ t分布を使う
```

つまり、t分布は標本分布の自然な延長です。

---

# 25. 統計検定2級での標本分布

統計検定2級では、標本分布について最低限これを押さえます。

```text
1. 標本分布 = 統計量の分布
2. 標本平均 X̄ の期待値は μ
3. 標本平均 X̄ の分散は σ²/n
4. 標準誤差は σ/√n
5. 中心極限定理
6. 標本平均の標準化
7. 標本比率の分布
8. 推定・検定との接続
```

特に、

```text
個々の値Xはσで割る
標本平均X̄はσ/√nで割る
```

ここは頻出ミスです。

---

# 26. 練習問題

## 問題1

母平均 `μ = 100`、母標準偏差 `σ = 15` の母集団から、標本サイズ `n = 25` の標本を取る。

標本平均 `X̄` の期待値と標準誤差を求めよ。

---

### 解答

標本平均の期待値は、

```text
E(X̄) = μ = 100
```

標準誤差は、

```text
σ / √n = 15 / √25 = 15 / 5 = 3
```

答えは、

```text
期待値 100
標準誤差 3
```

です。

---

## 問題2

母平均 `μ = 100`、母標準偏差 `σ = 15`、標本サイズ `n = 25` のとき、標本平均が106以上になる確率を求めよ。

ただし、`P(Z ≦ 2) = 0.9772` とする。

---

### 解答

標準誤差は、

```text
σ / √n = 15 / 5 = 3
```

標準化します。

```text
Z = (X̄ - μ) / (σ/√n)
= (106 - 100) / 3
= 2
```

したがって、

```text
P(X̄ ≧ 106)
= P(Z ≧ 2)
= 1 - P(Z ≦ 2)
= 1 - 0.9772
= 0.0228
```

答えは、

```text
0.0228
```

です。

## 問題3

母標準偏差が `σ = 20` の母集団から、標本サイズを `n = 100` にした。

標本平均の標準誤差はいくらか。

---

### 解答

```text
σ / √n = 20 / √100 = 20 / 10 = 2
```

答えは、

```text
2
```

です。

## 問題4

標本サイズを4倍にすると、標本平均の標準誤差は何倍になるか。

---

### 解答

標準誤差は、

```text
σ / √n
```

です。

標本サイズを4倍にすると、

```text
σ / √(4n)
= σ / (2√n)
```

なので、標準誤差は半分になります。

答えは、

```text
1/2倍
```

です。

## 問題5

母比率 `p = 0.3`、標本サイズ `n = 400` のとき、標本比率 `p̂` の標準誤差を求めよ。

---

### 解答

標本比率の標準誤差は、

```text
√{p(1-p)/n}
```

です。

```text
√{0.3 × 0.7 / 400}
= √{0.21 / 400}
= √0.000525
≒ 0.0229
```

答えは、

```text
約0.0229
```

です。

## 重要公式

標本平均について、

```text
E(X̄) = μ
```

```text
V(X̄) = σ²/n
```

```text
SE(X̄) = σ/√n
```

標準化：

```text
Z = (X̄ - μ) / (σ/√n)
```

標本比率について、

```text
E(p̂) = p
```

```text
V(p̂) = p(1-p)/n
```

```text
SE(p̂) = √{p(1-p)/n}
```


ここを混ぜると、推定・検定で一気に壊れます。

```text
データ数を増やす効果は √n で効く
```

です。
