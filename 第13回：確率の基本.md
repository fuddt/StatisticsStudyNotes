
# 第13回：確率の基本

## 今日のゴール

今日のゴールはこれです。

> 確率とは、  
> 「起こりやすさ」を0〜1の数で表したもの。

もう少し言うと、

> 全体のうち、目的の出来事がどれくらいの割合で起こるか

を表したものです。

---

# 1. 確率とは何か

確率は、ある出来事がどれくらい起こりやすいかを表します。

たとえば、サイコロを1回振る。

「1の目が出る確率」は、

```text
1 / 6
```

です。

なぜなら、サイコロの目は全部で6通りあり、そのうち「1」は1通りだからです。

つまり、

```text
確率 = 起こってほしい場合の数 / 全体の場合の数
```

です。

---

# 2. 確率は0から1の間

確率は必ず、

```text
0 ≦ 確率 ≦ 1
```

です。

|確率|意味|
|--:|---|
|0|絶対に起こらない|
|0.5|半分くらい起こる|
|1|必ず起こる|

例：

```text
サイコロで7が出る確率 = 0
サイコロで偶数が出る確率 = 3/6 = 0.5
サイコロで1〜6のどれかが出る確率 = 1
```

ここで大事なのは、確率は「回数そのもの」ではなく「割合」だということです。

---

# 3. 確率を割合として見る

確率の基本イメージは、割合です。

たとえば、袋の中に玉が10個あります。

```text
赤玉：3個
青玉：7個
合計：10個
```

このとき、赤玉を引く確率は、

```text
3 / 10 = 0.3
```

です。

つまり、

```text
全体10個のうち、赤玉が3個ある
↓
赤玉の割合は30%
↓
赤玉を引く確率は0.3
```

です。

確率は、かなり素直に言うと、

> 全体の中で、その出来事が占める割合

です。

---

# 4. 確率を面積として見る

ここが少し重要です。

確率は「割合」ですが、図では「面積」として見ることもできます。

たとえば、全体を1つの四角形だとします。

```text
全体 = 1
```

その中で、赤玉を引く確率が0.3なら、全体の30%を赤く塗るイメージです。

```text
全体
┌────────────────────┐
│██████              │
└────────────────────┘
  赤玉 0.3
```

この塗られた部分の面積が確率です。

---

## なぜ面積で見るのか

今はまだピンとこなくていいですが、後で正規分布をやると、確率は面積として出てきます。

たとえば、

```text
身長が160cm以上170cm以下である確率
```

は、正規分布のグラフの下の面積で表します。

だから今のうちに、

```text
確率 = 全体の中の割合
確率 = 図で見ると面積
```

という感覚を持っておくと、後が楽になります。

---

# 5. 事象とは何か

確率では、「出来事」のことを **事象** と言います。

たとえば、サイコロを1回振るとします。

|事象|内容|
|---|---|
|A|1の目が出る|
|B|偶数が出る|
|C|3以上の目が出る|

こういう「出来事」が事象です。

---

# 6. 全事象

全事象とは、

> 起こりうるすべての結果

です。

サイコロなら、

```text
Ω = {1, 2, 3, 4, 5, 6}
```

です。

`Ω` はオメガと読みます。

「全部の世界」だと思えばいいです。

サイコロの全事象は6通り。

だから、

```text
P(Ω) = 1
```

です。

全事象は必ず起こるので、確率は1です。

---

# 7. 空事象

空事象とは、

> 絶対に起こらない事象

です。

サイコロで「7が出る」という事象を考えると、

```text
∅
```

です。

`∅` は空集合です。

サイコロでは7は出ないので、

```text
P(∅) = 0
```

です。

---

# 8. 余事象

余事象はかなり重要です。

余事象とは、

> ある事象が起こらないこと

です。

たとえば、サイコロで、

```text
A = 偶数が出る
```

とします。

偶数は、

```text
A = {2, 4, 6}
```

です。

この余事象は、

```text
Aではない = {1, 3, 5}
```

です。

つまり奇数が出ることです。

---

## 余事象の公式

事象Aの余事象を `Aᶜ` と書きます。

公式は、

```text
P(Aᶜ) = 1 - P(A)
```

です。

たとえば、偶数が出る確率は、

```text
P(A) = 3/6 = 1/2
```

なので、偶数ではない確率は、

```text
P(Aᶜ) = 1 - 1/2 = 1/2
```

です。

---

## 余事象が便利な場面

余事象は、

```text
少なくとも1回起こる
```

を計算するときに便利です。

たとえば、サイコロを2回振って、

```text
少なくとも1回は6が出る確率
```

を求めたい。

これを直接考えると、

```text
1回目が6
2回目が6
両方6
```

などが絡んで少し面倒です。

でも余事象を使うと、

```text
少なくとも1回6が出る
=
1回も6が出ない、の反対
```

です。

1回も6が出ない確率は、

```text
5/6 × 5/6 = 25/36
```

なので、

```text
少なくとも1回6が出る確率
= 1 - 25/36
= 11/36
```

です。

ここは頻出です。

---

# 9. 和事象

和事象とは、

> AまたはBが起こること

です。

記号では、

```text
A ∪ B
```

と書きます。

読み方は「AカップB」または「AまたはB」です。

---

## 例

サイコロで、

```text
A = 偶数が出る = {2, 4, 6}
B = 3以上が出る = {3, 4, 5, 6}
```

とします。

このとき、

```text
A ∪ B = {2, 3, 4, 5, 6}
```

です。

なぜなら、

```text
偶数である、または3以上である
```

なので、2,3,4,5,6が該当するからです。

---

# 10. 積事象

積事象とは、

> AかつBが起こること

です。

記号では、

```text
A ∩ B
```

と書きます。

読み方は「AキャップB」または「AかつB」です。

---

## 例

さっきと同じく、

```text
A = 偶数が出る = {2, 4, 6}
B = 3以上が出る = {3, 4, 5, 6}
```

とします。

このとき、

```text
A ∩ B = {4, 6}
```

です。

なぜなら、

```text
偶数であり、かつ3以上
```

なのは、4と6だからです。

---

# 11. 図で見る：和事象と積事象

集合の図で見るとわかりやすいです。

```text
AまたはB：A ∪ B

┌─────────────────┐
│      █████      │
│   ███████████   │
│      █████      │
└─────────────────┘
AとBのどちらかに入っている部分全部
```

```text
AかつB：A ∩ B

┌─────────────────┐
│        ___      │
│      ██   ██    │
│        ‾‾‾      │
└─────────────────┘
AとBが重なっている部分だけ
```

言葉で覚えるなら、

```text
和事象 = または
積事象 = かつ
```

です。

---

# 12. 和事象の確率

和事象の確率には注意が必要です。

単純に、

```text
P(A ∪ B) = P(A) + P(B)
```

としてはいけない場合があります。

なぜなら、AとBが重なっていると、その部分を二重に数えてしまうからです。

---

## 正しい公式

```text
P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
```

です。

意味は、

```text
Aの確率
+
Bの確率
-
重なり部分
```

です。

---

## 例

サイコロで、

```text
A = 偶数が出る = {2, 4, 6}
B = 3以上が出る = {3, 4, 5, 6}
```

とします。

それぞれ、

```text
P(A) = 3/6
P(B) = 4/6
P(A ∩ B) = 2/6
```

です。

したがって、

```text
P(A ∪ B)
= P(A) + P(B) - P(A ∩ B)
= 3/6 + 4/6 - 2/6
= 5/6
```

です。

実際、

```text
A ∪ B = {2, 3, 4, 5, 6}
```

なので、5通り。  
確率は5/6です。

---

# 13. 排反

排反とは、

> AとBが同時には起こらないこと

です。

つまり、

```text
A ∩ B = ∅
```

です。

---

## 例

サイコロで、

```text
A = 1が出る
B = 2が出る
```

とします。

1回のサイコロで、1と2が同時に出ることはありません。

だからAとBは排反です。

このとき、

```text
P(A ∩ B) = 0
```

です。

---

## 排反なら和事象は足すだけ

排反なら、重なりがありません。

だから、

```text
P(A ∪ B) = P(A) + P(B)
```

でOKです。

たとえば、

```text
P(1が出る または 2が出る)
= 1/6 + 1/6
= 2/6
= 1/3
```

です。

---

# 14. 排反と独立は違う

ここはかなり危ないです。

排反と独立は、言葉の雰囲気が似ているけど別物です。

今日の段階では入口だけ押さえます。

---

## 排反

排反は、

> 同時に起こらない

です。

例：

```text
1回のサイコロで1が出る
1回のサイコロで2が出る
```

これは同時に起こらないので排反です。

---

## 独立

独立は、

> 一方が起こっても、もう一方の起こりやすさが変わらない

です。

例：

```text
1回目のサイコロで6が出る
2回目のサイコロで6が出る
```

1回目の結果は2回目に影響しません。  
だから独立です。

---

## 決定的な違い

|用語|意味|
|---|---|
|排反|同時に起こらない|
|独立|影響しない|

この2つを混同すると、確率で大きく崩れます。

特に危ないのは、

```text
排反だから独立
```

と思ってしまうことです。

これは基本的に間違いです。

---

# 15. 独立の入口

独立については第14回の条件付き確率で本格的に扱います。

今日は入口だけです。

AとBが独立なら、

```text
P(A ∩ B) = P(A)P(B)
```

です。

たとえば、サイコロを2回振って、

```text
1回目に6が出る
2回目に6が出る
```

確率は、

```text
1/6 × 1/6 = 1/36
```

です。

なぜなら、1回目と2回目は独立だからです。

---

# 16. 確率の基本公式まとめ

今回の重要公式をまとめます。

|内容|公式|
|---|---|
|余事象|`P(Aᶜ) = 1 - P(A)`|
|和事象|`P(A ∪ B) = P(A) + P(B) - P(A ∩ B)`|
|排反の和事象|`P(A ∪ B) = P(A) + P(B)`|
|独立の積事象|`P(A ∩ B) = P(A)P(B)`|

ただし、最後の独立はまだ入口です。  
本格的には次回やります。

---

# 17. 練習問題

## 問題1

サイコロを1回振る。  
偶数が出る確率を求めよ。

---

### 解答

偶数は、

```text
{2, 4, 6}
```

の3通り。

全体は、

```text
{1, 2, 3, 4, 5, 6}
```

の6通り。

したがって、

```text
3/6 = 1/2
```

答えは、

```text
1/2
```

です。

---

## 問題2

サイコロを1回振る。  
3以上の目が出る確率を求めよ。

---

### 解答

3以上は、

```text
{3, 4, 5, 6}
```

の4通り。

全体は6通り。

```text
4/6 = 2/3
```

答えは、

```text
2/3
```

です。

---

## 問題3

サイコロを1回振る。  
偶数または3以上が出る確率を求めよ。

---

### 解答

```text
A = 偶数 = {2, 4, 6}
B = 3以上 = {3, 4, 5, 6}
```

和事象は、

```text
A ∪ B = {2, 3, 4, 5, 6}
```

したがって、5通り。

```text
P(A ∪ B) = 5/6
```

答えは、

```text
5/6
```

です。

公式で解くなら、

```text
P(A) = 3/6
P(B) = 4/6
P(A ∩ B) = 2/6
```

なので、

```text
P(A ∪ B)
= 3/6 + 4/6 - 2/6
= 5/6
```

です。

---

## 問題4

サイコロを2回振る。  
少なくとも1回は6が出る確率を求めよ。

---

### 解答

これは余事象を使うと楽です。

```text
少なくとも1回6が出る
```

の反対は、

```text
1回も6が出ない
```

です。

1回のサイコロで6が出ない確率は、

```text
5/6
```

2回とも6が出ない確率は、

```text
5/6 × 5/6 = 25/36
```

したがって、

```text
少なくとも1回6が出る確率
= 1 - 25/36
= 11/36
```

答えは、

```text
11/36
```

です。

---

## 問題5

サイコロを1回振る。

```text
A = 1が出る
B = 2が出る
```

このとき、AとBは排反か？

---

### 解答

排反です。

1回のサイコロで、1と2が同時に出ることはありません。

したがって、

```text
A ∩ B = ∅
```

です。

# 19. 今日の盲点

今回の最大の盲点はこれです。

```text
「または」は、単純に足せばいいとは限らない
```

AとBが重なっていると、二重に数えてしまいます。

だから基本は、

```text
P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
```

です。

もう1つ危ないのはこれです。

```text
排反と独立を混同する
```

これは本当に危ない。

|用語|判断|
|---|---|
|排反|同時に起こるか？|
|独立|確率に影響するか？|

この2つは別の概念です。

---

# 20. 最低限の暗記

今回、最低限暗記するならこれです。

```text
確率 = 全体のうち、その事象が起こる割合
```

```text
余事象：P(Aᶜ) = 1 - P(A)
```

```text
和事象：P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
```

```text
排反：同時に起こらない
```

```text
独立：一方が起こっても、もう一方の確率が変わらない
```

次回の第14回は **条件付き確率** です。  
ここで「条件が変わると確率が変わる」という、確率で一番誤解しやすい部分に入ります。
