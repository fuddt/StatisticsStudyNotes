# 第25回：分散分析

## 今日のゴール

今日のゴールはこれです。

> 分散分析とは、  
> 複数グループの平均に差があるかを、ばらつきの比で判断する方法。

名前に「分散」と入っていますが、目的は多くの場合、

```text
平均に差があるかを調べる
```

ことです。

ここが最初の混乱ポイントです。

---

# 1. なぜ「平均の差」なのに分散分析なのか

たとえば、3つの勉強法があります。

```text
勉強法A
勉強法B
勉強法C
```

それぞれの平均点が違うかを調べたい。

普通に考えると、

```text
平均を比べるなら平均分析では？
```

と思うかもしれません。

でも、分散分析ではこう考えます。

> グループ平均の違いが、  
> グループ内の自然なばらつきに比べて大きいか？

つまり、平均差そのものを見るのではなく、

```text
グループ間のばらつき
÷
グループ内のばらつき
```

を見ます。

だから「分散分析」と呼ばれます。

---

# 2. t検定ではダメなのか

2グループの平均差なら、t検定でよいです。

たとえば、

```text
AクラスとBクラスの平均点に差があるか
```

なら、t検定で扱えます。

でも、3グループ以上になると問題が起きます。

たとえば、

```text
A, B, C の3グループ
```

なら、ペアごとにt検定をすると、

```text
A vs B
A vs C
B vs C
```

の3回検定することになります。

グループが4つなら、

```text
A vs B
A vs C
A vs D
B vs C
B vs D
C vs D
```

で6回です。

検定回数が増えると、偶然に有意になる確率が増えます。

これを **多重比較の問題** といいます。

---

# 3. 分散分析の考え方

分散分析では、まず全体のばらつきを2つに分けます。

|種類|意味|
|---|---|
|群間のばらつき|グループ平均同士の違い|
|群内のばらつき|同じグループ内の個体差・誤差|

---

## 例

3つの勉強法の平均点がこうだったとします。

```text
A：70点
B：71点
C：69点
```

グループ平均は近いです。

この場合、群間のばらつきは小さい。

一方で、

```text
A：60点
B：75点
C：90点
```

なら、グループ平均が大きく違います。

群間のばらつきは大きい。

でも、ここでさらに大事なのは、群内のばらつきです。

---

# 4. 群内のばらつき

たとえば、A・B・Cの平均が違っていても、各グループ内の点数がものすごくバラバラなら、平均差は偶然かもしれません。

例：

```text
A：40, 80, 60
B：45, 90, 90
C：30, 100, 140
```

この場合、平均は違って見えても、グループ内のばらつきがかなり大きい。

だから、

```text
本当に勉強法の差なのか？
単なる個人差では？
```

と疑う必要があります。

---

# 5. 群間と群内を比べる

分散分析の本質はこれです。

```text
F = 群間のばらつき / 群内のばらつき
```

より正確には、

```text
F = 群間平均平方 / 群内平均平方
```

です。

---

## F値の意味

|F値|解釈|
|--:|---|
|1に近い|群間差は群内ばらつきと同程度|
|大きい|群間差が群内ばらつきより大きい|
|とても大きい|グループ平均に差がありそう|

つまり、

```text
F値が大きいほど、平均に差がある可能性が高い
```

です。

ただし、どれくらい大きければ有意かはF分布で判断します。

---

# 6. 分散分析の仮説

一元配置分散分析では、仮説はこうなります。

3グループA・B・Cの母平均を、

```text
μA, μB, μC
```

とします。

帰無仮説は、

```text
H0：μA = μB = μC
```

です。

つまり、

```text
すべてのグループの母平均は等しい
```

という仮説です。

対立仮説は、

```text
H1：少なくとも1つの母平均は異なる
```

です。

ここで注意。

対立仮説は、

```text
全部の平均が違う
```

ではありません。

> 少なくとも1つ違う

です。

たとえば、

```text
AとBは同じだが、Cだけ違う
```

でも対立仮説に含まれます。

---

# 7. 一元配置分散分析とは

一元配置分散分析とは、

> 1つの要因によって複数グループの平均が違うかを見る分散分析

です。

例：

|要因|グループ|
|---|---|
|勉強法|A法、B法、C法|
|薬の種類|薬A、薬B、薬C|
|広告種類|広告A、広告B、広告C|
|馬場状態|良、稍重、重、不良|
|人気帯|上位、中位、下位|

「一元」とは、見る要因が1つという意味です。

---

# 8. 具体例

3つの勉強法について、各5人の点数を調べたとします。

|勉強法A|勉強法B|勉強法C|
|--:|--:|--:|
|68|72|82|
|70|75|85|
|69|73|83|
|71|76|86|
|72|74|84|

グループ平均は、

```text
A平均 = 70
B平均 = 74
C平均 = 84
```

全体平均は、

```text
全体平均 = 76
```

です。

このとき、Cはかなり高く見えます。

でも、それが偶然なのか、勉強法の違いなのかを判断したい。

そこで分散分析を使います。

---

# 9. 全体のばらつきを分解する

分散分析では、全体のばらつきを次のように分けます。

```text
全体のばらつき
=
群間のばらつき
+
群内のばらつき
```

記号では、

```text
SST = SSB + SSW
```

と書くことがあります。

|記号|意味|
|---|---|
|SST|全体平方和|
|SSB|群間平方和|
|SSW|群内平方和|

---

# 10. 平方和とは何か

平方和とは、

> ズレを二乗して足したもの

です。

分散でも出てきました。

```text
ズレ
↓
二乗する
↓
足す
```

です。

なぜ二乗するかは、プラス・マイナスのズレが打ち消し合わないようにするためです。

---

# 11. 全体平方和 SST

全体平方和は、

> 各データが全体平均からどれくらいズレているか

を表します。

```text
SST = Σ(各データ - 全体平均)²
```

全体のばらつきです。

---

# 12. 群間平方和 SSB

群間平方和は、

> 各グループ平均が全体平均からどれくらいズレているか

を表します。

```text
SSB = Σ 各群の人数 × (群平均 - 全体平均)²
```

グループ平均の違いが大きいほど、SSBは大きくなります。

---

## 例で計算

さっきの例では、

```text
A平均 = 70
B平均 = 74
C平均 = 84
全体平均 = 76
各群の人数 = 5
```

なので、

```text
SSB
= 5(70 - 76)² + 5(74 - 76)² + 5(84 - 76)²
= 5×36 + 5×4 + 5×64
= 180 + 20 + 320
= 520
```

です。

---

# 13. 群内平方和 SSW

群内平方和は、

> 各データが、自分のグループ平均からどれくらいズレているか

を表します。

```text
SSW = Σ(各データ - そのデータが属する群の平均)²
```

これは、グループ内の個体差・誤差を表します。

---

## 例で計算

勉強法A：

```text
68, 70, 69, 71, 72
```

A平均は70。

ズレは、

```text
-2, 0, -1, 1, 2
```

二乗すると、

```text
4, 0, 1, 1, 4
```

合計は、

```text
10
```

---

勉強法B：

```text
72, 75, 73, 76, 74
```

B平均は74。

ズレは、

```text
-2, 1, -1, 2, 0
```

二乗すると、

```text
4, 1, 1, 4, 0
```

合計は、

```text
10
```

---

勉強法C：

```text
82, 85, 83, 86, 84
```

C平均は84。

ズレは、

```text
-2, 1, -1, 2, 0
```

二乗すると、

```text
4, 1, 1, 4, 0
```

合計は、

```text
10
```

---

したがって、

```text
SSW = 10 + 10 + 10 = 30
```

です。

---

# 14. 自由度

分散分析では、平方和を自由度で割って平均平方を作ります。

自由度はこうです。

|種類|自由度|
|---|--:|
|群間|グループ数 - 1|
|群内|全データ数 - グループ数|
|全体|全データ数 - 1|

今回、

```text
グループ数 k = 3
全データ数 N = 15
```

なので、

|種類|自由度|
|---|--:|
|群間|`3 - 1 = 2`|
|群内|`15 - 3 = 12`|
|全体|`15 - 1 = 14`|

です。

---

# 15. 平均平方

平均平方は、

```text
平方和 / 自由度
```

です。

|種類|平方和|自由度|平均平方|
|---|--:|--:|--:|
|群間|520|2|260|
|群内|30|12|2.5|

つまり、

```text
群間平均平方 MSB = 520 / 2 = 260
```

```text
群内平均平方 MSW = 30 / 12 = 2.5
```

です。

---

# 16. F値を計算する

F値は、

```text
F = 群間平均平方 / 群内平均平方
```

です。

今回、

```text
F = 260 / 2.5 = 104
```

です。

かなり大きいです。

つまり、

```text
群間のばらつきが、群内のばらつきに比べて非常に大きい
```

ということです。

---

# 17. F分布で判断する

分散分析では、このF値をF分布と比較します。

今回の自由度は、

```text
分子自由度 = 群間自由度 = 2
分母自由度 = 群内自由度 = 12
```

です。

つまり、

```text
F(2, 12)
```

を使います。

有意水準5%で、F(2,12)の上側5%点を仮に、

```text
3.89
```

とします。

今回のF値は、

```text
104
```

なので、

```text
104 > 3.89
```

です。

棄却域に入ります。

---

# 18. 結論

```text
帰無仮説 H0：μA = μB = μC を棄却する。
有意水準5%で、少なくとも1つの勉強法の平均点は異なると言える。
```

ただし、ここで注意です。

分散分析で分かるのは、

```text
少なくとも1つの平均が違う
```

ということだけです。

どのグループとどのグループが違うかまでは、分散分析だけでは分かりません。

---

# 19. どこが違うかは多重比較

分散分析で有意になったあと、

```text
AとBが違うのか
AとCが違うのか
BとCが違うのか
```

を見たい場合は、**多重比較** を行います。

代表例は、

```text
Tukey法
Bonferroni法
Dunnett法
```

などです。

統計検定2級では、分散分析の基本構造を優先すればよいですが、

```text
ANOVAで有意 = 全ペアが違う
```

ではないことは必ず押さえます。

---

# 20. 分散分析表

分散分析の結果は、よく次のような表で示されます。

|変動要因|平方和|自由度|平均平方|F値|
|---|--:|--:|--:|--:|
|群間|520|2|260|104|
|群内|30|12|2.5||
|全体|550|14|||

この表を読めることが重要です。

# 21. 分散分析の前提

分散分析には前提があります。

主にこの3つです。

|前提|意味|
|---|---|
|独立性|各観測値が独立である|
|正規性|各群のデータが正規分布に近い|
|等分散性|各群の母分散が等しい|

---

## 独立性

観測値が互いに独立であることです。

たとえば、同じ人を何度も測っているのに、別人として扱うと独立性が崩れます。

競馬でいうと、同じレース内の馬を独立な観測として扱うと危ない場合があります。

なぜなら、同じレース内の馬は同じ馬場・展開・相手関係を共有するからです。

---

## 正規性

各グループのデータが正規分布に近いことです。

ただし、サンプルサイズがある程度大きければ、分散分析はある程度頑健です。

とはいえ、極端な外れ値や強い歪みがある場合は注意が必要です。

---

## 等分散性

各グループの分散が同じくらいであることです。

たとえば、A群はばらつきが小さいが、C群だけ極端にばらつきが大きい場合、通常の分散分析は危うくなります。

等分散性が怪しい場合は、Welchの分散分析などを使うことがあります。

---

# 22. なぜ分散分析でF分布を使うのか

分散分析では、

```text
F = 群間平均平方 / 群内平均平方
```

を使いました。

帰無仮説、

```text
H0：すべての母平均は等しい
```

が正しいなら、群間平均平方も群内平均平方も、同じ誤差分散を見積もっていると考えられます。

つまり、F値はだいたい1に近くなるはずです。

でも、グループ平均が大きく違うと、群間平均平方が大きくなります。

するとF値が大きくなります。

そのF値が、帰無仮説のもとでどれくらい珍しいかをF分布で判断します。

---

# 23. 分散分析とt検定の関係

2群だけの場合、一元配置分散分析とt検定は深く関係しています。

実は、2群の場合には、

```text
F = t²
```

という関係があります。

つまり、2群の平均差を見るだけなら、t検定でも分散分析でも同じような結論になります。

ただし、3群以上になると、分散分析の役割が大きくなります。

---

# 24. 競馬AI文脈での使い方

あなたの文脈なら、分散分析は次のような問いに使えます。

```text
馬場状態ごとにROI平均は違うか
人気帯ごとにモデル誤差平均は違うか
距離帯ごとにオッズ乖離平均は違うか
クラスタごとに期待値誤差は違うか
開催場ごとに市場バイアス指標の平均は違うか
```

ただし、注意が必要です。

---

## 注意1：ROIや払戻金は正規性が怪しい

競馬の払戻・ROIは、だいたい正規分布ではありません。

```text
ゼロが多い
右に長い
外れ値が強い
少数の高配当が平均を支配する
```

このようなデータに対して通常の分散分析をそのまま使うのは危険です。

---

## 注意2：有意差があっても経済価値とは限らない

たとえば、

```text
馬場状態ごとに平均ROIが有意に違う
```

としても、それだけで、

```text
その馬場で買えば儲かる
```

とは言えません。

見るべきは、

```text
平均差
信頼区間
分散
最大ドローダウン
期間分割の再現性
サンプルサイズ
外れ値依存
```

です。

検定は入口であって、投資判断そのものではありません。

---

## 注意3：カテゴリを後から作ると危険

分析後に、

```text
この分け方なら有意だった
```

とカテゴリを作ると、後出し最適化になります。

競馬AIの研究ではここが特に危ない。

カテゴリ設計は、できるだけ事前に固定した方がよいです。

---

# 25. 統計検定2級での分散分析

統計検定2級では、最低限これを押さえます。

```text
1. 分散分析は複数群の平均差を見る
2. 帰無仮説はすべての母平均が等しい
3. 対立仮説は少なくとも1つ異なる
4. 全体のばらつきを群間と群内に分解する
5. F = 群間平均平方 / 群内平均平方
6. 群間自由度 = k - 1
7. 群内自由度 = N - k
8. ANOVAで有意でも、どの群が違うかは別問題
9. 独立性・正規性・等分散性の前提
```

特に分散分析表の読み取りは重要です。

---

# 26. 練習問題

## 問題1

一元配置分散分析における帰無仮説として正しいものを答えよ。

3群の母平均を `μ1, μ2, μ3` とする。

---

### 解答

帰無仮説は、

```text
H0：μ1 = μ2 = μ3
```

です。

すべての群の母平均が等しい、という仮説です。

---

## 問題2

一元配置分散分析における対立仮説を簡潔に説明せよ。

---

### 解答

対立仮説は、

```text
少なくとも1つの母平均が異なる
```

です。

すべてが違う、という意味ではありません。

---

## 問題3

グループ数 `k = 4`、全データ数 `N = 40` の一元配置分散分析で、群間自由度と群内自由度を求めよ。

---

### 解答

群間自由度は、

```text
k - 1 = 4 - 1 = 3
```

群内自由度は、

```text
N - k = 40 - 4 = 36
```

答えは、

```text
群間自由度：3
群内自由度：36
```

です。

---

## 問題4

次の分散分析表の空欄を埋めよ。

|変動要因|平方和|自由度|平均平方|F値|
|---|--:|--:|--:|--:|
|群間|90|3|?|?|
|群内|120|24|?||
|全体|210|27|||

---

### 解答

群間平均平方は、

```text
90 / 3 = 30
```

群内平均平方は、

```text
120 / 24 = 5
```

F値は、

```text
30 / 5 = 6
```

したがって、

|変動要因|平方和|自由度|平均平方|F値|
|---|--:|--:|--:|--:|
|群間|90|3|30|6|
|群内|120|24|5||
|全体|210|27|||

です。

---

## 問題5

分散分析で有意差が出た。  
このとき、「すべてのグループ間に差がある」と言ってよいか。

---

### 解答

言ってはいけません。

分散分析で分かるのは、

```text
少なくとも1つの母平均が異なる
```

ということです。

どのグループ同士が違うかを知るには、多重比較が必要です。

# 27. 今日のまとめ

|用語|意味|
|---|---|
|分散分析|複数群の平均差をばらつきの比で見る方法|
|一元配置分散分析|1つの要因による平均差を見る分散分析|
|群間のばらつき|グループ平均同士のばらつき|
|群内のばらつき|同じグループ内の個体差・誤差|
|平方和|ズレの二乗和|
|平均平方|平方和を自由度で割ったもの|
|F値|群間平均平方と群内平均平方の比|
|多重比較|どの群同士が違うかを調べる手続き|

---

## 重要公式

全体のばらつき分解：

```text
SST = SSB + SSW
```

群間平方和：

```text
SSB = Σ n_j(群平均_j - 全体平均)²
```

群内平方和：

```text
SSW = Σ(各データ - 所属群の平均)²
```

自由度：

```text
群間自由度 = k - 1
```

```text
群内自由度 = N - k
```

平均平方：

```text
MSB = SSB / (k - 1)
```

```text
MSW = SSW / (N - k)
```

F値：

```text
F = MSB / MSW
```

# 28. 今日の盲点

今日の最大の盲点はこれです。

```text
分散分析は、名前は分散分析だが、主目的は複数群の平均差を見ること
```

ただし、平均差を直接見るのではなく、

```text
群間のばらつき / 群内のばらつき
```

として見る。

もう1つの盲点はこれです。

```text
ANOVAで有意 = どの群が違うか分かる
```

ではありません。

ANOVAで言えるのは、

```text
少なくとも1つ違う
```

までです。

どこが違うかは、多重比較が必要です。

# 29. 最低限の暗記

```text
分散分析 = 複数群の平均差をF分布で検定する方法
```

```

```

```text
群間自由度 = k - 1
```

```text
群内自由度 = N - k
```

```text
有意でも、どの群が違うかは多重比較しないと分からない
```
