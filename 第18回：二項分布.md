## 今日のゴール

今日のゴールはこれです。

> 二項分布とは、  
> 成功・失敗の試行を何回か繰り返したとき、  
> 成功回数がどう分布するかを表す分布。

たとえば、

```text
10回コインを投げて、表が何回出るか
20人に営業して、契約が何件取れるか
100個検査して、不良品が何個あるか
30レース買って、何回的中するか
```

こういう「成功回数」を扱うのが二項分布です。

---

# 1. 二項分布の入口

まず、1回だけの成功・失敗を考えます。

たとえば、くじを1回引く。

```text
当たり：成功
外れ：失敗
```

当たり確率を `p` とします。

このとき、確率変数 `X` をこう置きます。

```text
X = 1：成功
X = 0：失敗
```

これは前回やった0/1変数です。

|X|確率|
|--:|--:|
|1|p|
|0|1-p|

この1回勝負の分布を **ベルヌーイ分布** といいます。

---

# 2. ベルヌーイ分布から二項分布へ

ベルヌーイ分布は1回だけです。

二項分布は、それを `n` 回繰り返します。

たとえば、当たり確率 `p = 0.2` のくじを10回引く。

このとき、

```text
X = 10回中、当たりが出た回数
```

とします。

`X` が取りうる値は、

```text
0, 1, 2, 3, ..., 10
```

です。

この `X` が従う分布が二項分布です。

---

# 3. 二項分布の条件

二項分布になるには、基本的に4つの条件があります。

|条件|内容|
|---|---|
|試行回数が決まっている|`n` 回やる|
|各試行の結果は2択|成功か失敗|
|成功確率が一定|毎回 `p`|
|各試行が独立|前の結果が次の結果に影響しない|

この4つが大事です。

---

## 条件1：試行回数が決まっている

たとえば、

```text
10回コインを投げる
100個検査する
20人に営業する
```

のように、回数 `n` が決まっています。

---

## 条件2：成功か失敗の2択

結果は2種類に分けます。

```text
当たり / 外れ
合格 / 不合格
クリック / 非クリック
的中 / 不的中
不良品 / 良品
```

現実にはもっと複雑でも、目的に合わせて2択にします。

---

## 条件3：成功確率が一定

毎回の成功確率が同じ `p` である必要があります。

たとえば、コインが公平なら、

```text
表が出る確率 p = 0.5
```

が毎回同じです。

ただし、現実ではこの条件が崩れることがあります。

競馬で「30レース中何回的中するか」を二項分布で見る場合、すべてのレースで的中確率が同じとは限りません。

ここは実務ではかなり危ない仮定です。

---

## 条件4：各試行が独立

1回目の結果が2回目に影響しないことです。

コイン投げなら基本的に独立です。

```text
1回目に表が出た
↓
2回目に表が出る確率は変わらない
```

でも、玉を戻さずに引く場合は独立ではありません。

```text
赤玉を引いた
↓
袋の中の赤玉が減る
↓
次に赤玉を引く確率が変わる
```

この場合は、単純な二項分布ではありません。

---

# 4. 二項分布の表記

二項分布はこう書きます。

```text
X ~ B(n, p)
```

読み方は、

```text
X は、試行回数 n、成功確率 p の二項分布に従う
```

です。

たとえば、

```text
X ~ B(10, 0.3)
```

なら、

```text
10回試行して、各回の成功確率が0.3の二項分布
```

です。

---

# 5. 二項分布の確率

二項分布では、

```text
n回中、ちょうどx回成功する確率
```

を求めます。

公式はこれです。

```text
P(X = x) = nCx p^x (1-p)^(n-x)
```

この公式は見た目が重いですが、意味は分解できます。

|部分|意味|
|---|---|
|`nCx`|成功する場所の選び方|
|`p^x`|成功がx回起こる確率|
|`(1-p)^(n-x)`|失敗がn-x回起こる確率|

---

# 6. まず順番固定で考える

例として、

```text
n = 5
p = 0.3
x = 2
```

を考えます。

つまり、

```text
成功確率0.3の試行を5回行って、ちょうど2回成功する確率
```

です。

まず、成功・失敗の並びが次の1パターンに固定されているとします。

```text
成功 成功 失敗 失敗 失敗
```

この確率は、

```text
0.3 × 0.3 × 0.7 × 0.7 × 0.7
= 0.3^2 × 0.7^3
```

です。

---

# 7. 成功する場所は複数ある

でも、成功2回・失敗3回の並びはこれだけではありません。

たとえば、

```text
成功 失敗 成功 失敗 失敗
```

もあります。

```text
失敗 成功 失敗 成功 失敗
```

もあります。

5回中、成功する場所を2つ選べばよいので、パターン数は、

```text
5C2 = 10
```

です。

だから、全体の確率は、

```text
5C2 × 0.3^2 × 0.7^3
```

になります。

これが二項分布の公式です。

---

# 8. 例題1：ちょうど2回成功する確率

成功確率0.3の試行を5回行う。  
成功回数を `X` とする。

`P(X = 2)` を求めよ。

---

## 解答

```text
X ~ B(5, 0.3)
```

です。

公式は、

```text
P(X = x) = nCx p^x (1-p)^(n-x)
```

今回、

```text
n = 5
p = 0.3
x = 2
```

なので、

```text
P(X = 2)
= 5C2 × 0.3^2 × 0.7^3
```

それぞれ計算します。

```text
5C2 = 10
0.3^2 = 0.09
0.7^3 = 0.343
```

したがって、

```text
P(X = 2)
= 10 × 0.09 × 0.343
= 0.3087
```

答えは、

```text
0.3087
```

つまり約30.9%です。

---

# 9. 二項分布の期待値

二項分布では、期待値はこうです。

```text
E(X) = np
```

意味はかなり自然です。

成功確率 `p` の試行を `n` 回やるなら、平均的な成功回数は、

```text
n × p
```

です。

---

## 例

```text
X ~ B(10, 0.3)
```

なら、

```text
E(X) = 10 × 0.3 = 3
```

です。

つまり、10回やれば平均的には3回成功する、ということです。

---

# 10. 二項分布の分散

二項分布の分散はこうです。

```text
V(X) = np(1-p)
```

標準偏差は、

```text
SD(X) = √{np(1-p)}
```

です。

---

## 例

```text
X ~ B(10, 0.3)
```

なら、

```text
V(X) = 10 × 0.3 × 0.7 = 2.1
```

標準偏差は、

```text
SD(X) = √2.1 ≒ 1.45
```

です。

平均は3回ですが、実際にはその周りでブレます。

---

# 11. なぜ期待値がnpになるのか

これは0/1変数で考えると自然です。

10回の試行を、それぞれ0/1変数で表します。

```text
X1 = 1回目が成功なら1、失敗なら0
X2 = 2回目が成功なら1、失敗なら0
...
X10 = 10回目が成功なら1、失敗なら0
```

成功回数 `X` は、

```text
X = X1 + X2 + ... + X10
```

です。

各 `Xi` の期待値は、

```text
E(Xi) = p
```

なので、

```text
E(X)
= E(X1 + X2 + ... + X10)
= E(X1) + E(X2) + ... + E(X10)
= p + p + ... + p
= np
```

です。

期待値は独立でなくても足せますが、二項分布では独立が条件です。

---

# 12. なぜ分散がnp(1-p)になるのか

各0/1変数の分散は、前回やったように、

```text
V(Xi) = p(1-p)
```

です。

二項分布では各試行が独立なので、分散を足せます。

```text
V(X)
= V(X1 + X2 + ... + Xn)
= V(X1) + V(X2) + ... + V(Xn)
= np(1-p)
```

ここで独立が必要です。

独立でないと、分散は単純には足せません。

---

# 13. 二項分布の形

二項分布の形は、`n` と `p` で変わります。

## p = 0.5 の場合

成功と失敗が同じくらい起きるので、左右対称に近くなります。

```text
p = 0.5

        █
      █████
    █████████
  █████████████
    █████████
      █████
        █
```

---

## p が小さい場合

成功が少ない側に偏ります。

```text
p = 0.1

████████
████
██
█
```

成功回数0や1が多くなります。

---

## p が大きい場合

成功回数が多い側に偏ります。

```text
p = 0.9

        █
      ██
    ████
████████
```

成功回数がnに近いところに集まります。

---

# 14. 「ちょうど」と「以下」「以上」

二項分布の問題では、表現に注意が必要です。

|表現|数式|
|---|---|
|ちょうど2回成功|`P(X = 2)`|
|2回以下成功|`P(X ≦ 2)`|
|2回以上成功|`P(X ≧ 2)`|
|少なくとも1回成功|`P(X ≧ 1)`|
|1回も成功しない|`P(X = 0)`|

特に、

```text
少なくとも1回
```

は余事象を使うことが多いです。

---

# 15. 例題2：少なくとも1回成功

成功確率0.2の試行を5回行う。  
少なくとも1回成功する確率を求めよ。

---

## 解答

```text
X ~ B(5, 0.2)
```

求めたいのは、

```text
P(X ≧ 1)
```

です。

これは直接、

```text
P(X=1)+P(X=2)+...+P(X=5)
```

とやると面倒です。

余事象を使います。

```text
少なくとも1回成功
=
1回も成功しない、の反対
```

なので、

```text
P(X ≧ 1) = 1 - P(X = 0)
```

`P(X=0)` は、

```text
P(X=0)
= 5C0 × 0.2^0 × 0.8^5
= 1 × 1 × 0.8^5
= 0.32768
```

したがって、

```text
P(X ≧ 1)
= 1 - 0.32768
= 0.67232
```

答えは、

```text
0.67232
```

つまり約67.2%です。

---

# 16. 例題3：2回以下成功

成功確率0.3の試行を4回行う。  
成功回数を `X` とする。

`P(X ≦ 2)` を求めよ。

---

## 解答

```text
X ~ B(4, 0.3)
```

求めたいのは、

```text
P(X ≦ 2)
= P(X=0) + P(X=1) + P(X=2)
```

です。

---

### P(X=0)

```text
P(X=0)
= 4C0 × 0.3^0 × 0.7^4
= 1 × 1 × 0.2401
= 0.2401
```

---

### P(X=1)

```text
P(X=1)
= 4C1 × 0.3^1 × 0.7^3
= 4 × 0.3 × 0.343
= 0.4116
```

---

### P(X=2)

```text
P(X=2)
= 4C2 × 0.3^2 × 0.7^2
= 6 × 0.09 × 0.49
= 0.2646
```

---

### 合計

```text
P(X ≦ 2)
= 0.2401 + 0.4116 + 0.2646
= 0.9163
```

答えは、

```text
0.9163
```

です。

# 17. 二項分布と競馬での注意

競馬に二項分布を使うときは、かなり注意が必要です。

たとえば、

```text
30レース買って、何回的中するか
```

を二項分布で考えたくなります。

でも二項分布には、

```text
各試行の成功確率が一定
各試行が独立
```

という条件があります。

競馬では、この2つがかなり怪しいです。

---

## 成功確率が一定ではない

レースごとに的中確率は違います。

```text
堅いレース
荒れるレース
少頭数
多頭数
芝
ダート
人気集中
オッズ分散
```

全部違います。

だから、すべてのレースを同じ `p` とみなすのは雑です。

---

## 独立性も怪しい

レース結果そのものは独立に近くても、あなたのベット選定ロジックには依存が入ります。

たとえば、

```text
同じ市場バイアスを狙っている
同じ開催条件に偏っている
同じモデル誤差を共有している
同じオッズ形成の歪みに依存している
```

こういう場合、的中/不的中は完全独立とは言いづらい。

つまり、競馬で二項分布を使うなら、

```text
かなり強い近似を置いている
```

と自覚した方がいいです。

---

# 18. では競馬では無意味なのか

無意味ではありません。

二項分布は、まず荒い基準として使えます。

たとえば、

```text
的中率20%の戦略を30回試したら、
成功回数がどれくらいなら珍しいか
```

を見ることはできます。

ただし、それをそのまま「正確な現実」と見なすのは危険です。

実務的には、

```text
二項分布 = 理論上の基準線
実データ = その基準からどれくらいズレるか
```

という使い方がよいです。

あなたの研究文脈なら、二項分布を信じるというより、

```text
iid仮定を置いた場合の期待レンジ
```

として置くのが安全です。

---

# 19. 統計検定2級での二項分布

試験では、二項分布について主に次を押さえればよいです。

```text
1. 二項分布になる条件
2. X ~ B(n, p) の意味
3. P(X=x) の計算
4. E(X)=np
5. V(X)=np(1-p)
6. 少なくとも1回は余事象で解く
```

特に、`E(X)` と `V(X)` は狙われやすいです。

# 20. 練習問題

## 問題1

成功確率0.4の試行を10回行う。  
成功回数を `X` とする。

`X` は何分布に従うか。

---

### 解答

```text
X ~ B(10, 0.4)
```

です。

理由は、

```text
10回試行する
各回は成功/失敗
成功確率が0.4
各試行が独立と考える
```

からです。

---

## 問題2

```text
X ~ B(10, 0.4)
```

のとき、期待値と分散を求めよ。

---

### 解答

期待値は、

```text
E(X) = np = 10 × 0.4 = 4
```

分散は、

```text
V(X) = np(1-p)
= 10 × 0.4 × 0.6
= 2.4
```

答えは、

```text
期待値 4
分散 2.4
```

です。

---

## 問題3

成功確率0.5の試行を3回行う。  
ちょうど2回成功する確率を求めよ。

---

### 解答

```text
X ~ B(3, 0.5)
```

求めたいのは、

```text
P(X = 2)
```

です。

```text
P(X=2)
= 3C2 × 0.5^2 × 0.5^1
= 3 × 0.25 × 0.5
= 0.375
```

答えは、

```text
0.375
```

です。

---

## 問題4

成功確率0.1の試行を5回行う。  
少なくとも1回成功する確率を求めよ。

---

### 解答

```text
X ~ B(5, 0.1)
```

求めたいのは、

```text
P(X ≧ 1)
```

余事象を使います。

```text
P(X ≧ 1) = 1 - P(X = 0)
```

```text
P(X=0)
= 5C0 × 0.1^0 × 0.9^5
= 0.9^5
= 0.59049
```

したがって、

```text
P(X ≧ 1)
= 1 - 0.59049
= 0.40951
```

答えは、

```text
0.40951
```

つまり約41.0%です。

---

## 問題5

ある製品の不良率は2%である。  
100個を検査したとき、不良品の個数を `X` とする。

このとき、`X` は何分布に従うか。  
また、期待値を求めよ。

---

### 解答

不良品であることを成功と見ると、

```text
p = 0.02
n = 100
```

なので、

```text
X ~ B(100, 0.02)
```

です。

期待値は、

```text
E(X) = np = 100 × 0.02 = 2
```

答えは、

```text
X ~ B(100, 0.02)
期待値は2個
```

です。

# 22. 今日の盲点

今日の最大の盲点はこれです。

```text
二項分布は「成功回数」の分布である
```

1回の成功/失敗ではなく、**n回中の成功回数**です。

もう1つ危ないのは、

```text
二項分布には条件がある
```

という点です。

```text
n回固定
成功/失敗の2択
成功確率pが一定
各試行が独立
```

この条件が崩れると、二項分布をそのまま使うのは危険です。

特に現実データでは、

```text
pが一定ではない
独立ではない
```

ことがよくあります。

試験では条件に従えばいい。  
実務では仮定を疑う。

ここを分けるのが大事です。
