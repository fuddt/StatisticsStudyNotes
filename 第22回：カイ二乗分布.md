# 第22回：カイ二乗分布

## 今日のゴール

今日のゴールはこれです。

> カイ二乗分布とは、  
> 標準正規分布に従う変数を二乗して足し合わせたときに出てくる分布である。

ただ、最初からこの定義だけで理解しようとするとしんどいです。

もっと実用的に言うと、

> カイ二乗分布は、  
> 「ばらつき」や「観測値と期待値のズレ」を扱うための分布

です。

統計検定2級では、主に次の2つにつながります。

|用途|内容|
|---|---|
|母分散の推定・検定|標本分散から母分散を考える|
|カイ二乗検定|観測度数と期待度数のズレを見る|

今回はまず **カイ二乗分布そのもの** を理解します。  
カイ二乗検定は第24回で正式に扱います。

---

# 1. まず正規分布を思い出す

標準正規分布は、

```text
Z ~ N(0, 1)
```

でした。

これは、

```text
平均0
分散1
標準偏差1
```

の正規分布です。

正規分布は左右対称でした。

```text
        ／￣￣＼
      ／        ＼
_____/            \_____
        0
```

`Z` は、正にも負にもなります。

たとえば、

```text
Z = -2
Z = -1
Z = 0
Z = 1
Z = 2
```

のような値を取ります。

---

# 2. Zを二乗するとどうなるか

ここで、標準正規変数 `Z` を二乗します。

```text
Z²
```

すると、値は必ず0以上になります。

なぜなら、

```text
(-2)² = 4
(-1)² = 1
0² = 0
1² = 1
2² = 4
```

だからです。

つまり、二乗するとマイナスが消えます。

---

## 重要

`Z²` は、

> 平均0からどれくらいズレたかを、二乗で測ったもの

です。

たとえば、

```text
Z = 2
Z = -2
```

は方向は逆ですが、平均0からの距離は同じです。

二乗すると、

```text
Z² = 4
```

になります。

つまり、二乗は、

```text
方向ではなく、ズレの大きさを見る
```

操作です。

ここがカイ二乗分布の直感です。

---

# 3. カイ二乗分布の定義

標準正規分布に従う独立な確率変数を考えます。

```text
Z1, Z2, ..., Zk
```

それぞれが、

```text
N(0, 1)
```

に従うとします。

このとき、

```text
χ² = Z1² + Z2² + ... + Zk²
```

が従う分布を **自由度kのカイ二乗分布** といいます。

記号では、

```text
χ² ~ χ²(k)
```

と書きます。

読み方は、

```text
カイ二乗は、自由度kのカイ二乗分布に従う
```

です。

---

# 4. カイ二乗分布は0以上

カイ二乗分布は、二乗を足し合わせたものです。

だから、必ず0以上です。

```text
χ² ≥ 0
```

負の値にはなりません。

ここが正規分布やt分布と違うところです。

|分布|取る値|
|---|---|
|正規分布|負も正も取る|
|t分布|負も正も取る|
|カイ二乗分布|0以上だけ|

---

# 5. カイ二乗分布の形

カイ二乗分布は左右対称ではありません。

特に自由度が小さいと、右に長く伸びた形になります。

```text
自由度が小さいカイ二乗分布

█
██
███
████
██████
████
██
█
  \__________
      右に長い
```

正規分布のような左右対称の山型ではありません。

---

## 自由度が大きくなると

自由度が大きくなると、だんだん山型に近づきます。

```text
自由度が大きいカイ二乗分布

        ███
      ███████
    ███████████
  █████████████
███████████████
```

ただし、完全な標準正規分布ではありません。

---

# 6. 自由度とは何か

カイ二乗分布にも自由度があります。

自由度はざっくり言うと、

> 自由に動ける情報の数

です。

t分布でも自由度が出てきました。

母平均のt分布では、自由度はよく、

```text
n - 1
```

でした。

カイ二乗分布でも、母分散を扱うときには自由度 `n - 1` が出てきます。

---

# 7. なぜ分散と関係するのか

分散は、平均からのズレを二乗して平均したものでした。

```text
分散 = ズレ²の平均
```

一方、カイ二乗分布は、

```text
標準正規変数の二乗の合計
```

です。

つまり両方とも、

```text
ズレを二乗して扱う
```

という共通点があります。

ここで分散とカイ二乗分布がつながります。

---

# 8. 標本分散とカイ二乗分布

ここから統計検定2級で重要な形です。

母集団が正規分布に従うとします。

```text
X ~ N(μ, σ²)
```

この母集団から標本サイズ `n` の標本を取ります。

標本分散を `s²` とします。

このとき、次の統計量は自由度 `n-1` のカイ二乗分布に従います。

```text
χ² = (n - 1)s² / σ²
```

記号で書くと、

```text
(n - 1)s² / σ² ~ χ²(n - 1)
```

これはかなり重要です。

---

# 9. 公式の意味

```text
(n - 1)s² / σ²
```

を分解します。

|部分|意味|
|---|---|
|`s²`|標本分散|
|`σ²`|母分散|
|`s² / σ²`|標本分散が母分散に対してどれくらい大きいか|
|`n - 1`|自由度補正|

つまり、この統計量は、

> 標本分散が母分散からどれくらいズレているか

を見る量です。

---

# 10. 母平均ではなく母分散を見る

t分布では、母平均を扱いました。

```text
t = (X̄ - μ) / (s/√n)
```

これは、

> 標本平均が母平均からどれくらいズレているか

を見るものでした。

一方、カイ二乗分布では、

```text
χ² = (n - 1)s² / σ²
```

です。

これは、

> 標本分散が母分散に対してどれくらいズレているか

を見るものです。

---

# 11. 母分散の信頼区間

カイ二乗分布は、母分散の信頼区間で使います。

母平均の信頼区間では、

```text
X̄ ± t × s/√n
```

のような形でした。

母分散の場合は、左右対称ではありません。

なぜなら、カイ二乗分布は左右対称ではないからです。

---

## 母分散の信頼区間の形

正規母集団から標本サイズ `n` の標本を取り、標本分散を `s²` とします。

母分散 `σ²` の `100(1-α)%` 信頼区間は、

```text
((n-1)s² / χ²上側,  (n-1)s² / χ²下側)
```

のような形になります。

もう少し具体的に、95%信頼区間なら、

```text
((n-1)s² / χ²_{0.975},  (n-1)s² / χ²_{0.025})
```

のように書かれます。

ただし、表の形式によって添字の読み方が変わるので、試験では問題文や表の定義を必ず確認する必要があります。

---

# 12. なぜ上下が直感と逆っぽく見えるのか

母分散の信頼区間は、

```text
χ² = (n-1)s² / σ²
```

を変形して作ります。

この式では、`σ²` が分母にあります。

だから、カイ二乗値が大きいと、推定される `σ²` は小さくなります。

逆に、カイ二乗値が小さいと、推定される `σ²` は大きくなります。

ここがややこしい。

だから母分散の信頼区間では、上側・下側のカイ二乗値をどちらに入れるかで混乱しやすいです。

---

# 13. 母分散の検定

カイ二乗分布は、母分散の検定にも使います。

たとえば、

```text
この製品の重さの分散は本当に4以下なのか？
この工程のばらつきは以前より大きくなったのか？
測定誤差の分散は基準値と同じか？
```

のような問題です。

母分散の検定では、帰無仮説で仮定する母分散を `σ0²` として、

```text
χ² = (n - 1)s² / σ0²
```

を使います。

---

# 14. 例題1：カイ二乗統計量を計算する

ある正規母集団から、標本サイズ `n = 10` の標本を取った。  
標本分散は `s² = 6` だった。

帰無仮説で母分散を、

```text
σ0² = 4
```

とするとき、カイ二乗統計量を求めよ。

---

## 解答

使う公式は、

```text
χ² = (n - 1)s² / σ0²
```

です。

値を入れます。

```text
n = 10
s² = 6
σ0² = 4
```

なので、

```text
χ² = (10 - 1) × 6 / 4
   = 9 × 6 / 4
   = 54 / 4
   = 13.5
```

答えは、

```text
χ² = 13.5
```

です。

自由度は、

```text
n - 1 = 9
```

です。

---

# 15. カイ二乗分布表の読み方

カイ二乗分布表では、主に次を見ます。

```text
自由度
上側確率
```

たとえば、

```text
自由度9で、上側5%点
```

というのは、

> その値より大きくなる確率が5%になる点

です。

つまり、

```text
P(χ² ≥ ある値) = 0.05
```

となる値です。

---

## 注意

カイ二乗分布表は、表の形式がいくつかあります。

ある表は上側確率を載せています。  
別の表は下側確率を載せていることがあります。

だから必ず、

```text
この表は P(χ² ≧ x) を載せているのか？
この表は P(χ² ≦ x) を載せているのか？
```

を確認します。

ここは試験で地味に危ないです。

---

# 16. カイ二乗分布とカイ二乗検定

カイ二乗分布は、後で出てくるカイ二乗検定にも使います。

カイ二乗検定では、次のような統計量を使います。

```text
χ² = Σ (観測度数 - 期待度数)² / 期待度数
```

これは、

> 観測された度数が、期待される度数からどれくらいズレているか

を見る量です。

---

## 例

サイコロが公平なら、60回振ったとき各目はだいたい10回ずつ出るはずです。

でも実際には、

|目|観測度数|期待度数|
|--:|--:|--:|
|1|5|10|
|2|8|10|
|3|9|10|
|4|11|10|
|5|12|10|
|6|15|10|

だったとします。

このとき、

```text
観測度数と期待度数のズレ
```

を二乗して集計します。

それがカイ二乗検定です。

---

# 17. なぜ二乗するのか

カイ二乗検定でも二乗が出てきます。

```text
(観測度数 - 期待度数)²
```

です。

理由は分散と同じです。

ズレにはプラスとマイナスがあります。

```text
観測度数 - 期待度数 = +5
観測度数 - 期待度数 = -5
```

このまま足すと打ち消し合います。

だから二乗します。

```text
(+5)² = 25
(-5)² = 25
```

つまり、カイ二乗分布は、

```text
ズレを二乗して評価する
```

分布だと考えると理解しやすいです。

---

# 18. カイ二乗分布の2つの顔

ここまでを整理すると、カイ二乗分布には大きく2つの顔があります。

|顔|内容|
|---|---|
|分散の分布|標本分散から母分散を考える|
|ズレの分布|観測度数と期待度数のズレを見る|

統計検定2級では、両方出ます。

今回の第22回では、主に **分布としてのカイ二乗** を扱っています。

第24回のカイ二乗検定で、度数のズレとしての使い方を扱います。

---

# 19. カイ二乗分布の期待値と分散

自由度 `k` のカイ二乗分布について、

```text
χ² ~ χ²(k)
```

なら、期待値と分散は、

```text
E(χ²) = k
```

```text
V(χ²) = 2k
```

です。

統計検定2級では、計算でゴリゴリ使う場面は多くないかもしれませんが、性質として知っておくとよいです。

自由度が大きくなると、平均も分散も大きくなります。

---

# 20. t分布との関係

t分布にもカイ二乗分布が関係しています。

ざっくり言うと、t分布は、

```text
標準正規分布
÷
カイ二乗分布から来るばらつきの推定
```

によって作られます。

だから、t分布は「母標準偏差が未知」のときに出てきました。

つまり、t分布の背後には、

```text
標本分散の不確実性
```

があり、その標本分散の不確実性を扱うのがカイ二乗分布です。

---

# 21. F分布との関係

次回やるF分布も、カイ二乗分布から作られます。

F分布は、ざっくり言うと、

```text
2つのカイ二乗分布を、それぞれ自由度で割って比を取ったもの
```

です。

つまり、

```text
カイ二乗分布
↓
F分布
↓
分散比・分散分析・回帰
```

につながります。

だからカイ二乗分布は、今後のF分布・分散分析・回帰にもつながる土台です。

---

# 22. カイ二乗分布で何を覚えるべきか

統計検定2級では、まずこれを押さえればよいです。

```text
1. カイ二乗分布は0以上の分布
2. 左右対称ではなく、右に歪む
3. 自由度で形が変わる
4. 標準正規変数の二乗和として定義される
5. 標本分散と母分散の関係で出てくる
6. (n-1)s²/σ² ~ χ²(n-1)
7. 観測度数と期待度数のズレを見るカイ二乗検定につながる
```

---

# 23. 例題2：母分散の検定の判断

ある正規母集団から `n = 10` の標本を取った。  
標本分散は `s² = 6` であった。

母分散が4であるという帰無仮説を検定したい。

```text
H0：σ² = 4
H1：σ² > 4
```

有意水準5%で検定する。  
自由度9のカイ二乗分布の上側5%点を `16.919` とする。

---

## Step 1：検定統計量を計算する

```text
χ² = (n - 1)s² / σ0²
```

値を入れます。

```text
χ² = (10 - 1) × 6 / 4
   = 13.5
```

---

## Step 2：棄却域を見る

対立仮説は、

```text
σ² > 4
```

です。

つまり、標本分散が大きすぎるかを見ています。

右片側検定なので、棄却域は上側です。

```text
χ² > 16.919
```

---

## Step 3：判断する

今回の検定統計量は、

```text
χ² = 13.5
```

です。

```text
13.5 < 16.919
```

なので、棄却域に入りません。

---

## 結論

```text
帰無仮説 H0：σ² = 4 は棄却できない。
有意水準5%で、母分散が4より大きいとは言えない。
```

ここでも、

```text
母分散は4であると証明された
```

とは言いません。

あくまで、

```text
4より大きいと判断するほどの証拠はなかった
```

です。

---

# 24. カイ二乗分布の実務的な注意

カイ二乗分布を使う母分散の推定・検定では、かなり強い前提があります。

それは、

```text
母集団が正規分布である
```

という前提です。

標本平均については中心極限定理が強力でした。

でも、標本分散の分布は、母集団の分布形にかなり敏感です。

つまり、データが強く歪んでいたり、外れ値が多かったりすると、カイ二乗分布による母分散の推定・検定は危なくなります。

---

## 特に危ないデータ

```text
所得
売上
アクセス数
待ち時間
競馬の払戻金
ROI
損益
```

こういう右に歪んだデータや裾の厚いデータでは、正規母集団を仮定した分散検定はかなり危険です。

あなたの競馬AI文脈で言えば、払戻金やROIをそのまま正規分布・カイ二乗分布前提で扱うのは雑です。

分散そのものが外れ値に強く影響されるので、分散検定も外れ値に引っ張られます。

# 25. 練習問題

## 問題1

標準正規分布に従う独立な確率変数 `Z1, Z2, Z3` がある。

```text
Y = Z1² + Z2² + Z3²
```

とすると、`Y` は何分布に従うか。

---

### 解答

標準正規変数の二乗和なので、カイ二乗分布に従います。

3個足しているので自由度は3です。

```text
Y ~ χ²(3)
```

---

## 問題2

カイ二乗分布は負の値を取るか。

---

### 解答

取りません。

カイ二乗分布は、標準正規変数を二乗して足し合わせたものです。

二乗は必ず0以上なので、

```text
χ² ≥ 0
```

です。

---

## 問題3

正規母集団から `n = 16` の標本を取った。  
標本分散は `s² = 9` である。

母分散が `σ² = 12` のとき、

```text
(n-1)s² / σ²
```

を計算せよ。

---

### 解答

```text
n = 16
s² = 9
σ² = 12
```

なので、

```text
(n-1)s² / σ²
= (16 - 1) × 9 / 12
= 15 × 9 / 12
= 135 / 12
= 11.25
```

答えは、

```text
11.25
```

です。

自由度は、

```text
n - 1 = 15
```

です。

---

## 問題4

自由度 `k = 5` のカイ二乗分布の期待値と分散を求めよ。

---

### 解答

自由度 `k` のカイ二乗分布では、

```text
E(χ²) = k
```

```text
V(χ²) = 2k
```

です。

したがって、`k = 5` なら、

```text
E(χ²) = 5
```

```text
V(χ²) = 10
```

です。

---

## 問題5

カイ二乗分布を使って母分散を扱うとき、基本的にどのような母集団の仮定が必要か。

---

### 解答

基本的には、

```text
母集団が正規分布に従う
```

という仮定が必要です。

母集団が大きく歪んでいたり、外れ値が多かったりすると、カイ二乗分布を使った母分散の推定・検定は危険になります。

カイ二乗分布の定義：

```text

```

```text
χ² ~ χ²(k)
```

母分散との関係：

```text
(n - 1)s² / σ² ~ χ²(n - 1)
```

母分散の検定統計量：

```text
χ² = (n - 1)s² / σ0²
```

カイ二乗検定の入口：

```text
χ² = Σ (観測度数 - 期待度数)² / 期待度数
```

期待値と分散：

```text
E(χ²) = k
```

```text
V(χ²) = 2k
```
