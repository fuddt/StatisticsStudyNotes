前回の核心はこれでした。

> 標本の結果を、母集団の真実と混同しない。

今回は、その理由をもう少し深く見ます。

テーマは、

> 同じ母集団から標本を取っても、標本平均は毎回少し変わる

です。

ここが分かると、次に出てくる **標本分布・標準誤差・信頼区間** がかなり理解しやすくなります。

---

# 1. まずイメージで考える

全国の高校3年生の平均睡眠時間を知りたいとします。

本当の母平均が、仮に

```text
6.5時間
```

だったとします。

でも、私たちは全国の高校3年生全員を調べるわけではありません。

500人だけ調べます。

この500人の選ばれ方によって、平均睡眠時間は少し変わります。

たとえば、

|調査回|選ばれた500人の平均睡眠時間|
|---|--:|
|1回目|6.4時間|
|2回目|6.6時間|
|3回目|6.3時間|
|4回目|6.7時間|
|5回目|6.5時間|

こんな感じです。

本当の母平均は6.5時間でも、標本平均は毎回ぴったり6.5時間にはなりません。

理由は単純です。

> どの500人が選ばれるかによって、データが少し違うから。

---

# 2. 標本平均は「ランダムに動く値」

ここが重要です。

標本平均は、ただの平均ではありません。

**標本の取り方によって変わる値**です。

つまり、標本平均はランダムにブレます。

このブレのことを、ざっくり言うと、

> 標本誤差

と考えてよいです。

誤差と言っても、計算ミスではありません。

標本を一部だけ取る以上、自然に発生するズレです。

---

# 3. 例：5人だけ調べる場合

母集団として、10人の睡眠時間があるとします。

```text
5, 5, 6, 6, 6, 7, 7, 7, 8, 8
```

この10人全員の平均を計算します。

合計は、

```text
5 + 5 + 6 + 6 + 6 + 7 + 7 + 7 + 8 + 8 = 65
```

10人いるので、

```text
65 ÷ 10 = 6.5
```

つまり、母平均は **6.5時間** です。

---

では、この10人から3人だけ選んで平均を出します。

たとえば、

```text
5, 6, 7
```

を選んだら、標本平均は

```text
(5 + 6 + 7) ÷ 3 = 6
```

です。

別の3人として、

```text
7, 8, 8
```

を選んだら、

```text
(7 + 8 + 8) ÷ 3 = 7.67
```

になります。

さらに、

```text
6, 6, 7
```

なら、

```text
(6 + 6 + 7) ÷ 3 = 6.33
```

です。

つまり、同じ母集団から取っているのに、標本平均は変わります。

---

# 4. ここで勘違いしやすいこと

ここでよくある勘違いがあります。

> 標本平均が母平均と違うなら、その標本はダメなのでは？

違います。

標本平均が母平均とズレるのは普通です。

問題は、

> どれくらいズレそうか？

です。

統計では、この「ズレそうな幅」を考えます。

だから、1つの標本平均だけ見て断言しません。

---

# 5. 標本サイズが大きいほどブレは小さくなる

ここは直感的にも大事です。

3人だけ調べた平均と、500人調べた平均では、どちらが母平均に近そうですか？

普通は500人です。

なぜなら、人数が多いほど、たまたまの偏りがならされるからです。

たとえば、3人だけだと、

```text
たまたま寝不足の人ばかり選ばれる
```

ことがあります。

でも500人なら、寝不足の人も、よく寝る人も、普通の人も混ざりやすい。

だから平均は安定します。

まとめると、

|標本サイズ|標本平均のブレ|
|--:|---|
|小さい|大きい|
|大きい|小さい|

です。

---

# 6. 標本平均のブレを表すもの

ここから、少し統計検定2級っぽくなります。

標本平均のブレの大きさを表す指標があります。

それが、

> 標準誤差

です。

標準誤差は英語で **Standard Error** と言います。

よく **SE** と書きます。

ざっくり言えば、

> 標本平均がどれくらいブレるかを表す数値

です。

---

# 7. 標準誤差の式

母集団の標準偏差を σ、標本サイズを n とすると、標本平均の標準誤差は次の式です。

SE=\frac{\sigma}{\sqrt{n}}

読み方は、

```text
標準誤差 = 母標準偏差 ÷ ルートn
```

です。

まだ式が怖く見えるかもしれませんが、意味は単純です。

---

# 8. 式の意味

式の中身を見ます。

```text
SE = σ / √n
```

ここで、

- σ：母集団そのもののばらつき
    
- n：標本サイズ
    
- √n：標本サイズの平方根
    

です。

つまり、

> 元のデータのばらつきが大きいほど、標本平均もブレやすい  
> 標本サイズが大きいほど、標本平均はブレにくい

という意味です。

かなり自然な話です。

標本サイズが大きいほど標本平均が安定する理由は、もう少し分解するとこうです。


標本サイズが大きい
↓
たまたま極端なデータだけが選ばれる影響が薄まる
↓
高い値・低い値が平均の中でならされる
↓
標本平均が母平均の近くに集まりやすくなる
↓
標準誤差が小さくなる


###  補足のコラム：有限の母集団と無限の母集団
母集団が100人で、点数がこういう分布だとします。

```
低い人もいる普通の人もいる高い人もいる
```

10人だけ取ると、たまたま高い人が多く入るかもしれません。

```
10人標本A：高得点者多め → 平均が高くなる10人標本B：低得点者多め → 平均が低くなる10人標本C：普通くらい → 平均が中くらい
```

だから、10人標本の平均はブレます。

でも99人取ると、ほぼ全員です。

1人しか抜けません。

だから、どの99人を選んでも、平均はほとんど同じになります。

```
99人標本A：1人だけ抜ける99人標本B：別の1人だけ抜ける99人標本C：また別の1人だけ抜ける
```

この場合、平均はほぼ母平均です。
ただし「母集団が100人で99人取る」は有限母集団の話です。

たとえば、母平均の推定が「未来の事象」に対してだったとしましょう。

未来の事象は、100人の母集団のように固定された有限集合ではありません。
そのため、過去データをどれだけ増やしても、
「母集団のほとんどを見た」とは言えません。

ただし、標本サイズを増やす意味がないわけではありません。

標本サイズが大きくなれば、標本平均のランダムなブレは小さくなります。
つまり、たまたま極端なデータが混じった影響は薄まりやすくなります。

しかし、未来の構造そのものが変わる場合、
過去データを増やしても、未来の母平均を正しく推定できるとは限りません。

つまり、

標本サイズが大きい
↓
ランダムなブレは小さくなる

しかし、

未来の構造が変わる
↓
過去データの平均が未来の平均を表すとは限らない

ということです。


---

# 9. 具体例

母標準偏差が2時間だとします。

つまり、人によって睡眠時間のばらつきがそれなりにある状態です。

## n = 4 の場合

```text
SE = 2 / √4
   = 2 / 2
   = 1
```

標準誤差は1時間です。

## n = 100 の場合

```text
SE = 2 / √100
   = 2 / 10
   = 0.2
```

標準誤差は0.2時間です。

かなり小さくなりました。

---

# 10. ここで重要な観察

標本サイズを4人から100人に増やすと、標準誤差は

```text
1 → 0.2
```

になりました。

つまり、標本平均のブレが小さくなります。

ただし、ここで雑に考えてはいけません。

標本サイズを25倍にしたからといって、標準誤差が25分の1になるわけではありません。

なぜなら、式は

```text
n
```

ではなく、

```text
√n
```

で割るからです。

標本サイズを4倍にすると、標準誤差は2分の1です。

標本サイズを100倍にすると、標準誤差は10分の1です。

この感覚はかなり大事です。

---

# 11. 競馬AIで考えると

ここはあなたにとってかなり重要です。

ある条件で、過去10レースの回収率が120%だったとします。

このとき、

```text
お、期待値あるじゃん
```

と判断するのは早いです。

10レースしかないなら、その120%はかなりブレます。

たまたま人気薄が1回来ただけで、回収率が跳ねることがあります。

一方で、同じ条件で過去1000レースあり、回収率が120%なら話は違います。

まだ完全ではありませんが、10レースよりはかなり信頼できます。

つまり、

|データ数|回収率120%の信用度|
|--:|---|
|10レース|かなり怪しい|
|100レース|まだ不安|
|1000レース|検討価値あり|
|10000レース|かなり強い材料|

です。

ただし、10000レースあっても、リーク・期間偏り・オッズ変化・買い目選別の後出しがあればダメです。

ここは逃げ道にしない方がいいです。

# 12. 「平均との差の合計は0」とつながる

前に学んだ、

> 各データから平均との差を取って、それを全部足すと0になる

という話がありました。

あれは、平均が「中心」だからです。

今回の話では、さらに一段進みます。

標本平均も、いろいろな標本を取るとブレます。

でも、その標本平均たちは、母平均の周りに集まりやすい。

つまり、

```text
母平均を中心にして、標本平均が散らばる
```

というイメージです。

この「標本平均たちの散らばり」が、次に出てくる **標本分布** です。

---

# 13. 標本分布とは何か

標本分布とは、

> 標本から計算される統計量が、どんなふうに分布するか

です。

今は標本平均を見ているので、

> 標本平均の分布

を考えています。

たとえば、500人ずつ何回も標本を取ったとします。

そのたびに標本平均を計算します。

```text
1回目の標本平均：6.4
2回目の標本平均：6.6
3回目の標本平均：6.5
4回目の標本平均：6.7
5回目の標本平均：6.3
...
```

これをたくさん集めると、標本平均たちは母平均の周りに分布します。

これが標本分布のイメージです。

---

# 14. 今日の最重要ポイント

今日のポイントは3つです。

|ポイント|内容|
|---|---|
|1|標本平均は、標本の取り方によってブレる|
|2|標本サイズが大きいほど、標本平均のブレは小さくなる|
|3|標本平均のブレの大きさを標準誤差という|

特に重要なのはこれです。

> 標準偏差は「個々のデータのばらつき」  
> 標準誤差は「標本平均のばらつき」

ここを混同すると、推定と検定で詰まります。
