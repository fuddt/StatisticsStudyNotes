

# 第13回：確率分布

## 今日のゴール

今日のゴールはこれです。

> 確率分布とは、
> 「どの値が、どれくらいの確率で起こるか」を整理したもの
> だと理解する。

そして、統計検定2級でよく出る次の分布の入口まで押さえます。

| 分布      | 何を表すか            |
| ------- | ---------------- |
| ベルヌーイ分布 | 成功・失敗の1回勝負       |
| 二項分布    | 成功・失敗を何回も繰り返す    |
| ポアソン分布  | 一定時間・一定範囲で何回起こるか |
| 正規分布    | 平均の周りに山型に散らばるデータ |

---

# 1. 確率分布とは何か

まず、確率変数からです。

## 確率変数

確率変数とは、

> 偶然によって値が決まる変数

です。

たとえば、サイコロを1回振るとします。

出る目を `X` とすると、

```text
X = 1, 2, 3, 4, 5, 6
```

のどれかになります。

この `X` が確率変数です。

---

## 確率分布

サイコロなら、それぞれの目が出る確率はこうです。

|  X |  確率 |
| -: | --: |
|  1 | 1/6 |
|  2 | 1/6 |
|  3 | 1/6 |
|  4 | 1/6 |
|  5 | 1/6 |
|  6 | 1/6 |

これが確率分布です。

つまり、

> 確率変数がどの値をどの確率で取るかをまとめたもの

です。

---

# 2. 離散型と連続型

確率分布には大きく2種類あります。

## 離散型確率分布

値が飛び飛びになるものです。

例：

```text
サイコロの目：1, 2, 3, 4, 5, 6
表が出た回数：0回, 1回, 2回, ...
客が来た人数：0人, 1人, 2人, ...
```

整数で数えられるものが多いです。

---

## 連続型確率分布

値が連続的に動くものです。

例：

```text
身長
体重
タイム
気温
売上金額
```

身長は `170cm` だけではなく、

```text
170.1cm
170.01cm
170.001cm
```

のように細かく考えられます。

---

## ここはかなり大事

離散型では、

```text
X = 3 になる確率
```

のように、ピンポイントの確率を考えられます。

でも連続型では、

```text
身長がちょうど170.000000...cm
```

の確率は基本的に0と考えます。

その代わり、

```text
身長が165cm以上175cm以下
```

のように、範囲で確率を考えます。

ここは最初は変に感じるけど、統計ではかなり重要です。

---

# 3. 確率分布で見るべき3つ

確率分布では、最低限この3つを見ます。

| 見るもの | 意味            |
| ---- | ------------- |
| 期待値  | 平均的にどれくらいか    |
| 分散   | どれくらいバラつくか    |
| 標準偏差 | 分散を元の単位に戻したもの |

---

# 4. 期待値

期待値は、ざっくり言うと、

> 確率を考慮した平均

です。

たとえば、次のくじを考えます。

|   賞金 |  確率 |
| ---: | --: |
|   0円 | 1/2 |
| 100円 | 1/2 |

このとき、期待値は、

```text
0 × 1/2 + 100 × 1/2 = 50
```

なので、期待値は50円です。

これは、

> 1回引いたら必ず50円もらえる

という意味ではありません。

実際には0円か100円です。

でも、何回も繰り返すと、平均的には50円くらいになる、という意味です。

---

## 期待値の公式

確率変数 `X` が、

|  値 | 確率 |
| -: | -: |
| x1 | p1 |
| x2 | p2 |
| x3 | p3 |

のように分布しているとき、

```text
期待値 = x1p1 + x2p2 + x3p3 + ...
```

です。

記号ではこう書きます。

```text
E(X) = Σ xP(X = x)
```

難しく見えるけど、やっていることは、

```text
値 × 確率
```

を全部足しているだけです。

---

# 5. 分散

分散は、

> 期待値からどれくらいズレるか

を表します。

確率分布における分散は、次のように考えます。

```text
それぞれの値が期待値からどれくらい離れているか
↓
そのズレを二乗する
↓
確率で重みづけして足す
```

公式はこれです。

```text
V(X) = E{(X - μ)^2}
```

ここで `μ` は期待値です。

もっと実用的には、こっちの公式もよく使います。

```text
V(X) = E(X^2) - {E(X)}^2
```

これは計算で便利です。

---

# 6. ベルヌーイ分布

ここから具体的な分布に入ります。

まずはベルヌーイ分布です。

## ベルヌーイ分布とは

ベルヌーイ分布は、

> 成功か失敗かの1回勝負

です。

たとえば、

```text
コインを1回投げる
馬券が当たるか外れるか
メールを開封するかしないか
商品を買うか買わないか
```

こういうものです。

値は普通、こう置きます。

| 結果 |  値 |
| -- | -: |
| 成功 |  1 |
| 失敗 |  0 |

成功確率を `p` とすると、

|  X |    確率 |
| -: | ----: |
|  1 |     p |
|  0 | 1 - p |

---

## ベルヌーイ分布の期待値と分散

ベルヌーイ分布では、

```text
期待値 = p
分散 = p(1 - p)
```

です。

たとえば、成功確率が `p = 0.3` なら、

```text
期待値 = 0.3
分散 = 0.3 × 0.7 = 0.21
```

です。

---

# 7. 二項分布

次に二項分布です。

## 二項分布とは

二項分布は、

> 成功・失敗の試行をn回繰り返したとき、成功回数が何回になるか

を表す分布です。

たとえば、

```text
コインを10回投げて、表が何回出るか
10人に営業して、何人が契約するか
20レース買って、何回当たるか
```

こういうものです。

---

## 二項分布の条件

二項分布になるには、基本的に次の条件があります。

| 条件          | 内容               |
| ----------- | ---------------- |
| 試行回数が決まっている | n回やる             |
| 各試行は成功か失敗   | 結果は2択            |
| 成功確率が一定     | 毎回p              |
| 各試行が独立      | 1回目の結果が2回目に影響しない |

---

## 二項分布の表記

成功確率が `p`、試行回数が `n` の二項分布は、

```text
X ~ B(n, p)
```

と書きます。

これは、

> Xは、試行回数n、成功確率pの二項分布に従う

という意味です。

---

## 二項分布の確率

たとえば、

> 成功確率0.3の試行を5回行い、ちょうど2回成功する確率

を考えます。

これは、

```text
n = 5
p = 0.3
x = 2
```

です。

二項分布の確率は、

```text
P(X = x) = nCx p^x (1-p)^(n-x)
```

です。

今回なら、

```text
P(X = 2) = 5C2 × 0.3^2 × 0.7^3
```

です。

計算すると、

```text
5C2 = 10
0.3^2 = 0.09
0.7^3 = 0.343
```

なので、

```text
10 × 0.09 × 0.343 = 0.3087
```

つまり、約30.87%です。

---

## 二項分布の期待値と分散

二項分布では、

```text
期待値 = np
分散 = np(1 - p)
```

です。

たとえば、

```text
X ~ B(10, 0.3)
```

なら、

```text
期待値 = 10 × 0.3 = 3
分散 = 10 × 0.3 × 0.7 = 2.1
```

です。

つまり、10回やれば平均的には3回成功する、ということです。

---

# 8. ポアソン分布

次はポアソン分布です。

## ポアソン分布とは

ポアソン分布は、

> 一定時間・一定範囲で、ある出来事が何回起こるか

を表す分布です。

例：

```text
1時間に来る客の人数
1日に発生する問い合わせ件数
1ページあたりの誤字の数
1レースで発生する出遅れの数
一定期間の事故件数
```

ポイントは、

> 回数を数える分布

だということです。

---

## 二項分布との違い

二項分布は、

```text
10回中、何回成功するか
```

のように、試行回数 `n` が決まっています。

一方、ポアソン分布は、

```text
1時間に何件起こるか
```

のように、時間や範囲が決まっています。

ここが違います。

---

## ポアソン分布の表記

ポアソン分布は、

```text
X ~ Po(λ)
```

と書きます。

`λ` はラムダと読みます。

意味は、

> 平均発生回数

です。

たとえば、

```text
1時間に平均3件問い合わせが来る
```

なら、

```text
λ = 3
```

です。

---

## ポアソン分布の期待値と分散

ポアソン分布は特徴的です。

```text
期待値 = λ
分散 = λ
```

つまり、平均も分散も同じです。

たとえば、

```text
X ~ Po(3)
```

なら、

```text
期待値 = 3
分散 = 3
```

です。

---

## ポアソン分布の確率

ポアソン分布の確率は、

```text
P(X = x) = e^(-λ) λ^x / x!
```

です。

これは今すぐ完全に暗記しなくてもいいですが、統計検定2級では出る可能性があります。

たとえば、

> 1時間に平均2件問い合わせが来る。1時間にちょうど3件来る確率は？

なら、

```text
λ = 2
x = 3
```

なので、

```text
P(X = 3) = e^(-2) × 2^3 / 3!
```

です。

---

# 9. 正規分布

次に正規分布です。

ここは超重要です。

## 正規分布とは

正規分布は、

> 平均の周りに山型に分布する連続型の分布

です。

イメージはこうです。

```text
          平均
           ↓
        ／￣￣＼
      ／        ＼
    ／            ＼
___/                \___
```

平均に近い値が多く、平均から離れるほど少なくなります。

---

## 正規分布の例

正規分布に近い形になりやすいものには、次のようなものがあります。

```text
身長
テストの点数
測定誤差
部品のサイズのばらつき
```

ただし、何でも正規分布になるわけではありません。

ここは注意です。

「なんとなく山型だから正規分布」と決めつけるのは危険です。

---

## 正規分布の表記

平均が `μ`、分散が `σ^2` の正規分布は、

```text
X ~ N(μ, σ^2)
```

と書きます。

たとえば、

```text
X ~ N(50, 10^2)
```

なら、

```text
平均50、標準偏差10の正規分布
```

です。

注意点は、第2引数が **標準偏差ではなく分散** で書かれることが多いことです。

```text
N(50, 100)
```

なら、

```text
平均50
分散100
標準偏差10
```

です。

ここは試験でひっかかりやすいです。

---

# 10. 標準正規分布

正規分布の中でも、特に重要なのが標準正規分布です。

## 標準正規分布とは

標準正規分布は、

```text
平均0
分散1
標準偏差1
```

の正規分布です。

記号では、

```text
Z ~ N(0, 1)
```

と書きます。

---

## 標準化

普通の正規分布を、標準正規分布に変換することを標準化といいます。

公式はこれです。

```text
Z = (X - μ) / σ
```

これはかなり重要です。

意味は、

> Xが平均から標準偏差何個分離れているか

です。

たとえば、

```text
平均50
標準偏差10
得点70
```

なら、

```text
Z = (70 - 50) / 10 = 2
```

です。

つまり、70点は平均より標準偏差2個分高い、という意味です。

---

# 11. 正規分布の確率は「面積」で考える

連続型分布では、確率は面積です。

たとえば、

```text
P(40 ≦ X ≦ 60)
```

は、

```text
Xが40以上60以下になる確率
```

です。

正規分布のグラフでいうと、この範囲の下の面積が確率になります。

```text
        ／￣￣＼
      ／██████＼
    ／██████████＼
___/____40__50__60_\___
```

この黒く塗った面積が確率です。

---

# 12. よく使う正規分布の目安

正規分布では、次の感覚が大事です。

| 範囲         | 含まれる確率 |
| ---------- | -----: |
| 平均 ± 1標準偏差 |   約68% |
| 平均 ± 2標準偏差 |   約95% |
| 平均 ± 3標準偏差 | 約99.7% |

たとえば、

```text
平均50
標準偏差10
```

なら、

| 範囲    |     確率 |
| ----- | -----: |
| 40〜60 |   約68% |
| 30〜70 |   約95% |
| 20〜80 | 約99.7% |

です。

これは厳密計算というより、正規分布の感覚として重要です。

---

# 13. 分布の使い分け

ここまでの分布を整理します。

| 分布      | 使う場面           | 例              |
| ------- | -------------- | -------------- |
| ベルヌーイ分布 | 1回の成功・失敗       | 1レースで馬券が当たるか   |
| 二項分布    | n回中の成功回数       | 10レース中、何回当たるか  |
| ポアソン分布  | 一定時間・範囲内の発生回数  | 1日に何件問い合わせが来るか |
| 正規分布    | 平均周辺に連続的に散らばる値 | 身長、点数、測定誤差     |

試験では、いきなり公式を使うより先に、

> この問題はどの分布の話か？

を判断する力が必要です。

# 14. 練習問題

## 問題1

コインを1回投げて、表なら1、裏なら0とする。
表が出る確率を `p = 0.5` とする。

この確率変数 `X` は何分布か？

---

## 解答

ベルヌーイ分布です。

理由は、

```text
1回だけ
成功・失敗の2択
```

だからです。

---

## 問題2

成功確率0.2のくじを10回引く。
当たりの回数を `X` とする。

この `X` は何分布か？

---

## 解答

二項分布です。

```text
X ~ B(10, 0.2)
```

です。

理由は、

```text
10回繰り返す
各回は当たり・外れ
成功確率が0.2
```

だからです。

---

## 問題3

1時間に平均4件問い合わせが来る。
1時間に来る問い合わせ件数を `X` とする。

この `X` は何分布か？

---

## 解答

ポアソン分布です。

```text
X ~ Po(4)
```

です。

理由は、

```text
一定時間内に何件起こるか
```

を見ているからです。

---

## 問題4

あるテストの点数 `X` が、

```text
X ~ N(60, 10^2)
```

に従うとする。

この分布の平均、分散、標準偏差は？

---

## 解答

```text
平均 = 60
分散 = 10^2 = 100
標準偏差 = 10
```

です。

注意点は、正規分布の表記 `N(μ, σ^2)` の2つ目は分散であることです。

---

## 問題5

平均60、標準偏差10のテストで、80点を取った。
この点数の標準化得点 `Z` は？

---

## 解答

```text
Z = (X - μ) / σ
```

なので、

```text
Z = (80 - 60) / 10 = 2
```

です。

つまり、80点は平均より標準偏差2個分高い点数です。

# 16. ここでの盲点

ここは率直に言うと、分布名を暗記しても弱いです。

本当に必要なのは、

```text
これは何を数えているのか？
1回なのか、n回なのか？
時間・範囲あたりなのか？
連続量なのか？
```

を見抜くことです。

統計検定2級では、公式そのものよりも、

> どの分布を使うべきか

の判断を外すと崩れます。

特に危ないのはこの2つです。

```text
二項分布とポアソン分布の混同
正規分布の N(μ, σ^2) の第2引数を標準偏差だと思うミス
```

ここは今の段階で潰しておいた方がいいです。


# 危ないポイントはこの2つ。

```text
1. 二項分布とポアソン分布の混同
2. 正規分布 N(μ, σ^2) の第2引数を標準偏差だと思うミス
```

---

# 対策1：二項分布とポアソン分布を見分ける

## 結論

まず、判断基準はこれ。

|見るポイント|二項分布|ポアソン分布|
|---|---|---|
|何を数える？|n回中の成功回数|一定時間・一定範囲での発生回数|
|試行回数n|明確にある|基本的にない|
|成功確率p|明確にある|基本的にない|
|平均発生回数λ|基本的にない|明確にある|
|例|10回中3回当たる|1時間に3件来る|

---

## 二項分布の合図

問題文にこういう表現があったら、二項分布を疑う。

```text
n回中
○回試行して
○人に聞いて
○個の商品を検査して
成功確率p
不良率p
当たる確率p
```

例：

> ある商品を10個検査する。不良品である確率は0.05である。不良品の個数Xを考える。

これは二項分布。

```text
X ~ B(10, 0.05)
```

理由は、

```text
10個検査する → nがある
不良品か正常品か → 成功・失敗の2択
不良率0.05 → pがある
```

だから。

---

## ポアソン分布の合図

問題文にこういう表現があったら、ポアソン分布を疑う。

```text
1時間あたり
1日あたり
1ページあたり
一定期間に
一定範囲内で
平均○件発生する
λ = ○
```

例：

> あるコールセンターには1時間あたり平均3件の問い合わせが来る。1時間に問い合わせが5件来る確率を求めよ。

これはポアソン分布。

```text
X ~ Po(3)
```

理由は、

```text
1時間あたり → 一定時間内
平均3件 → λがある
問い合わせ件数 → 発生回数を数えている
```

だから。

# 見分けの最短フローチャート

試験中はこれで判定するといい。

```text
問題文に「n回中」「n個中」「n人中」がある？
    ↓
はい → 二項分布を疑う
    ↓
いいえ
    ↓
「1時間あたり」「1日あたり」「1ページあたり」「平均λ件」がある？
    ↓
はい → ポアソン分布を疑う
```

さらに圧縮すると、

```text
nとpがある → 二項分布
λがある → ポアソン分布
```

これでかなり防げる。

---

# 注意：二項分布とポアソン分布は近い関係にある

ここが混乱の原因。

実はポアソン分布は、二項分布の近似として出てくることがある。

条件はざっくりこれ。

```text
n が大きい
p が小さい
np がほどほど
```

このとき、

```text
B(n, p) ≒ Po(np)
```

として扱えます。

たとえば、

```text
X ~ B(1000, 0.002)
```

なら、

```text
np = 1000 × 0.002 = 2
```

なので、

```text
X ≒ Po(2)
```

と近似できます。

ただし、最初の段階ではこう覚えればいい。

> もともとは二項分布。  
> でも n が大きく p が小さいとき、ポアソン分布で近似できる。

ここを逆に覚えると危ない。

# 対策2：正規分布 N(μ, σ²) の第2引数ミス

これは本当に頻出の凡ミス。

## 結論

正規分布は普通こう書く。

```text
X ~ N(μ, σ²)
```

意味は、

```text
μ = 平均
σ² = 分散
σ = 標準偏差
```

つまり、第2引数は **標準偏差ではなく分散**。

---

## 例

```text
X ~ N(50, 100)
```

なら、

```text
平均 = 50
分散 = 100
標準偏差 = 10
```

です。

ここで、

```text
標準偏差 = 100
```

としてしまうとアウト。

---

## さらに危ない表記

次の2つは意味が同じです。

```text
X ~ N(50, 10²)
X ~ N(50, 100)
```

どちらも、

```text
平均50
分散100
標準偏差10
```

です。

つまり、`N(50, 10²)` と書かれていたら、標準偏差は10。

でも `N(50, 100)` と書かれていたら、標準偏差は√100で10。

ここを毎回確認する。

---

# 標準化でミスが爆発する

このミスが危ないのは、標準化で一気に誤答になるから。

標準化の公式は、

```text
Z = (X - μ) / σ
```

ここで使うのは **標準偏差σ** です。  
分散σ²ではありません。

---

## 例題

```text
X ~ N(50, 100)
```

のとき、`X = 70` を標準化せよ。

まず確認。

```text
平均 μ = 50
分散 σ² = 100
標準偏差 σ = 10
```

だから、

```text
Z = (70 - 50) / 10
  = 2
```

正解は、

```text
Z = 2
```

です。

間違って分散100で割ると、

```text
Z = (70 - 50) / 100
  = 0.2
```

になってしまう。

これは完全に別物。

# 試験中のチェックルール

この2つは、チェックルールを固定した方がいい。

## ルール1：分布名を見る前に、問題文の単位を見る

```text
n回中？ → 二項分布
1時間あたり？ → ポアソン分布
連続量？ → 正規分布の可能性
```

## ルール2：正規分布を見たら必ず3点セットに分解する

問題文に、

```text
X ~ N(60, 25)
```

と出たら、すぐにこう書く。

```text
μ = 60
σ² = 25
σ = 5
```

これを省略しない。

頭の中で処理するとミスる。  
本番では必ずメモ欄に書いた方がいい。

# ミス防止テンプレ

問題を解く前に、この形でメモするといい。

## 二項分布っぽいとき

```text
n = 
p =

```

## ポアソン分布っぽいとき

```text
λ = 
X = 一定時間・範囲内の発生回数
X ~ Po(λ)
```

## 正規分布っぽいとき

```text

σ =

```

これを固定化するだけで、かなり事故が減る。

# 追加の練習問題

## 問題1

ある製品の不良率は2%である。  
100個の製品を検査したとき、不良品の個数Xを考える。

これは何分布か？

---

### 解答

二項分布。

```text
X ~ B(100, 0.02)
```

理由：

```text
100個中 → nがある
不良品かどうか → 成功・失敗の2択
不良率2% → pがある
```

---

## 問題2

あるWebサイトには1分あたり平均4回アクセスがある。  
1分間のアクセス回数Xを考える。

これは何分布か？

---

### 解答

ポアソン分布。

```text
X ~ Po(4)
```

理由：

```text
1分あたり → 一定時間内
平均4回 → λがある
アクセス回数 → 発生回数
```

---

## 問題3

```text
X ~ N(70, 16)
```

このとき、平均・分散・標準偏差は？

---

### 解答

```text
平均 = 70
分散 = 16
標準偏差 = 4
```

---

## 問題4

```text
X ~ N(70, 16)
```

のとき、`X = 78` を標準化せよ。

---

### 解答

まず、

```text
μ = 70
σ² = 16
σ = 4
```

なので、

```text
Z = (78 - 70) / 4
  = 8 / 4
  = 2
```

答えは、

```text
Z = 2
```

## 問題5

ある病院では、1日に平均2人の救急患者が来る。  
1日に来る救急患者数Xを考える。

これは何分布か？

---

### 解答

ポアソン分布。

```text
X ~ Po(2)
```

理由：

```text
1日に平均2人 → 一定時間あたりの平均発生回数
```

# 最終的な覚え方

かなり実戦寄りに言うと、こう覚えるのがいい。

```text
二項分布：n回やって何回成功するか
ポアソン分布：一定時間・範囲で何回起こるか
正規分布：N(平均, 分散)
標準化：割るのは標準偏差
```

この4行は暗記していい。

特に最後。

```text
N(μ, σ²) の第2引数は分散
Z = (X - μ) / σ で割るのは標準偏差
```

ここを混ぜると、計算が全部ズレる。  
次回以降、標本分布・推定・検定に入ると、このミスはもっと致命傷になる。だから今の段階で、毎回 `μ, σ², σ` に分解する癖をつけた方がいい。
